Brightcove-Galerie™ Produkt-Updates

{{ page.description }}

Versionshinweise sind auch verfügbar für:

  • Versionshinweise zu Galerievorlagen

Deaktivieren der SSO-Authentifizierungskontextanforderung

15. April 2021

Eine neue Schnittstellenoption zum Deaktivieren der SSO-Authentifizierungskontextanforderung wurde zu Zugriffssteuerungsprofilen hinzugefügt. Weitere Informationen finden Sie unter Konfigurieren von Zugriffssteuerungsprofilen.

Auto-Play-Videoeinstellung

4 Sep. 2019

Eine neue Video automatisch abspielen Die Einstellung wurde hinzugefügt und kann verwendet werden, um das Wiedergabeverhalten von Videos auf Detailseiten zu steuern. Weitere Informationen finden Sie unter Anpassen der Player- und Lead-Formulareinstellungen für eine Portal-Erfahrung.

Neu gestaltete Galerie-Navigation

27. September 2018

Das Navigationsmenü beim Bearbeiten von Portal Experiences wurde so gestaltet, dass es intuitiver und konsistenter mit der Bearbeitungsumgebung von In-Page Experience ist. Weitere Informationen finden Sie unter Verwenden des Site-Editors zum Anpassen einer Portal-Erfahrung.

Suchoption zur Galerie-Homepage hinzugefügt

1. August 2018

EIN Suche Auf der Galerie-Homepage wurde ein Feld hinzugefügt, mit dem Sie die Liste der Erlebnisse durchsuchen können. Bei der Suche werden die folgenden Eigenschaften durchsucht:

  • Titel
  • Beschreibung
  • ICH WÜRDE
  • Vorlage
  • URLs

Hinzufügen von Videos zu einer In-Page-Erfahrung

18. Juli 2018

Beim Hinzufügen von Videos zu einer In-Page-Erfahrung besteht jetzt die Möglichkeit, eine Referenz-ID für die Wiedergabeliste zu verwenden. Durch die Verwendung einer Referenz-ID für die Wiedergabeliste können die Videos in der Erfahrung leicht geändert werden, indem die Referenz-ID mithilfe des Medienmoduls einer anderen Wiedergabeliste neu zugewiesen wird. Die Erfahrung muss nicht bearbeitet und erneut veröffentlicht werden, wenn Sie müssen die Videos ändern, die in der Erfahrung angezeigt werden. Weitere Informationen finden Sie unter Verwenden einer Wiedergabelisten-Referenz-ID.

Änderungen an Lead-Formularen

15. Mai 2018

Gallery verwendet jetzt das Zielgruppenmodul, um Lead-Formulare zu erstellen und zu verwalten. Die Zielgruppe sammelt die Lead-Daten und bietet eine Option für den Datenexport. Weitere Informationen finden Sie unter Verwenden eines Lead-Formulars mit Portal-Erfahrung.

In-Page-Vorlage für Live-Ereignisse

14. Mai 2018

Die Galerie bietet jetzt eine Live-Event-In-Page-Vorlage, mit der Live-Events als Teil einer In-Page-Erfahrung bereitgestellt werden können. Weitere Informationen finden Sie unter Erstellen einer Live-Event-In-Page-Erfahrung.

Aktualisierungen der Benutzeroberfläche von Portal Experience Site-Einstellungen

13. April 2018

Die Benutzeroberfläche für Site-Einstellungen wurde aktualisiert, um die Video-bezogenen Einstellungen aus dem zu entfernen Benutzerdefiniert Dialog zu einem neuen Video Einstellungsdialog. Weitere Informationen finden Sie unter Anpassen der Videoeinstellungen für eine Portal-Erfahrung.

Festzeltvorlageneinstellung

30. März 2018

Der Festzeltvorlage wurde eine neue Einstellung für das Videoverhalten hinzugefügt. Weitere Informationen finden Sie unter Anpassen der Einstellungen für Laufschriftvorlagen.

Benutzerdefinierte Kopf- und Fußzeilenoption

30. März 2018

Es ist jetzt möglich, die Standard-Kopf- und Fußzeile einzuschließen, wenn Sie eine benutzerdefinierte Kopf- und Fußzeile verwenden. Weitere Informationen finden Sie unter Konfigurieren von benutzerdefiniertem Kopf- / Fußzeilen-HTML.

In-Page Experience 3Play-Komponenten

1. Februar 2018

Interaktive Transkriptionskomponenten von 3Play Media können jetzt zu einer In-Page-Erfahrung hinzugefügt werden. Auf diese Weise können die Zuschauer ein Video-Transkript sehen, während ein Video abgespielt wird. Weitere Informationen finden Sie unter Hinzufügen von Komponenten zu einer In-Page-Erfahrung.

In-Page Experience Chat-Komponenten

17. Januar 2018

Chat-Komponenten können jetzt zu einer In-Page-Erfahrung hinzugefügt werden. Weitere Informationen finden Sie unter Hinzufügen von Komponenten zu einer In-Page-Erfahrung.

Benutzerdefinierte Zugriffskontrollseite

28. November 2017

Die Zugriffskontrollseite kann jetzt so gestaltet werden, dass sie enger mit dem Rest der Website verknüpft ist. Weitere Informationen finden Sie unter Gestalten der Zugriffskontrollseite.

Stilisierte 404 Seiten

6. November 2017

404 Seiten verwenden jetzt das mit der Site konfigurierte Styling (Kopf- und Fußzeilen), sodass sie enger mit dem Rest der Site verknüpft sind.

Live-Ereignisvorlage - Hintergrundbildoptionen

6. November 2017

Die Vorlage für Live-Ereignisse bietet jetzt Optionen für den Umgang mit Hintergrundbildern. Weitere Informationen finden Sie unter Anpassen der Live Event Portal-Einstellungen.

Sammlungsmetadaten

6. November 2017

Für alle Vorlagen (außer Marquee) können Sie jetzt Sammlungsbeschreibungen hinzufügen, die als Metadatenbeschreibung für die Kategorieseite dienen (für Browser, die die Sammlung teilen). Weitere Informationen finden Sie unter Hinzufügen von Video Cloud-Videos zu einer Portal-Erfahrung.

API-Schlüssel nicht mehr erforderlich

29. September 2017

Der API-Schlüssel wird nicht mehr benötigt und wurde von der Seite Einstellungen entfernt.

Mehrere In-Page-Erfahrungen auf derselben Seite

15. August 2017

Gallery unterstützt jetzt mehrere Instanzen derselben In-Page-Erfahrung auf derselben Seite. Auf diese Weise können Sie eine Erfahrung erstellen und dann dynamisch eine Wiedergabeliste oder eine Liste der anzuzeigenden Videos übergeben. Weitere Informationen finden Sie unter Verwenden dynamischer Videoüberschreibungen in einer In-Page-Erfahrung.

Event Experience Now Unterstützt die Verwendung der Referenz-ID

9. August 2017

Beim Konfigurieren des Live-Ereignisstatus für eine Ereigniserfahrung kann das Live-Video-Asset jetzt mithilfe einer Referenz-ID angegeben werden. Die Verwendung einer Referenz-ID bietet mehr Flexibilität, da die Referenz-ID dem Live-Video-Asset zugewiesen werden kann, ohne dass Änderungen an der Benutzererfahrung vorgenommen werden müssen. Weitere Informationen finden Sie unter Erstellen des Live-Ereignisstatus für eine Ereigniserfahrung.

Zusätzliche In-Page Experience Player-Einstellung

27. Juli 2017

Eine neue Player-Einstellung wurde hinzugefügt, damit eine In-Page-Erfahrung nach Abschluss eines Videos im Wiedergabezustand bleibt. Dies kann verwendet werden, um den Endbildschirm des Players anzuzeigen. Weitere Informationen finden Sie unter Konfigurieren der Einstellungen für eine In-Page-Erfahrung.

Dynamische Videoüberschreibungen in einer In-Page-Erfahrung

27. Juli 2017

Es ist jetzt möglich, die mit einer In-Page-Erfahrung verknüpften Videos dynamisch zu überschreiben, indem der Einbettungscode geändert wird. Weitere Informationen finden Sie unter Verwenden dynamischer Videoüberschreibungen in einer In-Page-Erfahrung.

In-Page-Erfahrungstraining jetzt verfügbar

21 Jul 2017

On-Demand-Schulungen zum Erstellen von In-Page-Erlebnissen sind jetzt verfügbar. Weitere Informationen finden Sie unter Training on Demand: Erstellen von In-Page-Erfahrungen mithilfe der Galerie.

YouTube-Video-Unterstützung für In-Page-Erlebnisse

11. Juli 2017

Es ist jetzt möglich, YouTube-Videos in einer In-Page-Erfahrung zu verwenden. Weitere Informationen finden Sie unter Hinzufügen von Videos zu einer In-Page-Erfahrung.

Verwenden Sie die Site-Beschreibung in Suchmaschinenergebnissen

10. Juli 2017

Die Galerie bietet jetzt die Möglichkeit, die Site-Beschreibung in Suchmaschinenergebnissen zu verwenden. Hierbei wird die Site-Beschreibung in den <meta> -Beschreibungs-Tags verwendet, einschließlich der Facebook- und Twitter-Meta-Tags. Weitere Informationen finden Sie unter Konfigurieren der Einstellungen für die Suchmaschinenoptimierung für eine Portal-Erfahrung.

Katalogvorlage

28. Juni 2017

Die Galerie bietet jetzt eine Katalogvorlage. Weitere Informationen finden Sie unter Übersicht über Galerieportalvorlagen.

In-Page-Erfahrungen

31. Mai 2017

Die Galerie bietet jetzt die Möglichkeit, In-Page-Erlebnisse zu erstellen. Weitere Informationen finden Sie unter Überblick: Erstellen, Bearbeiten und Veröffentlichen von In-Page-Erlebnissen.
