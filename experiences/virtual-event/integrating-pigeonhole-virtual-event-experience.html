Integration von Pigeonhole in ein virtuelles Event-Erlebnis

{{ page.description }}

Pigeonhole kann verwendet werden, um einem oder mehreren Videos in einem virtuellen Event-Erlebnis auf einfache Weise ereignisbasierte Interaktivität (Live-Fragen und -Antworten, Umfragen, Umfragen usw.) hinzuzufügen. In der Regel wird für das Live-Event ein neues Pigeonhole erstellt. Anschließend können für jedes Video, zu dem Sie Pigeonhole-Funktionen hinzufügen möchten, Sitzungen erstellt werden. Sobald die Sitzungen erstellt wurden, wird jedem Video mithilfe eines benutzerdefinierten Felds für die Video Cloud der Iframe-Einbettungscode zugeordnet. Der Einbettungscode bewirkt, dass der Pigeonhole-Pod für ausgewählte Videos neben dem Videoplayer angezeigt wird.

Erstellen eines neuen Pigeonhole-Ereignisses und einer neuen Sitzung

Sie sollten ein neues Pigeonhole für Ihr Live-Event erstellen und dann Sitzungen für jedes Video hinzufügen, dem Pigeonhole-Funktionen hinzugefügt wurden.

  1. Melden Sie sich bei Ihrem Pigeonhole-Konto an.
  2. Klicken + Pigeonhole hinzufügen und klicken Sie auf Ihr Konto, um es zu verwenden.
  3. Geben Sie ein Veranstaltungsname und Passcode.
  4. Eingeben Start und Ende Termine für die Veranstaltung. Diese Daten steuern, wie lange das Pigeonhole aktiv sein wird.
  5. Wählen Sie a Zeitzone und Benutzerdefiniertes Branding Option (falls zutreffend).
  6. Klicken Fortsetzen.
  7. Nachdem das Ereignis erstellt wurde, wird das Agenda Element wird in der linken Navigation ausgewählt. Klicken Sitzung hinzufügen und klicken Sie auf Erstelle neu.
  8. Wählen Sie a Sitzungstyp von Fragen und Antworten (Andere Sitzungstypen können ebenfalls verwendet werden).
  9. Geben Sie ein Sitzungsname.
  10. Eingeben Start und Ende Datum und Uhrzeit.
  11. Konfigurieren Sie die Q & A-Einstellungen wie angemessen.
  12. Klicken Fragen und Antworten hinzufügen und bestätigen Sie, dass die Sitzung erstellt wurde.
  13. Erstellen Sie nach Bedarf weitere Sitzungen.

Generieren des Pigeonhole-Einbettungscodes

Damit ein Pigeonhole neben einem Video angezeigt wird, wird dem Video mithilfe eines benutzerdefinierten Felds für die Video Cloud ein Iframe-Einbettungscode hinzugefügt. Das Format des Codes lautet:

<iframe src="Schubladen-URL„></iframe >

Gehen Sie folgendermaßen vor, um den Wert für die Pigeonhole-URL zu erhalten.

  1. Klicken Sie in Pigeonhole auf Führen Sie Ihre Veranstaltung aus in der linken Navigation.
  2. Klicken Sie auf den Link in der Laden Sie Ihre Teilnehmer ein Sektion.
  3. Stellen Sie sicher, dass eine neue Browser-Registerkarte mit einer Liste aller Sitzungen geöffnet wird.
  4. Klicken Sie auf den Link „Enter“ für die entsprechende Sitzung, z. B. Geben Sie diese Frage und Antworten ein. Beachten Sie, dass sich der Link basierend auf dem Sitzungstyp ändert.
  5. Stellen Sie sicher, dass die Schublade in einem neuen Browser-Tab geöffnet wird. Die Pigeonhole-URL wird in der Adressleiste des Browsers angezeigt.

Das iframe Das Tag mit der Pigeonhole-URL wird dem entsprechenden Video hinzugefügt.

Ereigniseinstellungen konfigurieren

Die Einstellungen für das Pigeonhole-Ereignis müssen ebenfalls konfiguriert werden, damit die SSO-Informationen zum Erstellen eines Benutzerprofils verwendet werden können.

  1. Klicken Sie in der linken Navigation auf Mehr Einstellungen.
  2. Drücke den Teilnehmer Tab.
  3. Überprüf den Ermöglichen Sie das Erstellen von Teilnehmerprofilen durch Einbetten Möglichkeit.
  4. Scrollen Sie nach unten zu Registrantenprofile Abschnitt auf der Seite.
  5. Überprüf den Aktivieren Sie das Registrierungsprofil für diese Veranstaltung Möglichkeit.

Erstellen eines benutzerdefinierten Felds

Damit die Vorlage für virtuelle Ereignisse weiß, welche Schublade für ein Video angezeigt werden soll, wird ein benutzerdefiniertes Feld verwendet. Führen Sie die folgenden Schritte aus, um ein benutzerdefiniertes Feld zu erstellen.

  1. Melden Sie sich bei Video Cloud Studio an.
  2. Klicken Sie in der Navigationskopfzeile auf ADMIN und dann auf Videofelder. Beachten Sie, dass Sie ein Kontoadministrator sein müssen, um auf das ADMIN-Menü zugreifen zu können.
  3. Erstellen Sie ein neues benutzerdefiniertes Feld mit dem Namen Chat einbetten.
  4. Bestätigen Sie, dass der interne Name lautet chat_embed.
  5. Stellen Sie die Art sein Text.
  6. Speichern Sie das benutzerdefinierte Feld.

Informationen zum Erstellen benutzerdefinierter Felder in Video Cloud finden Sie unter Benutzerdefinierte Metadatenfelder erstellen.

Hinzufügen einer Schublade zu einem Video

Führen Sie die folgenden Schritte aus, damit ein Pigeonhole neben einem Video in der Vorlage für virtuelle Ereignisse angezeigt wird.

  1. Öffne das Medien Modul.
  2. Klicken Sie auf ein Video, um die Videoeigenschaften zu öffnen.
  3. Suchen Sie die BENUTZERDEFINIERTE FELDER Abschnitt und klicken Bearbeiten.
  4. Fügen Sie den Einbettungscode für Pigeonhole in das ein Chat einbetten benutzerdefinierte Feld.
  5. Klicken speichern.

Weitergabe von Benutzer-SSO-Informationen an Pigeonhole

Wenn Ihre Virtual Event Experience mithilfe von SSO gesichert ist, können Sie SSO-Benutzerinformationen an Pigeonhole übergeben, damit Benutzer innerhalb von Pigeonhole verfolgt werden können. In den folgenden Abschnitten werden die Einstellungen und Schritte beschrieben, die zum Übergeben von SSO-Informationen an Pigeonhole erforderlich sind.

Ordnen Sie SSO-Attribute zu

Sie müssen sicherstellen, dass die richtigen SSO-Attribute zugeordnet sind, damit sie an Ihre Erfahrung und dann an Pigeonhole übergeben werden können. Die Integration Pigeonhole sucht nach folgenden Werten:

  • fname- Benutzername
  • nameID- E-Mail-Adresse des Benutzers

Das folgende Beispiel zeigt, wie diese Attribute auf der Okta SSO Admin-Seite verfügbar gemacht werden.

Stellen Sie Attribute im Zugriffssteuerungsprofil bereit

Stellen Sie sicher, dass das zur Sicherung der Benutzererfahrung verwendete Zugriffssteuerungsprofil über Folgendes verfügt: Legen Sie alle Attribute offen Option aktiviert. Wenn die SSO-Informationen nicht verfügbar sind, werden Benutzer in Pigeonhole als anonym angezeigt.

  1. Öffnen Sie das Galerie-Modul.
  2. Klicken die Einstellungen in der Kopfzeile.
  3. Klicken Sie in der linken Navigation auf Zugriffskontrollprofile.
  4. Klicken Sie auf das Bearbeitungssymbol für das Profil, das zum Sichern der Erfahrung verwendet wird.
  5. Stellen Sie sicher, dass die Einstellung Alle Attribute anzeigen aktiviert ist.

Informationen zum Konfigurieren der einmaligen Anmeldung finden Sie unter Konfigurieren von Zugriffssteuerungsprofilen. Wenn alles richtig konfiguriert ist, werden die Benutzerinformationen in Pigeonhole angezeigt, wenn sich der Betrachter bei der Erfahrung anmeldet.
