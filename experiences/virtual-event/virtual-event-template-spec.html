Spezifikationen für virtuelle Ereignisvorlagen

{{ page.description }}

Video und Audio

Video- und Audioempfehlungen sind unten aufgeführt.

Video

Formatieren MP4
Codec H.264, H.265
Bitrate 2x die Bitrate der höchsten Wiedergabe
Bildrate Konstant, normalerweise zwischen 15 und 60 Bildern pro Sekunde
Seitenverhältnis Ihr Video-Seitenverhältnis bleibt erhalten. Ein Seitenverhältnis von 16: 9 ist üblich.

Audio

Codec AAC (Advanced Audio Codec)
Bitrate Konstante 128 kbps
Kanäle Stereo

Empfehlungen zu bewährten Methoden zum Aufzeichnen, Exportieren, Hochladen und Veröffentlichen Ihrer Videos, um eine optimale Wiedergabe zu gewährleisten, finden Sie unter Empfohlene Vorgehensweise: Der Workflow zum Erstellen, Transcodieren und Hochladen von Videos.

Bilder

In der Vorlage für virtuelle Ereignisse werden verschiedene Bilder verwendet. Bildempfehlungen sind unten aufgeführt.

Hintergrundbild

Die Bildgröße hängt von den Haltepunkten ab, die der Kunde unterstützen möchte. Meistens funktioniert 1920 x 1080, aber das Bild sollte optimiert werden, damit es für die beste Leistung nicht gigantisch ist.

Header-Logo

Das Header-Logo wird proportional auf eine Breite von 178 Pixel verkleinert.

Agenda

Die Standardbildgröße beträgt 320 x 180 Pixel. Bilder werden auf diese Größe zugeschnitten.

Sprecher

Lautsprecherbilder sollten eine maximale Breite von 200 Pixel haben. Die Größe der Bilder wird nicht geändert. Standardmäßig konfiguriert das Vorlagen-CSS die Lautsprecherbilder mit einem Randradius von 50%, wodurch die Bilder kreisförmig werden. Die unter dem Sprecher angezeigte Sprecher-Biografie ist auf ungefähr 3 Zeilen mit 15 Wörtern begrenzt.

Sponsoren

Die Größe der Sponsorenbilder wird auf 100 Pixel geändert und die Deckkraft auf 80% eingestellt. Beispiel CSS wird bereitgestellt, um dieses Verhalten zu überschreiben.

Schriftarten

Die Vorlage ist für die Arbeit mit optimiert Roboto kondensiert Schriftart. Diese Schriftart kann mit der zugewiesen werden Stil Menü innerhalb der Galerie-Oberfläche. Andere Schriftarten können verwendet werden, erfordern jedoch möglicherweise Änderungen an CSS-Dateien. Informationen zur Verwendung benutzerdefinierter Schriftarten finden Sie unter Verwenden von benutzerdefinierten Schriftarten in einer Portal-Erfahrung.

Agenda

Die Veranstaltungsagenda listet den Zeitplan von Live-Events und anderen Sitzungen auf. Wenn eine Agenda erstellt und veröffentlicht wurde, erscheint sie auf der Veranstaltungshomepage. Informationen zum Erstellen der Veranstaltungsagenda finden Sie unter Verwaltung der Veranstaltungsagenda.
