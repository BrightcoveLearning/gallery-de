Verwaltung der Veranstaltungsagenda

{{ page.description }}

Das Agenda-Modul unterstützt das Hochladen einer externen CSV-Agenda-Datei, sodass nicht alle Agenda-Elemente manuell in das Agenda-Modul eingegeben werden müssen. Das Hochladen einer Agenda-Datei ist die Art und Weise, wie die Veranstaltungsagenda anfänglich erstellt wird. Nachdem eine Agenda hochgeladen wurde, kann sie mit dem Modul Agenda bearbeitet werden.

Hinweis: Informationen zum Erstellen einer Agendadatei für den Upload finden Sie unter Erstellen einer Ereignis-Agendadatei.

Hochladen einer Agenda-Datei

Bevor Sie Ihre bestehende Agenda in das Agenda-Modul importieren, sollte die Agenda-Datei in eine .csv-Datei exportiert werden. Das Modul Agenda kann nur eine CSV-Datei importieren.

Mit dem Galerie-Site-Editor können Sie die Veranstaltungsagenda hochladen und konfigurieren. Führen Sie die folgenden Schritte aus, um eine Agenda-Datei in ein Virtual Event Experience hochzuladen.

  1. Klicken Sie auf den Erlebnisnamen, um den Galerie-Site-Editor zu öffnen.
  2. Klicken Sie in der linken Navigation auf SITE-FUNKTIONEN> Agenda.
  3. Klicken Agenda hochladen.
  4. Durchsuchen und wählen Sie die hochzuladende Agenda-Datei aus.
  5. Bestätigen Sie, dass die Tagesordnung auf der Liste aufgeführt ist Meine Agenda Seite.

Wenn beim Import der Agenda Fehler auftreten, wird eine Meldung angezeigt. Nach dem Import wird die Status wird sein DEAKTIVIERT. Agendas mit dem Status DEAKTIVIERT werden in der Virtual Event Experience nicht angezeigt.

Bearbeiten des Agendatitels

Der Agendatitel wird auf der Erlebnishomepage über der Agenda angezeigt. Standardmäßig wird der Agendatitel auf den importierten Agendadateinamen gesetzt.

Um den Titel der Agenda zu bearbeiten, klicken Sie auf das Bearbeitungssymbol, geben Sie einen Titel ein und drücken Sie dann Eintreten.

Anzeigen der Tagesordnung

Nachdem eine Agenda-Datei hochgeladen wurde, erscheint sie auf dem Meine Agenda Seite. Um die Tagesordnungsdetails anzuzeigen, klicken Sie auf das Veranstaltungsname. Die Agenda wird geöffnet und zeigt die Details an.

Klicken Sie links auf der Seite auf ein Datum, um die Agenda für diesen Tag anzuzeigen. Die Tracks werden in den Spaltenüberschriften oben auf der Seite angezeigt.

Eine Sitzung hinzufügen

Gehen Sie folgendermaßen vor, um eine neue Sitzung hinzuzufügen.

  1. Klicken Sie auf die Agenda, um den Agenda-Editor zu öffnen.
  2. Klicken Sie auf ein offenes Zeitfenster.
  3. Geben Sie die Sitzungsinformationen ein.
    • Sitzungsname
    • Lautsprecher - Klick Lautsprecher auswählen und Wählen Sie einen oder mehrere Lautsprecher aus für die Sitzung
    • Startdatum
    • Startzeit
    • Dauer
    • Beschreibung
    • Track – Wählen Sie einen Track aus, dem die Sitzung zugewiesen werden soll
    • Sitzungstyp
      • Session
      • Keynote
      • Treffen
      • Andere
    • Video-ID – Verfügbar mit Sitzungs-, Keynote- und anderen Sitzungstypen
    • URL - Verfügbar mit Besprechungssitzungstyp
    • Titelbild
    • Zugehöriger Inhalt - Kann verwendet werden, um einen Link zu Sitzungsinhalten (Handouts oder Folien) bereitzustellen.
      • Label - Label für den Inhalt
      • URL - Link zum Inhalt
  4. Klicken speichern.

Bestätigen Sie, dass die Sitzung zur Tagesordnung hinzugefügt wurde. Wenn mehrere Sitzungen im selben Track gleichzeitig geplant sind, werden sie hervorgehoben und erscheinen nebeneinander.

Sitzungen bearbeiten und löschen

Um eine Sitzung zu bearbeiten oder zu löschen, klicken Sie auf eine Sitzung, um das Fenster Sitzung bearbeiten zu öffnen. Um die Sitzung zu bearbeiten, nehmen Sie alle Änderungen vor und klicken Sie auf Speichern. Um die Sitzung zu löschen, klicken Sie auf Sitzung löschen.

Hinzufügen von Sprechern zu einer Sitzung

Gehen Sie wie folgt vor, um Sprecher zu einer Sitzung hinzuzufügen.

  1. Klicken Sie auf eine Sitzung, um den Sitzungseditor zu öffnen.
  2. Klicken Lautsprecher auswählen.
  3. Wählen Sie einen oder mehrere Lautsprecher aus oder klicken Sie auf Lautsprecher erstellen um einen neuen Lautsprecher zu erstellen.
    Hinweis: Informationen zum Erstellen von Referenten finden Sie unter Verwalten von Ereignissprechern.
  4. Klicken speichern.
Die Suche Feld kann verwendet werden, um nach Sprechern zu suchen.

Hinzufügen eines neuen Titels

Tracks werden verwendet, um verwandte Sitzungen zu gruppieren. Der Marketing-Track enthält beispielsweise alle marketingbezogenen Sitzungen. Eine Agenda hat normalerweise mehrere Spuren. Gehen Sie folgendermaßen vor, um einen neuen Titel hinzuzufügen.

  1. Klicken Sie auf die Agenda, um den Agenda-Editor zu öffnen.
  2. Drücke den Neuen Track hinzufügen Link oben auf der Tagesordnung. Möglicherweise müssen Sie die Agenda nach rechts scrollen, damit der Link angezeigt wird.
  3. Geben Sie die Titelname und drücke Eintreten.

Bearbeiten eines Titels

Um einen Track zu bearbeiten, klicke auf den Tracknamen, nimm alle Änderungen vor und drücke Eintreten um die Änderungen zu speichern.

Tracks neu anordnen

Gehen Sie wie folgt vor, um die Reihenfolge der Titel in der Agenda zu ändern.

  1. Klicken Sie auf die Agenda, um den Agenda-Editor zu öffnen.
  2. Bewegen Sie den Mauszeiger über den Tracknamen, um ihn zu verschieben, klicken Sie auf das Grab-Symbol und ziehen Sie den Track an die neue Position.

Aktivieren und Deaktivieren einer Agenda

Der Agendastatus bestimmt, ob die Agenda in der zugehörigen Virtual Event Experience angezeigt wird oder nicht. Der Status wird im Seitenkopf unter dem Namen der Agenda angezeigt.

  • AKTIVIERT - Die Agenda wird in der zugehörigen Virtual Event Experience angezeigt
  • DEAKTIVIERT - Die Agenda wird in der zugehörigen Virtual Event Experience nicht angezeigt

Um eine Agenda zu aktivieren, klicken Sie auf aktivieren Sie oben auf der Seite. Um eine Agenda zu deaktivieren, klicken Sie darauf und klicken Sie auf Deaktivieren. Beachten Sie, dass das Erlebnis nach dem Aktivieren/Deaktivieren einer Agenda nicht erneut veröffentlicht werden muss.
