Erstellen einer virtuellen Event-Erfahrung

{{ page.description }}

Die Galerie bietet eine Vorlage für virtuelle Ereignisse, mit der virtuelle Ereigniserlebnisse zum Hosten und Bereitstellen virtueller Ereignisse erstellt werden können. Nach dem Erstellen einer neuen virtuellen Ereigniserfahrung werden Sie aufgefordert, die Erfahrung vorab auszufüllen. Brightcove empfiehlt die Auswahl Nein danke.

Der Vorgang zum Hinzufügen von Videos zur Erfahrung ähnelt anderen Erfahrungen mit dem Galerieportal, obwohl die Vorlage für virtuelle Ereignisse erwartet, dass die Sammlungen auf eine bestimmte Weise organisiert werden. Weitere Informationen finden Sie unter Hinzufügen von Videos zu einer virtuellen Ereigniserfahrung.

Hinzufügen der Konfigurationsdatei zur Vorlage

Für die Virtual Event Portal Experience muss eine Konfigurationsdatei in der Site enthalten sein. Die Konfigurationsdatei wird der Vorlage als Teil des Erfahrungsheaders hinzugefügt und steuert einige der allgemeinen Vorlagenoptionen und wird verwendet, um mehrere Abschnitte der Site zu konfigurieren und anzupassen. Brightcove empfiehlt, die Konfigurationsdatei direkt nach dem Erstellen einer neuen Virtual Event Portal Experience hinzuzufügen. Weitere Informationen finden Sie unter Hinzufügen der Konfigurationsdatei zur Erfahrung.

Layout anpassen

Der Gallery Site Editor wird verwendet, um das Layout von Seiten anzupassen. Ein Logo, Menü-Links, eine Begrüßungsnachricht und die Seitenfußzeile können mit dem Gallery Site Editor angepasst werden. Weitere Informationen zum Anpassen des Layouts finden Sie unter Anpassen des Inhalts von Virtual Event Experience-Seiten.

Veranstaltungsagenda konfigurieren

Die Veranstaltungsagenda listet den Zeitplan von Live-Events und anderen Sitzungen auf. Wenn eine Agenda erstellt und veröffentlicht wurde, erscheint sie auf der Veranstaltungshomepage. Informationen zum Erstellen der Veranstaltungsagenda finden Sie unter Verwaltung der Veranstaltungsagenda.

Verwaltung der Veranstaltungsredner

Der Bereich Speakers wird automatisch mit allen Event-Speakern gefüllt. Alle Sprecher der Sitzung werden im Abschnitt Sprecher angezeigt. Informationen zur Verwaltung der Veranstaltungsredner finden Sie unter Veranstaltungsredner verwalten.

Anpassen der Event-Sponsoren

Im Bereich Sponsoren werden die Sponsoren der Veranstaltung hervorgehoben. Weitere Informationen zum Anpassen der Event-Sponsoren finden Sie unter Anpassen der Event-Sponsoren.
