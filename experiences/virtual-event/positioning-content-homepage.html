Positionieren von Inhalten auf der Startseite

{{ page.description }}

Einige Objekte in der Konfigurationsdatei verfügen über Parameter, mit denen Sie festlegen können, wo das Objekt auf der Seite angezeigt werden soll. Beispielsweise können statische Inhalte, Sprecher, Titel und Sponsoren auf der Seite neu positioniert werden. Objekte, die positioniert werden können, haben die folgenden Attribute in der Konfigurationsdatei:

  • insert_target- Ein CSS-Selektor (eine einfache Klasse oder ID), der als Ausgangspunkt für das Rendern des angegebenen Objekts verwendet wird
  • insert_mode- Wie der Inhalt eingefügt wird
    • after
    • before
    • append
    • prepend

Ziel bestimmen

Das insert_target Der Parameter erwartet einen CSS-Selektor (unter Verwendung eines einfachen Klassen- oder ID-Formats). Das mit diesem Selektor übereinstimmende Element wird als Ausgangspunkt für das Rendern des angegebenen Objekts verwendet. Der beste Weg, um den zu verwendenden CSS-Selektor oder die zu verwendende ID zu bestimmen, besteht darin, zur Startseite für virtuelle Ereignisse zu navigieren und dann die Browser-Entwicklertools zu verwenden, um die Elemente auf der Seite zu überprüfen.

Sie werden feststellen, dass die Hauptabschnitte der Homepage mit organisiert sind <div> Stichworte.

Auswählen des Einfügemodus

Nachdem Sie festgelegt haben, wo in der Hierarchie das Objekt eingefügt werden soll, wird die insert_mode Das Attribut bestimmt, wie das Objekt in die Hierarchie eingefügt wird.

  • after- Setzt das Objekt nach dem angegebenen div auf demselben Level
  • before- Setzt das Objekt vor das angegebene div auf demselben Level
  • append- Setzt das Objekt in das angegebene Objekt ein div als letztes Kind
  • prepend- Setzt das Objekt in das angegebene Objekt ein div als erstes Kind

Beispiel für die Objektpositionierung

Die folgenden Codebeispiele zeigen, wie Sie mit einem statischen Inhaltsobjekt einen Titel über der sekundären Sammlung einfügen.


{
    "insert_target": ".bc-on-demand-section",
    "insert_mode": "prepend",
    "content": "<h2 class='bc-on-demand-section-title'>ON-DEMAND SESIONS</h2>",
    "require_roles": [],
    "omit_roles": []
}

Mit den Browser-Tools können Sie die sehen h2 wurde als erstes Element in die eingefügt bc-on-demand-section div.
