Anpassen der Event-Sponsoren

{{ page.description }}

Sponsorbilder können zur Homepage des virtuellen Ereignisses hinzugefügt werden.

Die Sponsoreninformationen werden durch Bearbeiten der angepasst sponsors Objekt in der Konfigurationsdatei.

  • name- Sponsorname
  • image_url- Sponsor-Logo, die Größe der Bilder wird auf 100 Pixel geändert
  • link_url- Ziel-URL für das Bild

Das sponsors Das Objekt in der Konfigurationsdatei verfügt auch über Parameter, mit denen gesteuert werden kann, wo auf der Startseite die Sponsoreninformationen angezeigt werden. Informationen zur Positionierung des Sponsorenbereichs finden Sie unter Positionieren von Inhalten auf der Startseite.

Entfernen der Deckkraft und Größe der Logo-Bilder

Standardmäßig werden Logo-Bilder mit einer Deckkraft von 80% gestaltet. Der folgende CSS-Stil kann verwendet werden, um die Deckkraft auf 100% festzulegen. Das maximale Breite Mit der Eigenschaft kann die Größe der Bilder geändert werden.


<style>
.bc-sponsors-section .bc-sponsors-list img {
    -webkit-filter: none;
    opacity: 1;
    max-width: 200;
}
</style>
Bearbeiten Sie das Erlebnis und fügen Sie das CSS oben im Feld Benutzerdefiniertes Header-HTML hinzu.
