Erste Schritte mit dem Agenda-Modul

{{ page.description }}

Um auf das Agenda-Modul zuzugreifen, klicken Sie auf ein virtuelles Ereigniserlebnis, um den Galerie-Site-Editor zu öffnen, und klicken Sie dann auf SITE-FUNKTIONEN> Agenda.

Von der Homepage aus können Sie:

  • Laden Sie eine Veranstaltungsagenda hoch
  • Verwalten Sie die Veranstaltungsagenda
  • Veranstaltungsredner verwalten
Um vom Agenda-Modul zum Galerie-Site-Editor zurückzukehren, klicken Sie auf das Zurück zu Meine Veranstaltung Link auf der linken Seite der Seite.

Hochladen einer Veranstaltungsagenda

Um zunächst eine Veranstaltungsagenda zu erstellen, muss eine Agenda-.csv-Datei hochgeladen werden. Informationen zum Hochladen einer Veranstaltungsagenda finden Sie unter Hochladen einer Agenda-Datei.

Verwaltung der Veranstaltungsagenda

Um die Agenda der Veranstaltung zu bearbeiten, klicken Sie darauf und nehmen Sie die gewünschten Änderungen vor. Informationen finden Sie unter Verwaltung der Veranstaltungsagenda.

Verwaltung der Veranstaltungsredner

Um die einer Veranstaltung zugeordneten Referenten zu verwalten, klicken Sie auf Lautsprecher in der linken Navigation. Informationen zum Verwalten von Lautsprechern finden Sie unter Veranstaltungsredner verwalten.
