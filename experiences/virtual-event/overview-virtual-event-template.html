Übersicht über die Vorlage für virtuelle Ereignisse

{{ page.description }}

Header

Die Kopfzeile wird oben auf allen Seiten angezeigt. Der Header besteht aus Werten in der Konfigurationsdatei. Der Header besteht aus folgenden Komponenten:

  • Header-Logo
  • Links
  • Suche

Informationen zum Anpassen des Headers finden Sie unter Anpassen des Inhalts von Virtual Event Experience-Seiten.

Willkommensnachricht

Die Begrüßungsnachricht kann verwendet werden, um Besucher auf der Website willkommen zu heißen. Sie kann Text und ein Bild enthalten. Benutzerdefiniertes HTML kann für erweiterte Nachrichten verwendet werden. Die Begrüßungsnachricht in diesem Beispiel wurde aus einer Kombination von Text und Bildern erstellt.

Informationen zum Hinzufügen einer Begrüßungsnachricht finden Sie unter Anpassen des Inhalts von Virtual Event Experience-Seiten.

Ausgewählte Sammlung

Die vorgestellte Sammlung ist die erste Videosammlung, die in der Vorlage angezeigt wird. Die vorgestellte Sammlung wird normalerweise verwendet, um Live-Videos zu präsentieren. Navigationspfeile stehen zur Verfügung, damit Benutzer durch die vorgestellten Videos navigieren können. Jedem Video kann auch ein Q & A-Widget zugeordnet werden. Informationen zum Erstellen von Sammlungen und zum Hinzufügen von Videos finden Sie unter Hinzufügen von Videos zu einer virtuellen Ereigniserfahrung.

Hinweis: Die Featured Collection unterstützt maximal 24 Videos.

Primärsammlung

Die primäre Sammlung ist die zweite Videosammlung, die in der Vorlage angezeigt wird. Navigationspfeile sind verfügbar, damit Benutzer durch die Videos navigieren können. Informationen zum Erstellen von Sammlungen und zum Hinzufügen von Videos finden Sie unter Hinzufügen von Videos zu einer virtuellen Ereigniserfahrung.

Hinweis: Die Primär-Sammlung unterstützt maximal 24 Videos.

Agenda

Die Agenda wird normalerweise verwendet, um den Zeitplan für Live-Events aufzulisten. Die Agenda zeigt den Sitzungsnamen, die Sprecher sowie das Datum und die Uhrzeit der Sitzung an. Neben der Sitzung wird ein Countdown-Timer angezeigt, und wenn die Sitzung aktiv ist, wird eine Schaltfläche "Anzeigen" angezeigt. Informationen zum Erstellen der Veranstaltungsagenda finden Sie unter Verwaltung der Veranstaltungsagenda.

Sprecher

Der Bereich Speakers wird automatisch mit allen Event-Speakern gefüllt. Alle Sprecher der Sitzung werden im Abschnitt Sprecher angezeigt. Der Abschnitt Lautsprecher zeigt Folgendes an:

  • Lautsprecherbild
  • Name
  • Titel
  • Gesellschaft
  • Kurzbiografie

Informationen zur Verwaltung der Veranstaltungsredner finden Sie unter Veranstaltungsredner verwalten.

Sekundärsammlung

Die sekundäre Sammlung ist die dritte Videosammlung, die in der Vorlage angezeigt wird. Navigationspfeile sind verfügbar, damit Benutzer durch die Videos navigieren können. Informationen zum Erstellen von Sammlungen und zum Hinzufügen von Videos finden Sie unter Hinzufügen von Videos zu einer virtuellen Ereigniserfahrung.

Hinweis: Die Secondary Collection unterstützt maximal 24 Videos.

Spuren

Der Tracks-Bereich besteht aus einer Reihe von Sammlungen, die normalerweise zum Anzeigen von VOD-Videos verwendet werden. Durch Klicken auf einen Titel werden die Videos in diesem Titel angezeigt. Informationen zum Erstellen von Sammlungen und zum Hinzufügen von Videos finden Sie unter Hinzufügen von Videos zu einer virtuellen Ereigniserfahrung.

Statischer Inhalt

Das Raster "Weitere Informationen" wird mithilfe eines statischen Textobjekts in der Konfigurationsdatei erstellt. Statische Textobjekte werden verwendet, um benutzerdefiniertes HTML in die Vorlage einzufügen.

Statischer Inhalt kann ein beliebiger HTML-Code sein und an einer beliebigen Stelle in der Vorlage eingefügt werden. Informationen zum Arbeiten mit statischen Inhaltsobjekten finden Sie unter Arbeiten mit statischen Inhalten.

Sponsoren

Im Bereich Sponsoren können die Sponsoren der Veranstaltung hervorgehoben werden. Für jeden Sponsor können Sie Folgendes anzeigen:

  • Sponsor-Logo
  • Link zur Webseite des Sponsors

Informationen zum Anpassen des Sponsorenbereichs finden Sie unter Anpassen der Event-Sponsoren.

Fusszeile

Die Fußzeile wird am Ende aller Seiten angezeigt. Die Fußzeile besteht aus Werten in der Konfigurationsdatei. Die Fußzeile besteht aus folgenden Komponenten:

  • Primärer Beschriftungsbereich
  • Rechtstext und Urheberrecht
  • Drei Spalten mit Links
  • Soziale Symbole
