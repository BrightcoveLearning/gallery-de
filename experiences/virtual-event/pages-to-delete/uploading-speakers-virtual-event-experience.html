Hochladen einer Lautsprecherdatei

In diesem Thema erfahren Sie, wie Sie eine externe Liste von Sprechern hochladen, um zunächst die Sprecherliste für eine virtuelle Ereigniserfahrung zu füllen.

Möglicherweise haben Sie eine Liste von Sprechern in Microsoft Excel oder einer anderen Tabellenkalkulationsanwendung erstellt. Das Agenda-Modul unterstützt das Hochladen einer externen CSV-Sprecherdatei, sodass nicht alle Sprecher manuell in das Agenda-Modul eingegeben werden müssen.

Hinweis: Brightcove empfiehlt, die Lautsprecherdatei hochzuladen, bevor Sie eine externe Agenda-Datei hochladen.

Dateianforderungen importieren

Um Ihre Sprecherliste in das Agenda-Modul zu importieren, sollte die Liste in einer Tabellenkalkulationsanwendung (z. B. Microsoft Excel, Apple Numbers oder Google Sheets) erstellt worden sein. Mit diesen Anwendungen können die Daten in eine CSV-Datei exportiert werden. Das Agenda-Modul kann nur eine CSV-Datei importieren.

Das Agenda-Modul sucht nach bestimmten Spaltennamen. Diese Spalten werden den entsprechenden Feldern im Agenda-Modul zugeordnet. In der CSV-Datei Ihrer Lautsprecher sollten mindestens die folgenden Felder vorhanden sein:

  • last-name- Vorname des Sprechers
  • first-name- Nachname des Sprechers
  • email- E-Mail-Adresse des Sprechers

Die folgenden Spalten sind optional. Wenn sie jedoch in Ihrer CSV-Datei vorhanden sind, werden sie auch in das Agenda-Modul importiert:

  • company- Sprecherfirma
  • title- Sprechertitel
  • bio- Sprecher Bio
  • social-url- Soziale URL des Sprechers (dh LinkedIn-URL)
  • homepage-url- Persönliche Website des Sprechers
