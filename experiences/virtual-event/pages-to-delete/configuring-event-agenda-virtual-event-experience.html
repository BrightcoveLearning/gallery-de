Event Agenda konfigurieren

In diesem Thema erfahren Sie, wie Sie die Veranstaltungsagenda einer virtuellen Ereigniserfahrung anpassen.

Die Veranstaltungsagenda wird normalerweise verwendet, um die geplanten Live-Ereignisse anzuzeigen, die in der vorgestellten Sammlung angezeigt werden. Die Schaltfläche, die bei jeder Sitzung angezeigt wird, hängt vom Status der Sitzung ab:

  • Vor der Veranstaltung - Es wird keine Schaltfläche angezeigt, der Countdown-Timer wird angezeigt
  • Während - JETZT LEBEN, wenn die Veranstaltung läuft
  • Nach dem Ereignis - Die Schaltfläche zeigt VIEW SESSION für Ereignisse in der Vergangenheit an.

Basierend auf der Anzahl der Tage auf der Tagesordnung, a Show Das Dropdown-Menü wird angezeigt, in dem der Betrachter den anzuzeigenden Tagesplan auswählen kann.

Die Agenda-Informationen werden durch Bearbeiten der angepasst schedule Objekt in der Konfigurationsdatei.

  • name- Sitzungsname
  • description- Sitzungsbeschreibung - Wird derzeit nicht auf der Seite angezeigt, kann jedoch mit benutzerdefiniertem JavaScript angezeigt werden
  • starts_at- Datum und Uhrzeit des Sitzungsstarts
  • ends_at- Datum und Uhrzeit des Sitzungsendes
  • live_id- Nicht benutzt
  • vod_id- Video ID
  • speakers- Informationen zum Sitzungssprecher
  • image_url- Sitzungsbild. Die Standardbildgröße beträgt 320 x 180 Pixel. Bilder werden auf diese Größe zugeschnitten.

Das schedule Objekt in der Konfigurationsdatei hat auch Parameter für die title Dies wird über dem Zeitplan angezeigt und um festzulegen, ob der Zeitplan in einem angezeigt wird expanded Modus.
