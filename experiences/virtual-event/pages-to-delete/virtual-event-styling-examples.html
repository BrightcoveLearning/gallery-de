Styling-Beispiele für virtuelle Events

Dieses Thema enthält Beispielcode, mit dem die Vorlage für virtuelle Ereignisse angepasst werden kann.

Social-Sharing-Symbole

Standardmäßig, wenn die Video-Sharing-Einstellungen sind für die Erfahrung aktiviert, werden sie nicht neben Videos auf den Videodetailseiten angezeigt. Das folgende CSS zeigt die sozialen Symbole auf den Videodetailseiten an.


.share-list {
  display: block !important;
}
.share-list a {
  display: inline-block !important;
}
.share-list li {
  display: block !important;
}
