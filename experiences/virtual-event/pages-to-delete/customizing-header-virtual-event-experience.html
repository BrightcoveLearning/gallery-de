Anpassen des Headers einer virtuellen Veranstaltungserfahrung

In diesem Thema erfahren Sie, wie Sie den Header von Virtual Event Experience-Seiten anpassen.

Die Kopfzeile wird oben auf allen Seiten angezeigt. Der Header besteht aus Werten in der Konfigurationsdatei. Der Header besteht aus folgenden Komponenten:

  • Header-Logo
  • Links
  • Soziale Symbole
  • Suche

Header-Logo

Das Header-Bild wird durch das angegeben images Objekt in der Konfigurationsdatei.

  • header_logo- URL zum Bild, die in der Kopfzeile angezeigt werden soll

.Png- und .jpg-Dateien werden unterstützt. Das Header-Logo wird proportional auf 178 Pixel Breite verkleinert.

Links

Standardmäßig ist die HEIMAT , LIVEÜBERTRAGUNG und SITZUNGEN AUF ANFRAGE Links sind in der Vorlage fest codiert:

  • ZUHAUSE - Geht nach Hause, oben auf der Seite
  • LIVEÜBERTRAGUNG - Geht zur vorgestellten Sammlung (Top-Video-Player)
  • ON-DEMAND-SESSIONEN - Geht zum Abschnitt Tracks

Zusätzliche Links können durch Bearbeiten der hinzugefügt werden header_links Objekt in der Konfigurationsdatei.

  • link_url- Ziel-URL
  • label- Beschriftung, die in der Kopfzeile angezeigt werden soll

Soziale Symbole

Standardmäßig werden in der Kopfzeile soziale Symbole angezeigt:

  • Facebook
  • Instagram
  • Twitter
  • LinkedIn
  • Youtube
Die sozialen Symbole, die in der Kopfzeile angezeigt werden, können durch Bearbeiten von bearbeitet / entfernt werden social Objekt in der Konfigurationsdatei. Weitere Informationen finden Sie unter Konfigurieren der sozialen Symbole für eine virtuelle Ereigniserfahrung.

Suche

Um die Suche zu aktivieren und das Suchfeld anzuzeigen, Aktivieren Sie die Suche nach der Erfahrung. Standardmäßig wird im Suchfeld in der Kopfzeile eine einzelne Zeile für die Texteingabe angezeigt.

Das Suchfeld im Beispiel wurde so gestaltet, dass ein Feld um das Suchfeld angezeigt wird.

Header-spezifisches Styling

Links auf Mobilgeräten

Standardmäßig werden dem Header hinzugefügte Links bei Verwendung eines Mobilgeräts nicht angezeigt. Das folgende JavaScript behebt dieses Problem.


for (let i = 0, l = ciabConfig.header_links.length; i < l; i++) {
  let link = ciabConfig.header_links[i];
  let li = document.createElement('li');
  let ul = document.getElementById('main-nav-cat');
  li.className = 'main-nav-category main-nav-category-home';
  li.innerHTML = ('' + link.label + ''); ul.appendChild(li);
}

Formatieren des Suchfelds
