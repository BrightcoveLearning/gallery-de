Anpassen der Veranstaltungslautsprecher

In diesem Thema erfahren Sie, wie Sie die Ereignislautsprecher anpassen, die auf der Virtual Event Experience-Homepage angezeigt werden.

Sprecherbilder und eine kurze Biografie können zur Homepage des virtuellen Ereignisses hinzugefügt werden.

Die Sprecherinformationen werden durch Bearbeiten der angepasst speakers Objekt in der Konfigurationsdatei.

  • name- Name des Sprechers
  • description- Lautsprecherinformationen, die unter ihrem Bild angezeigt werden. Dies ist auf ungefähr 3 Zeilen mit 15 Wörtern begrenzt.
  • image_url- Lautsprecherbild. Empfehlen Sie eine maximale Breite von 200px. Die Größe der Bilder wird nicht geändert
  • link_url- (optional) Ziel-URL für das Bild (kann beispielsweise auf die LinkedIn-Seite des Sprechers verweisen)

Standardmäßig konfiguriert die CSS-Vorlage die Sprecherbilder mit einem border-radius von 50%, wodurch die Bilder kreisförmig werden.

Das speakers Das Objekt in der Konfigurationsdatei verfügt auch über Parameter, mit denen gesteuert werden kann, wo auf der Startseite die Sprecherinformationen angezeigt werden. Informationen zum Positionieren der Lautsprecher finden Sie unter Positionieren von Inhalten auf der Startseite.
