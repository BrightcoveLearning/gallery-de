Anpassen der Fußzeile einer virtuellen Veranstaltungserfahrung

In diesem Thema erfahren Sie, wie Sie die Fußzeile der Virtual Event Experience-Seiten anpassen.

Die Fußzeile wird am Ende aller Seiten angezeigt. Die Fußzeile besteht aus Werten in der Konfigurationsdatei. Jedem Fußzeilenelement in der Konfigurationsdatei ist Folgendes zugeordnet:

  • URL verknüpfen
  • Etikette
  • Gruppe

Gruppen werden verwendet, um zu steuern, wo die Beschriftung / der Link in der Fußzeile angezeigt wird. Es stehen vier Gruppen zur Verfügung:

  • Gruppe 0 - Wird für HTML-Beschriftungen verwendet
  • Gruppe 1 - Wird für Rechtstext- und Urheberrechtsinformationen verwendet
  • Gruppen 2 & 3 - Wird für Linklisten verwendet (empfohlen, maximal 5 Links zu verwenden)

Die sozialen Symbole, die in der Fußzeile angezeigt werden, können durch Bearbeiten von bearbeitet / entfernt werden social Objekt in der Konfigurationsdatei. Weitere Informationen finden Sie unter Konfigurieren der sozialen Symbole für eine virtuelle Ereigniserfahrung.

Hinzufügen von Links zur Fußzeile

Fußzeilenlinks werden durch Bearbeiten der konfiguriert footer_links Objekt in der Konfigurationsdatei.

  • label- Beschriftung, die in der Fußzeile angezeigt werden soll
