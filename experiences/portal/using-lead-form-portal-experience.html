Verwenden eines Lead-Formulars mit Portal-Erfahrung

{{ page.description }}

Lead-Formulare können verwendet werden, um Zuschauerinformationen zu erfassen, während Zuschauer Videos ansehen, und können mit den folgenden Arten von Erfahrungen verwendet werden:

  • Portal
  • Live-Event-Portal
  • Landingpage

Mit Portal, Live Event Portal und einigen Landing Page-Erlebnissen werden Lead-Formulare im Player angezeigt. Lead-Formulare können so konfiguriert werden, dass sie angezeigt werden, nachdem eine bestimmte Anzahl von Videos angesehen wurde oder nachdem ein vorbestimmtes Zeitintervall verstrichen ist. Lead-Daten können in durch Kommas getrennten oder Excel-Dateiformaten heruntergeladen werden.

Die Schritte zum Verwenden eines Lead-Formulars sind wie folgt:

  • Erstellen Sie mit dem Zielgruppenmodul ein Lead-Formular
  • Weisen Sie das Lead-Formular einem Audience-fähigen Player zu
  • Konfigurieren Sie eine Erfahrung für die Verwendung des Lead-Formulars
  • Exportieren Sie die Lead-Daten

Erstellen eines Lead-Formulars

Lead-Formulare werden mit dem Audience-Modul erstellt. Vollständige Schritte zum Erstellen eines Lead-Formulars mit allen verfügbaren Optionen finden Sie unter Erstellen eines Zielgruppen-Lead-Formulars. Gehen Sie folgendermaßen vor, um ein Lead-Formular zu erstellen:

  1. Öffnen Sie das Zielgruppenmodul.
  2. Klicken Bleiformulare in der linken Navigation.
  3. Klicken Lead-Formular erstellen.
  4. Geben Sie ein Name für die Bleiform.
  5. Konfigurieren Sie die Lead-Formulareinstellungen und klicken Sie dann auf speichern.
Hinweis: Für die eingegebenen Lead-Formulardaten wird keine Validierung durchgeführt.
Hinweis: Wenn ein Lead-Formular für eine Landing-Page-Erfahrung verwendet wird und das Erlebnis so konfiguriert ist, dass das Formular im Hauptteil der Seite angezeigt wird, werden die Einstellungen für das Timing und die Anzeigeeinstellungen für das Leadformular ignoriert.

Zuweisen eines Lead-Formulars zu einem Audience-fähigen Player

Lead-Formulare können nur in Audience-fähigen Playern angezeigt werden. Ein Audience-fähiger Player ist ein Brightcove-Player, der mit dem Audience-Plugin konfiguriert wird. Audience-fähige Player zeichnen Video-Engagement-Daten auf, die im Audience-Modul angezeigt werden können. Gehen Sie folgendermaßen vor, um einen Audience-fähigen Player zu erstellen:

  1. Öffnen Sie das Zielgruppenmodul.
  2. Klicken Publikum Spieler.
  3. Klicken Spieler erstellen um einen neuen Audience-fähigen Player zu erstellen.
    Hinweis: Es ist auch möglich, das Audience-Plugin einem vorhandenen Player hinzuzufügen.
  4. Suchen Sie Ihren Player und verwenden Sie die Lead Form Dropdown-Liste zur Auswahl des anzuzeigenden Lead-Formulars.
  5. (Optional) Wenn Sie Audience zum Verfolgen von Video-Engagement-Daten verwenden, sollte der Player über das Internet mit einer Datenverbindung konfiguriert werden Verbunden mit Dropdown-Liste.

Konfigurieren einer Erfahrung zur Verwendung eines Lead-Formulars

Nachdem ein Lead-Formular erstellt und einem Audience-fähigen Player zugewiesen wurde, kann es in einer Portal-Erfahrung verwendet werden. Gehen Sie folgendermaßen vor, um eine Erfahrung für die Verwendung eines Lead-Formulars zu konfigurieren:

  1. Öffnen Sie das Galerie-Modul.
  2. Bearbeiten Sie die Erfahrung, zu der Sie das Lead-Formular hinzufügen möchten.
  3. Klicken VIDEO UND PLAYBACK> Spieler- und Lead-Formular in der linken Navigation.
  4. Wählen Sie einen Audience-fähigen Player aus, der mit dem anzuzeigenden Lead-Formular konfiguriert ist.
  5. Nachdem ein Spieler ausgewählt wurde, wird der Lead Form und Verbindung Informationen werden angezeigt. Bestätigen Sie, dass dies korrekt ist.
    Hinweis: Wenn ein Landing Page-Erlebnis so konfiguriert ist, dass ein Lead-Formular im Hauptteil der Seite angezeigt wird, werden die Timing- und Anzeigeeinstellungen für das Leadformular ignoriert.
  6. Klicken speichern.

Stellen Sie sicher, dass Veröffentlichen Sie die Erfahrung erneut nach Änderungen.

Lead-Daten exportieren

Alle aus Galerie-Lead-Formularen gesammelten Lead-Daten werden mithilfe des Audience-Moduls exportiert. Sehen Exportieren von Galerie-Lead-Daten Informationen zum Exportieren der Galerie-Lead-Daten.
