    Verwenden eines Lead-Formulars mit einem Portal Experience

    In diesem Thema erfahren Sie, wie Sie ein Audience Lead-Formular für die Verwendung in einem Portal Experience konfigurieren.

    Lead-Formulare können verwendet werden, um Zuschauerinformationen zu erfassen, während die Zuschauer Videos ansehen, und können mit den folgenden Arten von Erlebnissen verwendet werden:

    • Portal
    • Live-Eventportal
    • Landing Page

    Mit Portal, Live Event Portal und einigen Landing Page-Erlebnissen werden Lead-Formulare im Player angezeigt. Lead-Formulare können so konfiguriert werden, dass sie angezeigt werden, nachdem eine bestimmte Anzahl von Videos angesehen wurde oder nachdem ein vorgegebenes Zeitintervall abgelaufen ist.     Lead-Daten können in kommagetrennten oder Excel-Dateiformaten heruntergeladen werden.

    Die Schritte zur Verwendung eines Lead-Formulars sind wie folgt:

    • Erstellen Sie ein Lead-Formular mit dem Audience-Modul
    • Weisen Sie das Lead-Formular einem Zielgruppe-aktivierten Spieler zu
    • Konfigurieren Sie ein Erlebnis für die Verwendung des Lead-Formulars
    • Exportieren der Lead-Daten

    Lead-Formular erstellen

    Lead-Formulare werden mit dem Audience-Modul erstellt. Vollständige Schritte zum Erstellen eines Lead-Formulars einschließlich aller verfügbaren Optionen finden Sie unter Erstellen eines Audience Lead-Formulars. Gehen Sie folgendermaßen vor, um ein Lead-Formular zu erstellen:

    1. Öffnen Sie das Audience-Modul.
    2. Klicken Sie in der linken Navigationsleiste auf Lead Forms .
    3. Klicken Sie auf Create Lead Form.
    4. Geben Sie einen Namen für das Lead-Formular ein.
    5. Konfigurieren Sie die Lead-Formulareinstellungen und klicken Sie dann auf Speichern.
    Hinweis: Es wird keine Validierung der eingegebenen Lead-Formulardaten durchgeführt.
    Hinweis: Wenn ein Lead-Formular für ein Landing-Page-Erlebnis verwendet wird und das Erlebnis so konfiguriert ist, dass das Formular im Hauptteil der Seite angezeigt wird, werden das Timing des Lead-Formulars und die Anzeigeeinstellungen ignoriert.

    Zuweisen eines Lead-Formulars an einen zielgruppenfähigen Spieler

    Lead-Formulare können nur in Audience-aktivierten Spielern angezeigt werden. Ein Audience aktivierter Player ist ein Brightcove Player, der mit dem Audience-Plugin konfiguriert ist. Audienzaktivierte Spieler zeichnen Videointeraktionsdaten auf, die innerhalb des Audience-Moduls angezeigt werden können. Um einen Audience aktivierten Player zu erstellen, folge diesen Schritten:

    1. Öffnen Sie das Audience-Modul.
    2. Klicken Sie auf Zielgruppenspieler
    3. Klicken Sie auf Player erstellen , um einen neuen, auf Zielgruppen aktivierten Player zu erstellen.
      Hinweis: Es ist auch möglich, das Audience-Plugin einem bestehenden Player hinzuzufügen.
    4. Suchen Sie Ihren Spieler und wählen Sie mithilfe der Dropdown-Liste Lead-Formular das anzuzeigende Lead-Formular aus.
    5. (Optional) Wenn Sie Audience verwenden, um Videointeraktionsdaten zu verfolgen, sollte der Player mithilfe der Dropdown-Liste Verbinden mit einer Datenverbindung konfiguriert werden.

    Konfigurieren eines Erlebnisses zur Verwendung eines Lead-Formulars

    Nachdem ein Lead-Formular erstellt und einem Zielgruppenaktivierten Player zugewiesen wurde, kann es in einem Portal Experience verwendet werden. Gehen Sie folgendermaßen vor, um ein Erlebnis für die Verwendung eines Lead-Formulars zu konfigurieren:

    1. Öffnen Sie das Gallery-Modul.
    2. Bearbeiten Sie das Erlebnis, zu dem Sie das Lead-Formular hinzufügen möchten.
    3. Klicken Sie in der linken Navigationsleiste auf VIDEO UND > PLAYBACK Player und Lead Form .
    4. Wählen Sie einen für Zielgruppen aktivierten Player aus, der mit dem anzuzeigenden Lead-Formular konfiguriert ist.
    5. Nachdem ein Spieler ausgewählt wurde, werden das Lead-Formular und die Verbindungsinformationen angezeigt. Bestätige, dass das richtig ist.
      Hinweis: Wenn ein Landing Page-Erlebnis so konfiguriert ist, dass ein Lead-Formular im Hauptteil der Seite angezeigt wird, werden das Timing des Lead-Formulars und die Anzeigeeinstellungen ignoriert.
    6. Klicken Sie auf Save.

    Stellen Sie sicher, dass Sie das Erlebnis nach Änderungen erneut veröffentlichen .
