Steuern des Zugriffs auf ein Portalerlebnis

{{ page.description }}

Zugriffssteuerungsprofile werden verwendet, um den Zugriff auf Portal-Erlebnisse zu steuern. Zugriffssteuerungsprofile werden als Teil der Galerieeinstellungen erstellt und können den Zugriff auf eine Site einschränken durch:

  • Einschränkung des Zugriffs basierend auf der IP-Adresse
  • Aufforderung an den Benutzer zur Eingabe von Anmeldeinformationen (SSO)
  • Der Benutzer muss einen Zugangscode eingeben

Informationen zum Erstellen von Zugriffssteuerungsprofilen finden Sie unter Konfigurieren von Zugriffssteuerungsprofilen. Sobald Profile erstellt wurden, können sie Portal Experiences zugewiesen werden, um den Zugriff zu steuern. Gehen Sie folgendermaßen vor, um einer Site ein Zugriffssteuerungsprofil zuzuweisen:

  1. Bearbeiten Sie Ihre Portal-Erfahrung.
  2. Drücke den STANDORTKONFIGURATION> Zugriffskontrolle in der linken Navigation.
  3. Wählen Sie ein Zugriffssteuerungsprofil für die Site aus. Standardmäßig sind alle Websites öffentlich.
  4. Klicken speichern.

Stellen Sie sicher, dass Sie die Site erneut veröffentlichen, nachdem Sie Änderungen an den Zugriffssteuerungseinstellungen vorgenommen haben.
