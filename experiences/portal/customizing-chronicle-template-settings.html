Anpassen der Chronikvorlageneinstellungen

{{ page.description }}

Wenn eine Portalerfahrung mithilfe der Chronikvorlage erstellt wird, können mehrere vorlagenspezifische Einstellungen konfiguriert werden. Um die Einstellungen für die Chronikvorlage zu konfigurieren, bearbeiten Sie die Erfahrung und klicken Sie auf ERSCHEINUNGSBILD UND VERHALTEN> Vorlageneinstellungen in der linken Navigation.

Stellen Sie einen Wert für die ein Sequenzetikettenformat. Dieser Wert wird neben den Videos auf der Website angezeigt:

  • Keiner - Etiketten werden nicht angezeigt
  • Datum - Das Datum, an dem das Video zur Video Cloud hinzugefügt wurde, wird angezeigt
  • Benutzerdefinierte Feld - Der Wert für das angegebene benutzerdefinierte Feld wird angezeigt
