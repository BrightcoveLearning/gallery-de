Exportieren von Galerie-Lead-Daten

{{ page.description }}

Daten zum Galerie-Lead-Formular werden erfasst, wenn eine Galerie-Erfahrung für die Verwendung eines Lead-Formulars konfiguriert wurde. Informationen zum Konfigurieren einer Portal Experience zum Sammeln von Lead-Daten finden Sie unter Verwenden eines Lead-Formulars mit Portal-Erfahrung.

Beim Exportieren von Galerie-Leads aus dem Audience-Modul werden nur Daten exportiert, die mit Audience-Lead-Formularen erfasst wurden. Lead-Daten aus benutzerdefinierten Lead-Formularen werden in Ihrer Marketing-Automatisierungssoftware gespeichert.

Hinweis: Nur Benutzer mit Administratorrechten können die Lead-Daten herunterladen.

Führen Sie die folgenden Schritte aus, um Galerie-Lead-Daten zu exportieren.

  1. Öffne das Publikum Modul.
  2. Klicken Galerie-Leads exportieren in der linken Navigation.
  3. Wählen Sie eine Galerieerfahrung aus, aus der die Leads exportiert werden sollen.
  4. Geben Sie ein Anfangsdatum und Endtermin für die führt.
  5. Wählen Sie ein Exportformat, Komma-getrennte Werte oder Excel.
  6. Drücke den Export Schaltfläche zum Herunterladen der Lead-Daten.

Die folgenden Felder werden in die exportierte Datei aufgenommen:

  • Alle möglichen Lead-Formularfelder
    • EmailAddress
    • Vorname
    • LastName
    • BusinessPhone
    • Land
    • CompanyName
    • Industrie
  • externalId - ID zur eindeutigen Identifizierung des Betrachters
  • playerId - Spieler-ID, die das Lead-Formular anzeigt
  • videoId - Video-ID des Videos, das beim Sammeln von Daten angesehen wird
  • pageUrl - Seiten-URL-Viewer war eingeschaltet, als Daten gesammelt wurden
  • hergestellt in - Datums- und Uhrzeitdaten wurden gesammelt
  • experience - Name der Erfahrung, die bei der Datenerfassung angezeigt wird
