Konfigurieren der Details für ein Portalerlebnis

{{ page.description }}

Der Name, die Beschreibung und die Sprache der Erfahrung können als Teil der Übersichtseinstellungen konfiguriert werden. Um die Übersichtseinstellungen zu konfigurieren, bearbeiten Sie die Erfahrung und klicken Sie dann auf ÜBERBLICK in der linken Navigation.

Die Übersichtsseite wird angezeigt:

  • Erfahrungstyp - Zeigt die Art der Erfahrung und die verwendete Vorlage an
  • Erleben Sie Titel und Beschreibung - Wenn Sie zum ersten Mal eine neue Erfahrung erstellen, werden Sie aufgefordert, die einzugeben Titel erleben und Erfahrungsbeschreibung. Diese Werte können hier bearbeitet werden. Das Titel erleben wird beim Öffnen der Site auf der Registerkarte Browser angezeigt.
  • Sprache erleben - Setzt den galeriespezifischen Text auf der Site auf diese Sprache. Die Sprachunterstützung hängt von der verwendeten Vorlage ab.

Wenn Sie fertig sind, klicken Sie auf speichern um Ihre Änderungen zu speichern.

Hinweis: Stellen Sie nach Änderungen an einem Erlebnis sicher, dass Sie das Erlebnis erneut veröffentlichen, damit die Änderungen auf Websites angezeigt werden, auf denen das Erlebnis angezeigt wird.
