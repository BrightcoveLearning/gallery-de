Anpassen der Katalogvorlageneinstellungen

{{ page.description }}

Wenn eine Portalerfahrung mithilfe der Katalogvorlage erstellt wird, können mehrere vorlagenspezifische Einstellungen konfiguriert werden. Um die Einstellungen für die Katalogvorlage zu konfigurieren, bearbeiten Sie die Erfahrung und klicken Sie auf ERSCHEINUNGSBILD UND VERHALTEN> Vorlageneinstellungen in der linken Navigation.

Folgende Einstellungen können konfiguriert werden:

  • Primäre Navigation
  • Feature Karussell
  • Karussell-Paging-Tasten
  • Video-Miniaturansichten
  • Thumbnail-Wiedergabesymbole
  • Videobeschreibung

Primäre Navigation

Das Primäre Navigation Die Einstellung steuert das Layout der Site-Navigation:

  • Linke Navigation - Wenn mehr als 3 Sammlungen vorhanden sind, werden die Sammlungsnamen in der linken Navigation angezeigt. Wenn Sie diese Option verwenden, a ZUHAUSE Das Layout wird mit einem Platzhalter-Textobjekt versehen, damit die Betrachter zur Startseite zurückkehren können. Dieser Text kann bei Bedarf angepasst werden.
  • Reduzierte Menünavigation - Ein zusammenklappbares Hamburger-Menü wird angezeigt

Feature Karussell

Das Feature Karussell Die Einstellung steuert, wie die Videos in der aktuellen Sammlung angezeigt werden:

  • Vorschaubild - Unter dem Player wird eine Miniaturansicht der Videos angezeigt
  • Nur Video - Es wird nur ein einziges Video angezeigt. <> Auf dem Player werden Symbole angezeigt, mit denen Sie zu anderen Videos in der Sammlung navigieren können

Karussell-Paging-Tasten

Das Karussell-Paging-Tasten Die Einstellung bestimmt, wann Karussell-Paging-Schaltflächen angezeigt werden:

  • Paging-Schaltflächen immer anzeigen - Paging-Schaltflächen werden immer angezeigt
  • Paging-Schaltflächen nur beim Hover anzeigen - Paging-Schaltflächen werden nur angezeigt, wenn sich die Maus über dem Player oder den Videoinformationen befindet

Video-Miniaturansichten

Das Video-Miniaturansichten Die Einstellung steuert die Größe der Miniaturbilder des Videos:

  • Klein
  • Groß

Thumbnail-Wiedergabesymbole

Das Thumbnail-Wiedergabesymbole Die Einstellung bestimmt, ob im Player eine Wiedergabetaste angezeigt wird:

  • Keine Wiedergabesymbole anzeigen - Es wird kein Wiedergabesymbol angezeigt
  • Standardmäßig werden Wiedergabesymbole angezeigt - Das Wiedergabesymbol wird angezeigt

Videobeschreibung

Das Videobeschreibung Die Einstellung steuert die Länge der angezeigten Videobeschreibung:

  • Beschreibung abschneiden - Zeigt eine abgeschnittene Videobeschreibung an
  • Vollständige Beschreibung anzeigen - Zeigt die gesamte Videobeschreibung an
