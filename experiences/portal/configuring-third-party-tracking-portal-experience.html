Konfigurieren von Drittanbieter-Tracking für eine Portalerfahrung

{{ page.description }}

Bearbeiten Sie die Erfahrung und klicken Sie auf, um die Tracking-Einstellungen von Drittanbietern für eine Portal-Erfahrung zu konfigurieren STANDORTKONFIGURATION> Nachverfolgung durch Dritte in der linken Navigation. Portal-Erlebnisse können in andere Websites von Drittanbietern integriert werden oder von diesen bereitgestellte Dienste nutzen:

  • Google Analytics
  • Google Tag Manager
  • Eloqua
  • Marketo

Um auf die Tracking-Einstellungen zuzugreifen, bearbeiten Sie die Erfahrung und klicken Sie auf STANDORTKONFIGURATION> Nachverfolgung durch Dritte in der linken Navigation.

Hinweis: Die verfügbaren Einstellungen basieren auf der ausgewählten Site-Vorlage. Wenn die Vorlage die Option nicht unterstützt, wird sie nicht angezeigt.

Integration in Google Analytics

Die Galerie bietet die Möglichkeit, Tracking-Daten an Google Analytics zu senden. Wenn Sie eine neue Eigenschaft zum Verfolgen in Google Analytics erstellen, stellt Google eine Verfolgungs-ID und einen Verfolgungscode bereit, die Ihrer Website hinzugefügt werden müssen. Die Tracking-ID wird verwendet, um Google Analytics mitzuteilen, an welches Konto und welche Eigenschaft Daten gesendet werden sollen, und in das Feld Google Analytics eingegeben werden soll. Der Tracking-Code kann auf eine der folgenden Arten zu Ihrer Site hinzugefügt werden:

  • Benutzerdefiniertes Kopf- / Fußzeilen-HTML - Wenn Sie verwenden Benutzerdefiniertes Kopf- / Fußzeilen-HTML Für Ihre Site können Sie den Tracking-Code als Teil des Kopf- / Fußzeilen-HTML-Codes einfügen
  • Geben Sie eine benutzerdefinierte JavaScript-Datei an - Speichern Sie den Tracking-Code in einer auf Ihrer Domain gehosteten .js-Datei und zeigen Sie dann mit der benutzerdefinierten JavaScript-Eigenschaft für die Site

Durch Hinzufügen der Tracking-ID in den Site-Einstellungen erhalten Sie die Standardanalyse "Seitenaufruf". Galerie fügt das minimal erforderliche JavaScript für dasselbe hinzu. Durch Hinzufügen von Tracking-Code als Teil eines benutzerdefinierten Headers oder in einer benutzerdefinierten JavaScript-Datei können Sie spezifischere Analysen hinzufügen. Sie können beispielsweise benutzerdefinierten Code für Klick- oder andere Ereignisse hinzufügen.

Weitere Informationen zu Google Analytics und zum Erstellen einer Tracking-ID für Ihre Website finden Sie unter Google Analytics Webseite.

Integration in Google Tag Manager

Die Galerie bietet die Möglichkeit, Code aus dem Google Tag Manager einzuschließen. Fügen Sie den entsprechenden Code in das Feld ein <head> oder <body> Abschnitte.

Weitere Informationen zu Google Tag Manager finden Sie unter Google Webseite.

Integration mit Eloqua

Die Galerie enthält Eloqua-Seitenverfolgungsskripte, mit denen Sie Besuche auf Ihrer Website verfolgen können. Wenn Sie Ihre Site in Eloqua integrieren möchten, sollten Sie Ihre Eloqua-Site-ID eingeben.

Integration in Marketo

Die Galerie enthält Marketo-Seitenverfolgungsskripte, mit denen Sie Besuche auf Ihren Videoseiten verfolgen können. Wenn Sie Ihre Site in Marketo integrieren möchten, sollten Sie Ihre Marketo-Konto-ID eingeben.
