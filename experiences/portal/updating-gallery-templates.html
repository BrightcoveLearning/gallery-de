Aktualisieren von Galerievorlagen

{{ page.description }}

Alle Galerieerlebnisse basieren auf einer Vorlage. Wenn Sie eine neue Erfahrung erstellen, müssen Sie zunächst eine Vorlage auswählen, die verwendet werden soll. Die Vorlage steuert das Gesamtlayout und das Styling der Erfahrung. Im Laufe der Zeit können von Brightcove neue Funktionen und Verbesserungen an Vorlagen vorgenommen werden. Brightcove wird wichtige Vorlagenaktualisierungen nicht auf spannende Erlebnisse übertragen. Publisher sind dafür verantwortlich, ihre Erfahrungen zu aktualisieren, um neue Vorlagen zu verwenden, und diese dann vor dem Veröffentlichen zu testen.

Hinweis: Dieses Thema gilt nur für Portal-, Landing Page- und Event-Erlebnisse.

Festlegen, wann Vorlagenaktualisierungen verfügbar sind

Wenn Sie Erfahrungen mit Vorlagenaktualisierungen haben, wird beim Öffnen des Galeriemoduls oben auf der Seite ein Banner angezeigt. Drücke den Link, um nur die Erfahrungen anzuzeigen, für die Vorlagenaktualisierungen erforderlich sind.

Der Filter oben auf der Seite kann auch verwendet werden, um die Erfahrungen anzuzeigen, für die eine Vorlagenaktualisierung erforderlich ist.

Alle Erfahrungen, für die eine Vorlagenaktualisierung erforderlich ist, zeigen den Vorlagenstatus von an Muss aktualisiert werden.

Hinweis: Es ist möglich, Erfahrungen mit verschiedenen Versionen derselben Vorlage zu sammeln. Wenn beispielsweise ein Update für die Auswahlrechteckvorlage verfügbar ist, wird die Muss aktualisiert werden Der Status wird nur auf Laufschrift-Websites angezeigt, für die eine Vorlagenaktualisierung erforderlich ist. Möglicherweise haben Sie andere Marquee-Websites, die mit der neuesten Version der Marquee-Vorlage erstellt wurden, die kein Update erfordern.

Informationen zum Vorlagenupdate finden Sie unter Details anzeigen Link auf der rechten Seite.

Durch Klicken auf den Link wird ein Dialogfeld mit den Details zur Vorlagenaktualisierung angezeigt.

Klicken Stornieren um den Dialog zu schließen oder Update starten um mit der Vorlagenaktualisierung zu beginnen.

Anwenden eines Vorlagenupdates auf eine Erfahrung

Führen Sie die folgenden Schritte aus, um eine Vorlage zu aktualisieren.

  1. Drücke den Details anzeigen Link, um die Änderungen an der Vorlage zu überprüfen.
    Hinweis: Möglicherweise wird ein Vorlagenstatus für die Aktualisierung des Lebenslaufs angezeigt, wenn das Vorlagenupdate angewendet wurde und die Erfahrung nicht veröffentlicht wurde.
  2. Klicken Update starten. Das Vorlagenupdate wird auf die Erfahrung angewendet und der Galerie-Site-Editor wird geöffnet.
    Hinweis: Sie sehen möglicherweise a Vorlagenstatus von Update fortsetzen Schaltfläche, wenn die Vorlagenaktualisierung angewendet wurde und die Erfahrung nicht veröffentlicht wurde.
  3. Drücke den Vorschau Ihrer Website Verknüpfung. Eine Vorschau mit der neuen Vorlage wird in einem neuen Browser-Tab geöffnet. Überprüfen Sie die Website, um sicherzustellen, dass alle Player-Anpassungen, benutzerdefiniertes CSS / JavaScript, benutzerdefinierte Kopf- / Fußzeilen und andere von Ihnen vorgenommene Anpassungen korrekt angezeigt werden.
  4. (Optional) Drücke den Verwerfen Sie das Update Link oder Update verwerfen Taste zu Verwerfen Sie das Vorlagen-Update.
  5. Klicken Sie auf, um die Site mit der aktualisierten Vorlage zu veröffentlichen Veröffentlichen.

Sobald die Site mit der aktualisierten Vorlage veröffentlicht wurde, kann nicht mehr auf die ältere Vorlage zugegriffen werden.

Verwerfen eines Vorlagenupdates

Drücke den Verwerfen Sie das Update Link oder Update verwerfen Schaltfläche, um die Vorlagenaktualisierung zu verwerfen.

Sie werden aufgefordert, das Update zu verwerfen.

Klicken Update verwerfen um die Site wieder auf die ältere Version der Vorlage zurückzusetzen. Beachten Sie, dass Brightcove die Aktualisierung auf die neueste Version von Vorlagen empfiehlt. Wenn Sie das Update verwerfen, werden ältere Versionen oder Vorlagen irgendwann von Brightcove nicht mehr unterstützt.
