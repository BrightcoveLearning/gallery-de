Anpassen des Inhalts von Portalerlebnisseiten

{{ page.description }}

Objekte wie Text, Bilder und Links können zu Portal Experience-Seiten hinzugefügt werden, sodass Sie Ihr Videoerlebnis anpassen und mit einem Branding versehen können.

Hinweis: Der Vorgang zum Hinzufügen von Objekten zu einer Seite variiert je nach Vorlage geringfügig. Die Screenshots in diesem Thema wurden mit der Marquee-Vorlage aufgenommen.
Hinweis: Einige Seiten bieten auch die Möglichkeit, das Player-Layout zu ändern. Weitere Informationen finden Sie unter Ändern des Player-Layouts in einer Portal-Erfahrung.

Hinzufügen von Objekten zu Seiten

Mit dem Site-Editor können Sie einer Seite Objekte wie Bilder, Text und Links hinzufügen. Bearbeiten Sie zunächst eine Erfahrung und klicken Sie dann auf SEITEN in der linken Navigation. Die Seitenvorlage zeigt Plussymbole () gibt an, dass Objekte zur Seite hinzugefügt werden können. Bewegen Sie den Mauszeiger über das Pluszeichen und klicken Sie darauf, um das entsprechende Objekt hinzuzufügen.

Die Seitenvorlage kann auch Textobjekte enthalten, die Teil der Vorlage sind. Wenn Sie mit der Maus über sie fahren, wird ein angezeigt Bearbeiten Taste. Klicken Sie auf das Objekt oder Bearbeiten Schaltfläche zum Bearbeiten des Objekts.

Beachten Sie, dass die Objekte, die hinzugefügt oder bearbeitet werden können, auf der Vorlage basieren, die beim ersten Erstellen der Erfahrung ausgewählt wurde.

Hinzufügen von Text zu einer Seite

Text kann auf verschiedene Arten zu einer Seite hinzugefügt werden:

  • Durch Klicken auf die + Bild oder Text hinzufügen Verknüpfung
  • Durch Klicken auf einen der Textplatzhalter auf der Seite

Fügen Sie Text hinzu, indem Sie auf klicken + Bild oder Text hinzufügen Verknüpfung

Um der Seite Text hinzuzufügen, klicken Sie auf + Bild oder Text hinzufügen Verknüpfung:

  1. Klicke auf .
  2. Geben Sie Ihren Text ein. Die Formatierungsleiste kann zum Formatieren des Texts verwendet werden. Klicken speichern wenn fertig.

Fügen Sie Text hinzu, indem Sie auf einen Textplatzhalter klicken

Seiten können zahlreiche Textobjekte als Teil der Vorlage enthalten. Klicken Sie auf das Textobjekt, um es zu bearbeiten.

So fügen Sie Ihren Text hinzu:

  1. Klicken Sie auf eines der Platzhalterobjekte.
  2. Geben Sie Ihren Text ein und klicken Sie auf speichern.

Hinzufügen eines Bildes zum Seitenkopf

Die Größe der Bilder wird basierend auf der Vorlage geändert (niemals hochskaliert). Der Bild-Uploader zeigt möglicherweise die empfohlenen Bildabmessungen an, je nachdem, wo das Bild verwendet wird. JPEG-, PNG- und GIF-Bilder werden unterstützt. So fügen Sie dem Seitenkopf ein Bild hinzu:

  1. Klicke auf .
  2. Drücke den Bild Tab. Der Bildbrowser-Dialog wird geöffnet.
  3. Klicken Durchsuche Durchsuchen Sie dann ein Bild aus dem Dateisystem und wählen Sie es aus. Klicken speichern wenn fertig.

Hinzufügen von Navigationslinks zum Seitenkopf

Navigationslinks können oben auf der Seite hinzugefügt werden und Benutzern die Möglichkeit bieten, zu anderen Teilen Ihrer Website zu navigieren. So fügen Sie einer Seite einen Navigationslink hinzu:

  1. Klicken + Menü Link hinzufügens im Seitenkopf.
  2. Geben Sie die Text und Verknüpfung. Prüfen Neues Fenster Wenn Sie möchten, dass der Link in einem neuen Browserfenster geöffnet wird. Klicken speichern wenn fertig.

Zusätzliche Navigationslinks können durch Klicken eingegeben werden + Link hinzufügen. Um einen Link zu bearbeiten, klicken Sie darauf.

Hinzufügen von Navigationslinks zur Seitenfußzeile

Navigationslinks können auch zur Seitenfußzeile hinzugefügt werden. Dies ist nützlich, wenn Sie einen Titel und dann eine Reihe von Links unten haben möchten. Beispielsweise:

Das Navigationsobjekt für die Seitenfußzeile besteht aus zwei Teilen: a PLATZHALTER für den Titel und + Menü-Links hinzufügen für die zugehörigen Links.

So fügen Sie der Seitenfußzeile Navigationslinks hinzu:

  1. Klicken PLATZHALTER Geben Sie einen Wert für die Kopfzeile ein und klicken Sie auf speichern.
  2. Klicken Sie auf +Menü-Links hinzufügen und geben Sie die Text-, Link-URL und Listenreihenfolge ein. Sie können auch die Reihenfolge der Links festlegen, indem Sie die Option festlegen Liste bestellen Wert, wenn der Link erstellt wird.

Zusätzliche Navigationslinks können durch Klicken eingegeben werden + Menü-Links hinzufügen.

Objekte von einer Seite löschen

Text, Bilder und Links können auf verschiedene Arten von der Seite entfernt werden.

So entfernen Sie Text oder Bilder aus dem Seitenkopf

  1. Bewegen Sie den Mauszeiger über das Objekt und klicken Sie auf das Löschsymbol.

So entfernen Sie Navigationslinks aus dem Seitenkopf

  1. Klicken Sie auf den Link in der Navigationsüberschrift.
  2. Klicken Löschen.

So entfernen Sie Navigationslinks aus der Seitenfußzeile

  1. Klicken Sie auf den Link in der Seitenfußzeile.
  2. Klicken Löschen.

So entfernen Sie den Titel aus der Seitenfußzeile

  1. Klicken Sie auf den Titel in der Seitenfußzeile.
  2. Löschen Sie den Titeltext und klicken Sie auf speichern.
