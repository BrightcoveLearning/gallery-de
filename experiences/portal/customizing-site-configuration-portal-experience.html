Anpassen der Site-Konfiguration einer Portalerfahrung

{{ page.description }}

Bearbeiten Sie die Erfahrung und klicken Sie auf, um die Site-Konfigurationseinstellungen für eine Portal-Erfahrung zu konfigurieren STANDORTKONFIGURATION in der linken Navigation. Folgende Einstellungen stehen zur Verfügung:

  • URL - Konfigurieren Sie die Erfahrung für die Verwendung einer benutzerdefinierten Domäne
  • Suchmaschinenoptimierung - Konfigurieren Sie Einstellungen, um die Suchmaschinenoptimierung zu verbessern
  • Zugangskontrolle - Konfigurieren Sie Single Sign-On- und IP-Einschränkungen
  • Tracking von Drittanbietern - Konfigurieren Sie die Integration mit Google, Eloqua und Marketo

Verwenden einer benutzerdefinierten Domain

Durch die Verwendung einer benutzerdefinierten Domain können Sie eine Portal Experience unter Ihrer eigenen URL hosten. Informationen zur Verwendung einer benutzerdefinierten Domain finden Sie unter Zuweisen einer benutzerdefinierten Domäne zu einer Portalerfahrung.

SEO-Einstellungen konfigurieren

Die Einstellungen für die Suchmaschinenoptimierung werden verwendet, um das Site-Image, die Keywords und Meta-Tags zu konfigurieren und auf die URLs der Site-Map und des Such-Feeds zuzugreifen. Informationen zum Konfigurieren der Einstellungen für die Suchmaschinenoptimierung finden Sie unter Konfigurieren der Einstellungen für die Suchmaschinenoptimierung für eine Portal-Erfahrung.

Konfigurieren der Zugriffssteuerungseinstellungen

Mit den Einstellungen für die Zugriffssteuerung können Sie steuern, wer auf eine Portalerfahrung zugreifen kann. Gallery kann den Zugriff mithilfe eines einfachen Zugriffscodes einschränken, in Single Sign-On-Lösungen integrieren und den Zugriff auf Websites basierend auf der IP-Adresse des Viewers einschränken. Informationen zum Steuern des Zugriffs auf Galerie-Websites finden Sie unter Steuern des Zugriffs auf eine Portal-Erfahrung.

Tracking-Einstellungen konfigurieren

Portal Experiences können in andere Websites von Drittanbietern integriert werden oder Dienste nutzen, die von anderen Websites von Drittanbietern bereitgestellt werden, z.

  • Google Analytics
  • Google Tag Manager
  • Eloqua
  • Marketo

Informationen zur Integration in Websites von Drittanbietern finden Sie unter Konfigurieren des Tracking von Drittanbietern für eine Portal-Erfahrung.
