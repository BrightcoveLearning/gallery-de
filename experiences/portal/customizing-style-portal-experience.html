Anpassen des Stils einer Portalerfahrung

{{ page.description }}

Mit dem Galerie-Site-Editor können Sie den Stil einer Portal-Erfahrung anpassen, indem Sie das Thema, die Farben und die Schriftarten ändern. Um den Stil zu bearbeiten, bearbeiten Sie die Erfahrung und klicken Sie dann auf STIL in der linken Navigation.

Thema auswählen

Auswählen von a Thema Wendet einen vordefinierten Satz von Schriftarten und Farben auf die Site an. So ändern Sie die Site Thema::

  1. Klicken STIL in der linken Navigation.
  2. Wählen Sie a Thema aus der Dropdown-Liste. Wenn Sie ein Thema auswählen, wird die Site aktualisiert, sodass Sie eine Vorschau des Themas anzeigen können.
  3. Klicken Sie auf, um zum Site-Editor zurückzukehren <Zurück zur Erfahrung.

Farben anpassen

Das Farben Einstellungen können verwendet werden, um die Farben, die von einem Thema bereitgestellt werden, fein abzustimmen. Beispielsweise können Farbeinstellungen verwendet werden, um die Hintergrund- und Schriftfarben fein abzustimmen. So passen Sie das an Farben::

  1. Klicken STIL Symbol in der linken Navigation. Das Farben Im Abschnitt werden alle Seitenobjekte und ihre Farben angezeigt.
  2. Um eine neue Farbe auszuwählen, klicken Sie auf den Farbkreis.
  3. Wählen Sie mit der Farbauswahl eine Farbe aus, indem Sie auf die Farbe im Farbquadrat klicken oder eine hexadezimale Farbnummer unter dem Farbquadrat eingeben. Beachten Sie, dass Sie den Farbbereich ändern können, indem Sie in das Farbspektrum unter dem Farbquadrat klicken. Klicken X. um den Farbwähler zu schließen und die Änderung zu speichern.

    Um eine Farbauswahl zurückzusetzen, klicken Sie auf den Pfeil nach links neben dem Farbkreis.

  4. Klicken Sie auf, um zum Site-Editor zurückzukehren <Zurück zur Erfahrung.

Schriftarten auswählen

Die auf der Site verwendeten Schriftarten können ebenfalls geändert werden. So wählen Sie verschiedene Schriftarten aus:

  1. Klicken STIL in der linken Navigation.
  2. Neu auswählen Schriftarten Verwenden der Dropdown-Listen.

    Klicken Sie auf, um eine Schriftartenauswahl zurückzusetzen zurücksetzen Link über der Schriftart-Dropdown-Liste.

    Hinweis: Die Anzahl der verfügbaren Schriftarten hängt vom ausgewählten Site-Thema ab.
  3. Klicken Sie auf, um zum Site-Editor zurückzukehren <Zurück zur Erfahrung.
Hinweis: Informationen zur Verwendung benutzerdefinierter Schriftarten in einem Portal Experience finden Sie unter Verwenden von benutzerdefinierten Schriftarten in a Portal Experience.
