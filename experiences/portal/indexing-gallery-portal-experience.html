Indizieren einer Galerieportal-Erfahrung

{{ page.description }}

Sobald eine Portal Experience veröffentlicht wurde, können verschiedene Maßnahmen ergriffen werden, um die Website von Suchmaschinen zu bewerben und zu indizieren:

  • Verlinken Sie von vorhandenen Web-Eigenschaften, die bereits indiziert sind, auf die neue Site. Beispielsweise möchten Sie möglicherweise einen Blog-Beitrag mit einem Link zur neuen Website schreiben und / oder einen Link zur Website auf Ihrer Hauptwebsite hinzufügen
  • Teile es und bewerbe es in sozialen Netzwerken
  • Ermutigen Sie Kunden, die Website zu besuchen, und hoffentlich teilen sie sie selbst sozial
  • Senden Sie die URL oder eine Sitemap an Suchmaschinen, damit diese die Site indizieren

Dieses Thema konzentriert sich auf die Erstellung einer Sitemap für Ihre Portal Experience, die an Suchmaschinen gesendet werden kann.

Generieren einer Sitemap für eine Portal-Erfahrung

Sitemaps sind eine Möglichkeit, Suchmaschinen über Seiten auf Ihrer Website zu informieren. Galerie generiert eine XML-Sitemap, in der die Seiten Ihrer Website aufgelistet sind. Suchmaschinen analysieren dann das XML, um zu verstehen, welche Inhalte sich auf der Website befinden.

Um auf die Sitemap für eine Portal-Erfahrung zuzugreifen, bearbeiten Sie die Erfahrung und klicken Sie auf SITE-KONFIGURATION> Suchmaschinenoptimierung in der linken Navigation. Die URL zur Sitemap wird unten auf der Seite angezeigt.

Hinweis: Die Sitemap ist erst verfügbar, wenn das Erlebnis veröffentlicht wurde.

Sitemaps werden 24 Stunden lang zwischengespeichert. Nach 24 Stunden werden sie aus dem Cache entfernt und beim erneuten Zugriff neu generiert. Wenn Videos zu einer Portal-Erfahrung hinzugefügt werden, werden sie nicht in die Sitemap aufgenommen, bis die Sitemap neu generiert wird (nach 24 Stunden) oder bis die Site erneut veröffentlicht wird.

Sie können die Sitemap-URL kopieren und in die Google / Bing-Webmaster-Tools einfügen. Weitere Informationen zum Senden einer Sitemap an Suchmaschinen finden Sie unter folgenden Links:

  • Senden Sie eine Sitemap an die Google Webmaster-Tools
  • Senden von Sitemaps an Bing

Wenn Sie eine Sitemap an Google senden, besteht ein Schritt in diesem Prozess darin, den Besitz der Website zu überprüfen. Brightcove empfiehlt die Verwendung der HTML-Tag-Option. Das von Google generierte Meta-Tag kann als Teil des zu Ihrer Erfahrung hinzugefügt werden Suchmaschinenoptimierung die Einstellungen.

Galerie erstellt auch eine robots.txt Datei, die Suchmaschinen anweist, die Sitemap zu verwenden.
