Konfigurieren der Details für eine In-Page-Erfahrung

{{ page.description }}

Der Name der Erfahrung, die Beschreibung und die Layoutgröße können alle als Teil der Übersichtseinstellungen konfiguriert werden. Um die Übersichtseinstellungen zu konfigurieren, bearbeiten Sie die Erfahrung und klicken Sie dann auf ÜBERBLICK in der linken Navigation.

Die Übersichtsseite wird angezeigt:

  • Erfahrungstyp - Zeigt die Vorlage an, die für die In-Page-Erfahrung verwendet wird
  • Erleben Sie Titel und Beschreibung - Wenn Sie zum ersten Mal eine neue Erfahrung erstellen, werden Sie aufgefordert, die einzugeben Titel erleben und Erfahrungsbeschreibung. Diese Werte können hier bearbeitet werden.
  • Sprache erleben - Wählen Sie die Sprache aus, in der UI-Nachrichten angezeigt werden sollen
  • Layoutgröße - Bestimmt die Größe für die In-Page-Erfahrung. Sehen Einstellen der Erlebnisgröße für Details.

Wenn Sie fertig sind, klicken Sie auf speichern um Ihre Änderungen zu speichern.

Hinweis: Stellen Sie nach Änderungen an einem Erlebnis sicher, dass Sie das Erlebnis erneut veröffentlichen, damit die Änderungen auf Websites angezeigt werden, auf denen das Erlebnis angezeigt wird.

Einstellen der Erlebnisgröße

In-Page-Erlebnisse können auf drei Arten bewertet werden:

  • Reaktionsschnell - Die Größe der Erfahrung wird automatisch an den Container angepasst.
    Hinweis: Bei Auswahl dieser Option muss der empfohlene Einbettungscode verwendet werden. Der Einbettungscode „Kein JavaScript“ reagiert nicht.
  • Feste Größe - Die Größe der Erfahrung wird gemäß der angegebenen Größe bemessen Breite und Höhe Werte.
  • Feste Breite -Die Breite der Erfahrung wird auf den angegebenen Wert eingestellt Breite Wert.

Wenn Sie fertig sind, klicken Sie auf speichern um Ihre Änderungen zu speichern.

Hinweis: Zum Ändern der Layoutgröße muss die Erfahrung erneut veröffentlicht und der gesamte vorhandene eingebettete Veröffentlichungscode aktualisiert werden. Weitere Informationen finden Sie unter Vorschau und Veröffentlichung einer In-Page-Erfahrung.
