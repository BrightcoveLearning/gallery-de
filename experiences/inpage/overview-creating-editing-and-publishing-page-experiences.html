Überblick: Erstellen, Bearbeiten und Veröffentlichen von In-Page-Erfahrungen

{{ page.description }}

In-Page-Erlebnisse sind auf einer Seite Ihrer eigenen Website enthalten und können grundlegende Interaktivitätsfunktionen (Links und Karten) enthalten, die Ihr Publikum ansprechen.

Hinweis: Galerie bietet auch die Möglichkeit, Portal-Erlebnisse zu erstellen, bei denen es sich um eigenständige Websites handelt, auf die unter einer bestimmten URL zugegriffen wird. Weitere Informationen finden Sie unter Überblick: Erstellen, Bearbeiten und Veröffentlichen von Portalerlebnissen.

Das Folgende ist ein Beispiel für eine In-Page-Erfahrung, die mit Gallery erstellt wurde. In-Page-Erlebnisse sind in Ihre Webseiten eingebettet und werden zusammen mit anderen Seiteninhalten angezeigt.

Eine schrittweise Anleitung zum Erstellen einer In-Page-Erfahrung mit Brightcove Gallery finden Sie unter Schritt für Schritt: Erstellen und Veröffentlichen einer In-Page-Erfahrung.

Die wichtigsten Schritte zum Erstellen und Veröffentlichen einer In-Page-Erfahrung sind:

  • Wählen Sie eine Vorlage
  • Fügen Sie Videos hinzu und passen Sie die Erfahrung an
  • Vorschau und Veröffentlichung der Erfahrung
  • Überprüfen Sie die Erfahrungsanalyse

Vorlage auswählen

Wenn Sie eine neue Galerieerfahrung erstellen, wählen Sie zunächst eine Vorlage aus. Vorlagen steuern das allgemeine Erscheinungsbild einer Erfahrung. Die Galerie bietet eine Reihe von Vorlagen, mit denen Sie die Flexibilität haben, verschiedene Arten von Erlebnissen zu liefern:

  • Portal
  • Auf Seite
  • Landingpage
  • Veranstaltung

Weitere Informationen zu In-Page Experience-Vorlagen und einige Beispielerfahrungen finden Sie in der Übersicht über In-Page-Erlebnisvorlagen für die Galerie Thema.

Galerie bietet die Möglichkeit, Vorlagen basierend auf der Art der von Ihnen erstellten Erfahrung zu filtern. Nachdem Sie eine In-Page Experience-Vorlage ausgewählt haben, können Sie den Titel, die Beschreibung und die Größe für die Erfahrung festlegen.

Hinzufügen von Videos und Anpassen der Erfahrung

Sobald die Erfahrung erstellt wurde, bietet der Layout-Editor die Möglichkeit:

  • Passen Sie die Erfahrung mit Text und Bildern an
  • Fügen Sie der Erfahrung Videos hinzu
  • Gestalten Sie das Erlebnis, indem Sie die verwendeten Schriftarten und Farben auswählen
  • Passen Sie die Erlebniseinstellungen an
  • Bearbeiten Sie die Details und die Größe der Erfahrung

Vorschau und Veröffentlichung der Erfahrung

In-Page-Erlebnisse können in der Vorschau angezeigt werden, um zu sehen, wie sie auf Desktop-, Tablet- und Mobilgeräten angezeigt werden. Durch das Veröffentlichen der Erfahrung werden eine Vorschau-URL und der Veröffentlichungscode generiert, die zum Einbetten der Erfahrung in Ihre Website erforderlich sind. Weitere Informationen finden Sie unter Vorschau und Veröffentlichung einer In-Page-Erfahrung.

Überprüfung der Erfahrungsanalyse

Während Sie In-Page-Erlebnisse veröffentlichen, sammelt Gallery Analysen zu diesen Erlebnissen und stellt sie in verschiedenen Berichten zur Verfügung. Weitere Informationen finden Sie unter Überprüfen von In-Page Experience Analytics.
