Verwenden von dynamischen Videoüberschreibungen in einem In-Page-Erlebnis

{{ page.description }}

Es kann vorkommen, dass Sie die Videos in einer In-Page-Erfahrung ändern möchten. Beispielsweise müssen sich die Videos in einer Erfahrung möglicherweise basierend auf einer Auswahl, die der Benutzer trifft, oder basierend auf der Navigation zu anderen Teilen einer Site ändern. Anstatt neue In-Page-Erlebnisse mit jeweils unterschiedlichen Videos zu erstellen, können Sie den Einbettungscode ändern, um einem Erlebnis die entsprechenden Videos zuzuweisen. Änderungen am Einbettungscode hängen von der Art des verwendeten Einbettungscodes ab:

  • Empfohlener Einbettungscode
  • Kein JavaScript-Einbettungscode

Empfohlener Einbettungscode

Der typische In-Page Experience-Einbettungscode (empfohlene Option) sieht folgendermaßen aus:

<div data-experience="597a2aa114b2ed00103052af"></div>
<script src="https://players.brightcove.net/20318290001/experience_597a2aa114b2ed00103052af/live.js"></script>

Um die mit einer In-Page-Erfahrung verknüpften Videos dynamisch zu überschreiben, kann der Einbettungscode geändert werden, um dem ein Attribut hinzuzufügen div Tag, um die Videos oder die Wiedergabeliste anzugeben, die verwendet werden sollen.

Einzelne Videos angeben

Fügen Sie das hinzu, um einzelne zu verwendende Videos anzugeben data-video-ids Parameter zum div Markieren Sie und geben Sie dann eine durch Kommas getrennte Liste von Video-IDs an.

<div data-experience="597a2aa114b2ed00103052af" 
     data-video-ids="12345,56789"></div>
<script src="https://players.brightcove.net/20318290001/experience_597a2aa114b2ed00103052af/live.js"></script>

Angeben einer Wiedergabeliste

Fügen Sie die hinzu, um eine zu verwendende Wiedergabeliste anzugeben data-playlist-id Parameter zum div Tag und geben Sie dann die zu verwendende Wiedergabelisten-ID an.

<div data-experience="597a2aa114b2ed00103052af" 
     data-playlist-id="12345"></div>
<script src="https://players.brightcove.net/20318290001/experience_597a2aa114b2ed00103052af/live.js"></script>

Kein JavaScript-Einbettungscode

Der typische Einbettungscode für In-Page Experience (keine JavaScript-Option) sieht folgendermaßen aus:

<iframe style="display:block;border:none;margin-left:auto;margin-right:auto;"
src="https://players.brightcove.net/20318290001/experience_597a2aa114b2ed00103052af/index.html"
allowfullscreen webkitallowfullscreen mozallowfullscreen></iframe>

Um die mit einer In-Page-Erfahrung verknüpften Videos dynamisch zu überschreiben, kann der Einbettungscode geändert werden, um dem Parameter einen Parameter hinzuzufügen src Attribut.

Einzelne Videos angeben

Fügen Sie das hinzu, um einzelne zu verwendende Videos anzugeben videoIds Parameter und geben Sie dann eine durch Kommas getrennte Liste von Video-IDs an.

<iframe style="display:block;border:none;margin-left:auto;margin-right:auto;"
src="https://players.brightcove.net/20318290001/experience_597a2aa114b2ed00103052af/index.html?videoIds=12345,56789""
allowfullscreen webkitallowfullscreen mozallowfullscreen></iframe>

Angeben einer Wiedergabeliste

Fügen Sie die hinzu, um eine zu verwendende Wiedergabeliste anzugeben playlistId Parameter und geben Sie dann die zu verwendende Wiedergabelisten-ID an.

<iframe style="display:block;border:none;margin-left:auto;margin-right:auto;"
src="https://players.brightcove.net/20318290001/experience_597a2aa114b2ed00103052af/index.html?playlistId=12345"
allowfullscreen webkitallowfullscreen mozallowfullscreen></iframe>
