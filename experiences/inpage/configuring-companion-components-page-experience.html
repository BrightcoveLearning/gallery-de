    Konfigurieren von Companion-Komponenten für ein In-Page-Erlebnis

    In diesem Thema erfahren Sie, wie Sie Begleitkomponenten in einem In-Page Experience konfigurieren.

    Alle In-Page Experience-Vorlagen bieten die Möglichkeit, Begleitkomponenten zu haben. Zu den Companion-Komponenten

    • Custom HTML
    • Bilder
    • Text

    Bei Begleitkomponenten können sich Inhalte außerhalb des Players je nachdem, welches Video abgespielt wird, ändern. Wenn beispielsweise ein Produktvideo abgespielt wird, befindet sich möglicherweise rechts neben dem Player ein anklickbares Bild, das weitere Details zu diesem Produkt öffnet. Companion-Komponenten ähneln Karten- und Link-Interaktionen, der Unterschied besteht darin, dass Karten und Links über dem Videoplayer selbst angezeigt werden. Companion-Komponenten werden rechts neben dem Videoplayer angezeigt, wenn sich das Video im Status PLAYING befindet.

    Hinweis: Informationen zum Hinzufügen von Link- und Karteninteraktionen finden Sie unter Interaktionen zu einem In-Page Experience hinzufügen.
    Hinweis: Nachdem Sie Änderungen an einem Erlebnis vorgenommen haben, stellen Sie sicher, dass Sie das Erlebnis erneut veröffentlichen , damit die Änderungen auf Websites angezeigt werden, auf denen das Erlebnis angezeigt wird.

    Anzeigen von Begleitkomponenten

    Um die konfigurierten Begleitkomponenten anzuzeigen, bearbeiten Sie das Erlebnis und klicken Sie dann im linken Navigationsbereich auf VIDEOS . Videos, die mit Interaktionen konfiguriert wurden, zeigen ein Symbol in der Spalte Interaktivität an, das die Art der Interaktion (en) angibt.

    Zu den Begleitkomponenten gehört HTML (), Bilder ( ) und Text ().

    Hinweis: Link () - und card () -Symbole können angezeigt werden, die darauf hinweisen, dass für das Video Links und Karten konfiguriert sind. Informationen zu Links und Karten finden Sie unter Hinzufügen von Interaktionen zu einem In-Page-Erlebnis.

    Konfigurieren einer begleitenden HTML-Komponente

    Gehen Sie folgendermaßen vor, um eine begleitende HTML-Komponente zu konfigurieren:

    1. Bearbeiten Sie das Erlebnis und klicken Sie dann in der linken Navigationsleiste auf VIDEOS .
    2. Klicke auf ein Video, um es auszuwählen.
    3. Klicken Sie auf Interactivity > Companion HTML hinzufügen oder klicken Sie auf das HTML () -Symbol in der Spalte Aktionen . Der Interaktivitäts-Editor wird geöffnet.
    4. Geben Sie einen Namen für die HTML-Komponente ein.
    5. Fügen Sie Ihren HTML-Code ein.
    6. Um die Startzeit und die Dauer für die Komponente zu konfigurieren, klicken Sie auf den Link Konfigurieren .
    7. Wählen Sie eine Startzeit und Dauer:
      • Show für das gesamte Video - Die Komponente wird für das gesamte Video angezeigt
      • Wählen Sie eine Startzeit und Dauer - Sie legen die Startzeit und -dauer für die Komponente manuell fest
    8. Legen Sie die Dauer für die Komponente fest.
      Hinweis: Die Dauer kann auch mit dem Schieberegler unter dem Player eingestellt werden. Ziehen Sie den Pfeil nach links auf den Zeitpunkt, an dem die Komponente erscheinen soll, und ziehen Sie dann den Rechtspfeil darauf, wann die Komponente verschwinden soll.
    9. Aktivieren Sie Bis zum Ende des Videos , wenn Sie nur eine Startzeit festlegen und die Komponente so lange erscheinen lassen möchten, bis das Video endet.
    10. Aktivieren Sie Diese Interaktion auf alle Videos im Layout anwenden , wenn diese Komponente in allen Videos des Erlebnisses angezeigt werden soll.
    11. Klicken Sie auf Save, um die Änderungen zu speichern.
    Hinweis: Um die Interaktion einem anderen Video zuzuweisen, wählen Sie mithilfe der Dropdown-Liste der Videoauswahl über dem Player ein anderes Video aus.

    Konfigurieren einer Companion Image-Komponente

    Um eine Begleitbildkomponente zu konfigurieren, führen Sie die folgenden Schritte aus:

    1. Bearbeiten Sie das Erlebnis und klicken Sie dann in der linken Navigationsleiste auf VIDEOS .
    2. Klicke auf ein Video, um es auszuwählen.
    3. Klicken Sie auf Interactivity > Companion Image hinzufügen oder klicken Sie auf das Bildsymbol () in der Spalte Aktionen . Der Interaktivitäts-Editor wird geöffnet.
    4. Geben Sie einen Namen für die Image-Komponente ein.
    5. Um ein Bild anzugeben, haben Sie folgende Vorteile:
      • Ziehen Sie ein Bild aus dem Dateisystem und legen Sie es dann auf den Drop-Bereich
      • Klicken Sie auf „ Durchsuchen “ und wählen Sie ein Bild aus dem Dateisystem
      • Geben Sie die URL zu einer Remote-Image-Datei ein
    6. Stellen Sie die Bildgrößeein.
      • Das Bild wird auf Container skaliert - Bilder, die größer sind als der Container, in dem sie sich befinden, werden so verkleinert, dass sie passen. Kleinere Bilder werden nicht hochskaliert.
      • Breite und Höhe - Geben Sie manuell eine Breite und Höhe für das Bild ein
    7. Stellen Sie das Bild Ausrichtungein.
    8. Geben Sie eine Clickthrough-URL ein. Dies ist die URL, zu der Sie gehen müssen, wenn auf das Bild geklickt wird. Sie können den Link auch in einem neuen Fenster öffnen.
    9. Um die Startzeit und die Dauer für die Komponente zu konfigurieren, klicken Sie auf den Link Konfigurieren .
    10. Wählen Sie eine Startzeit und Dauer:
      • Show für das gesamte Video - Die Komponente wird für das gesamte Video angezeigt
      • Wählen Sie eine Startzeit und Dauer - Sie legen die Startzeit und -dauer für die Komponente manuell fest
    11. Legen Sie die Dauer für die Komponente fest.
      Hinweis: Die Dauer kann auch mit dem Schieberegler unter dem Player eingestellt werden. Ziehen Sie den Pfeil nach links auf den Zeitpunkt, an dem die Komponente erscheinen soll, und ziehen Sie dann den Rechtspfeil darauf, wann die Komponente verschwinden soll.
    12. Aktivieren Sie Bis zum Ende des Videos , wenn Sie nur eine Startzeit festlegen und die Komponente so lange erscheinen lassen möchten, bis das Video endet.
    13. Aktivieren Sie Diese Interaktion auf alle Videos im Layout anwenden , wenn diese Komponente in allen Videos des Erlebnisses angezeigt werden soll.
    14. Klicken Sie auf Save, um die Änderungen zu speichern.
    Hinweis: Um die Interaktion einem anderen Video zuzuweisen, wählen Sie mithilfe der Dropdown-Liste der Videoauswahl über dem Player ein anderes Video aus.

    2. Klicke auf ein Video, um es auszuwählen.

    4. Geben Sie einen Namen für die Textkomponente ein.
    5. Geben Sie den Text ein, der zum Erlebnis hinzugefügt werden soll. Die Symbolleiste kann verwendet werden, um den Text zu formatieren.
    6. Um die Startzeit und die Dauer für die Komponente zu konfigurieren, klicken Sie auf den Link Konfigurieren .
    7. Wählen Sie eine Startzeit und Dauer:

    Bearbeiten/Löschen von Begleitkomponenten

    Wenn Sie den Mauszeiger über ein Symbol in der Spalte Interaktivität zeigen, werden alle Interaktionen dieses Typs angezeigt. In diesem Beispiel wurde eine Image-Komponente konfiguriert.

    Bewegen Sie den Mauszeiger über ein Symbol im Feld, um eine Komponente zu bearbeiten Interaktivität Spalte und klicken Sie auf die Bearbeitung ( ) Symbol neben den Komponentent oder klicken Sie auf den Videotitel. Der Interaktivitäts-Editor wird geöffnet. Bearbeiten Sie die Komponente und speichern Sie dann die Änderungen.

    Um eine Komponente zu löschen, bewegen Sie den Mauszeiger über ein Symbol in der Spalte Interaktivität und klicken Sie auf das Symbol Löschen () neben der Komponente.

    Komponenten können auch durch Klicken auf das Bearbeitungssymbol () in der Spalte Aktionen bearbeitet/gelöscht werden. Dadurch wird eine Liste aller Interaktionen für das Video angezeigt.

    Klicken Sie auf das Bearbeitungssymbol, um die Komponente zu bearbeiten oder das Symbol zum Löschen der Interaktion zu löschen.

    Auf Companion Komponenten kann auch über das Menü LAYOUT zugegriffen werden. Wählen Sie den Status PLAYING und die Begleitersymbole werden auf dem Erlebnis angezeigt.
