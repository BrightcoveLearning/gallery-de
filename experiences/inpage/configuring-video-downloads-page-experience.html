Konfigurieren von Video-Downloads für ein In-Page-Erlebnis

{{ page.description }}

Galerie-In-Page-Erlebnisse können so konfiguriert werden, dass ein Download-Link angezeigt wird, über den Zuschauer Videoinhalte herunterladen können. In-Page-Erlebnisse können so konfiguriert werden, dass ein Download-Link für angezeigt wird Alle Videos oder bestimmte Videos basierend auf einem benutzerdefinierten Feldwert. Wenn ein Video heruntergeladen werden kann, wird neben dem Video ein Download-Link angezeigt. Die Wiedergabe von höchster Qualität wird heruntergeladen.

Hinweis: Videodownloads werden auf iOS-Geräten nicht unterstützt.

Informationen zum Bereitstellen eines Download-Links für alle Videos in einer Erfahrung finden Sie unter Konfigurieren der Einstellungen für eine In-Page-Erfahrung Dokumentation. Wenn Sie selektiv nur bestimmte Videos herunterladen möchten, müssen Sie die folgenden Schritte ausführen:

  1. Erstellen Sie ein benutzerdefiniertes Feld, um herunterladbare Inhalte anzuzeigen
  2. Weisen Sie dem benutzerdefinierten Feld einen geeigneten Wert zu
  3. Konfigurieren Sie die In-Page-Erfahrung, um Downloads zu ermöglichen

Erstellen eines benutzerdefinierten Felds, um herunterladbare Inhalte anzuzeigen

Videos, die heruntergeladen werden können, werden durch Festlegen des Werts eines benutzerdefinierten Felds angezeigt. Erstellen Sie zunächst ein benutzerdefiniertes Feld in Ihrem Konto, das angibt, dass ein Video heruntergeladen werden kann.

  1. Melden Sie sich bei Video Cloud Studio an.
  2. Klicken Sie in der Navigationsüberschrift auf ADMINISTRATOR und dann Videofelder. Beachten Sie, dass Sie ein Kontoadministrator sein müssen, um auf das ADMIN-Menü zugreifen zu können.
  3. Klicken Benutzerdefinierte Felder hinzufügen.
  4. Erstellen Sie ein neues benutzerdefiniertes Feld, in dem angegeben wird, ob ein Video heruntergeladen werden kann oder nicht. Der Name und der zugewiesene Wert spielen keine Rolle. Im folgenden Beispiel ein neues benutzerdefiniertes Feld herunterladbar wird erstellt. Das Art ist eingestellt auf Aufführen mit 2 Werten, wahr und falsch. Für Videos, die heruntergeladen werden können, wird dieser Wert auf gesetzt wahr als Teil der Videoeigenschaften.
  5. Klicken Änderungen speichern und dann speichern.

Zuweisen von benutzerdefinierten Feldwerten

Jedem Video, das heruntergeladen werden kann, sollte dem benutzerdefinierten Feld, das erstellt wurde, ein Wert zugewiesen werden.

  1. Öffne das Medien Modul.
  2. Um ein Video als herunterladbar zu markieren, klicken Sie auf den Videonamen, um die Videoeigenschaften zu öffnen.
  3. Suchen Sie die Benutzerdefinierte Felder Abschnitt und klicken Bearbeiten.
  4. Stellen Sie den Wert des benutzerdefinierten Felds entsprechend ein. In diesem Beispiel ist der Wert von Herunterladbar ist eingestellt auf wahr.
  5. Klicken speichern.
  6. Bearbeiten Sie die Videoeigenschaften für zusätzliche Videos nach Bedarf.
Hinweis: Verwenden Sie die Funktion „ Schnellbearbeitung“ im Medienmodul, um die benutzerdefinierten Feldwerte für mehrere Videos gleichzeitig zu bearbeiten.

Konfigurieren einer In-Page-Erfahrung, um Downloads zu ermöglichen

Führen Sie die folgenden Schritte aus, um eine In-Page-Erfahrung so zu konfigurieren, dass Video-Downloads basierend auf einem benutzerdefinierten Feldwert möglich sind.

  1. Öffnen Sie das Galerie-Modul.
  2. Suchen Sie Ihre Erfahrung und klicken Sie auf Bearbeiten.
  3. Klicken EINSTELLUNGEN> Video in der linken Navigation.
  4. In dem Video-Downloads Abschnitt, wählen Sie die Stellen Sie für jedes Video einen Download-Link bereit, der auf einem benutzerdefinierten Feldwert basiert Möglichkeit.
  5. Geben Sie den benutzerdefinierten Feldnamen und den Wert ein, den Sie in Schritt 4 oben erstellt haben. In diesem Beispiel wurde das erstellte benutzerdefinierte Feld benannt herunterladbar und der Wert wurde auf gesetzt wahr für Videos, die heruntergeladen werden können.
  6. Klicken speichern und veröffentlichen Sie die Erfahrung erneut.
  7. Vergewissern Sie sich, dass für alle Videos mit dem folgenden Link ein Download-Link angezeigt wird herunterladbar benutzerdefiniertes Feld eingestellt auf wahr.
