    Hinzufügen von Komponenten zu einem In-Page-Erlebnis

    In diesem Thema erfahren Sie, wie Sie einem In-Page Experience Komponenten hinzufügen.

    Komponenten können zu einem In-Page Experience hinzugefügt werden, mit denen Sie das Erlebnis anpassen und gestalten können. Um Komponenten zu einem In-Page Experience hinzuzufügen, bearbeiten Sie das Erlebnis und klicken Sie dann im linken Navigationsbereich auf LAYOUT .

    Klicken Sie auf das Plus-Symbol () , in dem Sie eine Komponente hinzufügen möchten, und wählen Sie dann den Typ der hinzuzufügenden Komponente aus.

    Die folgenden Arten von Komponenten können hinzugefügt werden:

    • Werbung
    • Chat
    • Benutzerdefiniertes HT
    • bild
    • Text
    • 3Abspielen
    • Twitter-F
    Hinweis: Nachdem Sie Änderungen an einem Erlebnis vorgenommen haben, stellen Sie sicher, dass Sie das Erlebnis erneut veröffentlichen , damit die Änderungen auf Websites angezeigt werden, auf denen das Erlebnis angezeigt wird.

    Hinzufügen einer Werbekomponente

    So fügen Sie eine Werbung zu einem Erlebnis hinzu:

    1. Klicken Sie auf der Komponentenauswahl auf Werbung .
    2. Geben Sie einen Namen für die Werbung ein.
    3. Geben Sie den Header-Anzeigencode und den Anzeigeneinheitencode ein Header Ad Code wird in der Erfahrung für alle Anzeigen geteilt.
    4. Wählen Sie aus, wann die Werbung angezeigt werden soll.
    5. Klicken Sie auf Save. Die Werbung wird auf dem Erlebnis erscheinen.
    Hinweis: Wenn Sie ein In-Page Experience mit mehreren Anzeigen haben, stellen Sie sicher, dass jeder Ad Unit Code eine eindeutige div ID hat. Möglicherweise müssen Sie die ID des div und des Container-Parameters von bearbeiten, adblock1 um dieser ID zu entsprechen.

    Hinzufügen einer Chat-Komponente

    Ein Chat-Widget eines Drittanbieters kann zu einem Erlebnis hinzugefügt werden, um Chat-Funktionen bereitzustellen. So fügen Sie eine Chat-Komponente zu einem Erlebnis hinzu:

    1. Klicken Sie bei der Komponentenauswahl auf Chat .
    2. Geben Sie einen Namen für die Chat-Komponente ein.
    3. Geben Sie den CHat-HTML-Code ein.
    4. Wählen Sie aus, wann der Chat angezeigt werden soll.
    5. Klicken Sie auf Save. Der Chat wird über das Erlebnis gerendert.

    Hinzufügen einer benutzerdefinierten HTML-Komponente

    Benutzerdefinierte Komponenten können verwendet werden, um ein eigenes benutzerdefiniertes HTML zu einem Erlebnis hinzuzufügen. Wenn Sie beispielsweise ein benutzerdefiniertes Lead-Formular mit einer Marketing-Automatisierungsplattform (MAP) erstellt haben, kann das Lead-Formular-HTML aus dem MAP mithilfe der Komponente Custom HTML zu einem In-Page Experience hinzugefügt werden.

    So fügen Sie einem Erlebnis benutzerdefiniertes HTML hinzu:

    1. Klicken Sie in der Komponentenauswahl auf Benutzerdefiniert .
    2. Geben Sie einen Namen für die HTML-Komponente ein.
    3. Geben Sie den benutzerdefinierten HTML-Code ein, der zum Erlebnis hinzugefügt werden soll.
    4. Wählen Sie aus, wann der HTML-Code angezeigt werden soll.
    5. Klicken Sie auf Save. Der HTML-Code wird für das Erlebnis gerendert.

    Wenn kein HTML eingegeben wird, wird ein HTML-Platzhalter angezeigt.

    Hinweis: Die Autorenumgebung der Galerie entfernt Skripte aus Sicherheitsgründen aus benutzerdefinierten HTML-Komponenten. Die Skripte werden ausgeführt, wenn die In-Page Experience in der Vorschau angezeigt und veröffentlicht wird. Dies könnte dazu führen, dass benutzerdefinierte HTML-Komponenten anders aussehen, insbesondere wenn sie viele JavaScript-Inhalte enthalten.

    Hinzufügen einer Image-Komponente

    So fügen Sie einem Erlebnis ein Bild hinzu:

    1. Klicken Sie auf der Komponentenauswahl auf Bild .
    2. Geben Sie einen Namen für das Bild ein.
    3. Um ein Bild anzugeben, haben Sie folgende Vorteile:
      • Ziehen Sie ein Bild aus dem Dateisystem und legen Sie es dann auf den Drop-Bereich
      • Klicken Sie auf „ Durchsuchen “ und wählen Sie ein Bild aus dem Dateisystem
      • Geben Sie die URL zu einer Remote-Image-Datei ein
    4. Stellen Sie die Bildgrößeein.
      • Das Bild wird auf Container skaliert - Bilder, die größer sind als der Container, in dem sie sich befinden, werden so verkleinert, dass sie passen. Kleinere Bilder werden nicht hochskaliert.
      • Breite und Höhe - Geben Sie manuell eine Breite und Höhe für das Bild ein
    5. Stellen Sie das Bild Ausrichtungein.
    6. Geben Sie eine Clickthrough-URL ein. Dies ist die URL, zu der Sie gehen müssen, wenn auf das Bild geklickt wird. Sie können den Link auch in einem neuen Fenster öffnen.
    7. Wählen Sie aus, wann das Bild angezeigt werden soll.
    8. Klicken Sie auf Speichern , um das Bild zum Erlebnis hinzuzufügen.

    Wenn kein Bild ausgewählt ist, wird ein Bildplatzhalter angezeigt.

    Hinzufügen einer Textkomponente

    So fügen Sie Text zu einem Erlebnis hinzu:

    1. Klicken Sie in der Komponentenauswahl auf Text .
    2. Geben Sie den Text ein, der zum Erlebnis hinzugefügt werden soll. Die Symbolleiste kann verwendet werden, um den Text zu formatieren.
    3. Geben Sie einen Namen für den Text ein.
    4. Wählen Sie aus, wann der Text angezeigt werden soll.
    5. Klicken Sie auf Speichern , um den Text hinzuzufügen. Der Text wird auf dem Erlebnis erscheinen.

    Wenn kein Text eingegeben wird, wird ein Textplatzhalter angezeigt.

    Hinzufügen einer 3Play-Komponente

    3Play-Komponenten können verwendet werden, um einem Erlebnis ein interaktives 3Play-Transkript hinzuzufügen. Diese Komponente funktioniert nur, wenn Ihre Videountertitel von 3Play Mediageneriert wurden. Wenn diese Komponente zu einem Erlebnis hinzugefügt wird, wird ein Link Interactive Transcript anzeigen angezeigt. Wenn Sie darauf klicken, wird das Videotranskript angezeigt. Im folgenden Beispiel wurde unter dem Videoplayer eine 3Play-Komponente hinzugefügt und so konfiguriert, dass sie während des Wiedergabe-Status angezeigt wird.

    Ein Live-Beispiel finden Sie unter diesem Link und spielen Sie ein Video ab.

    Hinweis: Um die 3Play-Komponente für Ihr Konto zu aktivieren, wenden Sie sich an Ihren Account Manager oder den Brightcove Support.

    So fügen Sie eine 3Play-Komponente zu einem Erlebnis hinzu:

    1. Klicken Sie in der Komponentenauswahl auf 3Play .
    2. Geben Sie einen Namen für die 3Play-Komponente ein.
    3. Geben Sie Ihre 3Play-Projekt-IDein. Um Ihre 3Play-Projekt-ID zu finden, melden Sie sich bei Ihrem 3Play-Konto an und klicken Sie im Seiten-Header auf den Link Kontoübersicht . Die Projekt-ID wird auf dieser Seite angezeigt.
    4. Wählen Sie aus, wann die 3Play-Komponente angezeigt werden soll.
    5. Klicken Sie auf Speichern , um die 3Play-Komponente hinzuzufügen. Die 3Play-Komponente wird in der Erfahrung angezeigt.

    Hinzufügen einer Twitter Feed-Komponente

    So fügen Sie einem Erlebnis einen Twitter-Feed hinzu:

    1. Klicken Sie in der Komponentenauswahl auf Twitter Feed .
    2. Geben Sie einen Namen für den Twitter-Feed ein.
    3. Erstellen Sie auf der Twitter-Website ein Widget. Informationen zum Erstellen eines Twitter-Widgets finden Sie in der Twitter-Entwicklerdokumentation.
    4. Kopieren Sie den HTML-Code für das Twitter-Widget, und fügen Sie ihn in die Twitter Feed-Einstellungen ein.
    5. Wähle aus, wann der Twitter Feed angezeigt werden soll.
    6. Klicken Sie auf Speichern , um den Twitter-Feed hinzuzufügen Der Twitter-Feed wird in der Erfahrung angezeigt.
