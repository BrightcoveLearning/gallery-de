Anpassen der Live-Event-Portal-Einstellungen

{{ page.description }}

Wenn eine Live Event Portal-Erfahrung erstellt wird, können mehrere vorlagenspezifische Einstellungen konfiguriert werden. Bearbeiten Sie die Erfahrung und klicken Sie auf, um die Einstellungen für die Live Event Portal-Vorlage zu konfigurieren ERSCHEINUNGSBILD UND VERHALTEN> Vorlageneinstellungen in der linken Navigation.

Die Live Event Portal-Vorlage verfügt über verschiedene Einstellungen, die für jeden Startseitenstatus konfiguriert werden können.

Klicken Sie auf eine Registerkarte, um die Einstellungen für jeden Status zu konfigurieren. Die Einstellungen werden auf der entsprechenden Seite für den Status der Startseite dokumentiert. Weitere Informationen finden Sie in einem der folgenden Themen:

  • Erstellen des Vorereignisstatus für eine Portalereigniserfahrung
  • Erstellen des Live-Ereignisstatus für eine Portalereigniserfahrung
  • Erstellen des Post-Event-Status für eine Portal-Event-Erfahrung
