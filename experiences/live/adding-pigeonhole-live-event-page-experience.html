Hinzufügen von Pigeonhole-Fragen und Antworten zu einem Live-Event-In-Page-Erlebnis

{{ page.description }}

Pigeonhole kann verwendet werden, um auf einfache Weise ereignisbasierte Interaktivität (Live-Fragen und -Antworten, Umfragen, Umfragen usw.) zu einem Live-Event-In-Page-Erlebnis hinzuzufügen. In der Regel wird eine neue Schublade für das Ereignis erstellt, und anschließend werden Sitzungen für die Live-Ereignisse erstellt. Sobald die Sitzung erstellt wurde, wird der Pigeonhole-Einbettungscode in die Live Event In-Page-Vorlage eingefügt. Der Einbettungscode bewirkt, dass der Pigeonhole Q & A-Pod neben dem Live-Event-Player angezeigt wird.

Erstellen eines Pigeonhole-Ereignisses und einer Sitzung

Sie sollten ein neues Pigeonhole für Ihr Live-Event erstellen und dann Sitzungen für jedes Video hinzufügen, für das Fragen und Antworten aktiviert sind.

  1. Melden Sie sich bei Ihrem Pigeonhole-Konto an.
  2. Klicken + Pigeonhole hinzufügen und klicken Sie auf das zu verwendende Konto.
  3. Geben Sie ein Veranstaltungsname und Passcode.
  4. Eingeben Start und Ende Termine für die Veranstaltung. Diese Daten steuern, wie lange das Pigeonhole aktiv sein wird.
  5. Wählen Sie a Zeitzone und Benutzerdefiniertes Branding Option (falls zutreffend).
  6. Klicken Fortsetzen.
  7. Nachdem das Ereignis erstellt wurde, wird die Agenda Element wird in der linken Navigation ausgewählt. Klicken Sitzung hinzufügen > Erstelle neu.
  8. Wählen Sie a Sitzungstyp von Fragen und Antworten.
  9. Geben Sie ein Sitzungsname.
  10. Eingeben Start und Ende Datum und Uhrzeit.
  11. Konfigurieren Sie die Q & A-Einstellungen wie angemessen.
  12. Klicken Sie auf Fragen und Antworten hinzufügen. Bestätigen Sie, dass die Sitzung erstellt wurde.

Generieren des Pigeonhole-Einbettungscodes

Damit das Pigeonhole in der Erfahrung angezeigt wird, wird der folgende Code mithilfe einer benutzerdefinierten Komponente in der Erfahrung hinzugefügt. Das Format des Codes lautet:

Gehen Sie folgendermaßen vor, um die Werte für den Ereignis-Passcode und die Sitzungs-ID abzurufen.

  1. Klicken Sie in Pigeonhole auf Agenda in der linken Navigation.
  2. Auf der Sitzungen Klicken Sie auf der Registerkarte auf den Dropdown-Pfeil neben dem Ereignis. Das Session-ID wird Angezeigt werden.
  3. Um den Ereignispasscode zu erhalten, suchen Sie den Ereignisnamen in der Kopfzeile. Der Ereignispasscode wird unter dem Ereignisnamen angezeigt.

Der obige Code mit Ihrem Ereignispasscode und Ihrer Sitzungs-ID wird Ihrer Erfahrung nach verwendet.

Hinzufügen einer Schublade zu einer Erfahrung

Brightcove empfiehlt, das Pigeonhole auf der rechten Seite Ihrer Live-Event-In-Page-Erfahrung hinzuzufügen. Derzeit zielt das Pigeonhole-Plugin speziell auf die rechte Seite ab.

Hinweis: Wenn Sie das Pigeonhole auf der linken Seite des Erlebnisses hinzufügen müssen, wenden Sie sich an Ihren Kundenbetreuer.

Das Hinzufügen einer Schublade zu einer Live-Event-In-Page-Erfahrung erfolgt mithilfe einer benutzerdefinierten Komponente. Befolgen Sie diese Schritte, um eine Schublade hinzuzufügen.

  1. Öffnen Sie das Galerie-Modul.
  2. Klicken Sie zum Bearbeiten auf die Live-Event-In-Page-Erfahrung.
  3. Klicken Sie auf das Plus-Symbol () auf der rechten Seite der Erfahrung.
  4. Klicken Benutzerdefiniert auf der Komponentenauswahl.
  5. Geben Sie ein Name für die HTML-Komponente.
  6. Fügen Sie das Pigeonhole-HTML aus dem vorherigen Abschnitt in das ein Benutzerdefinierter HTML-Code Feld.
  7. Wählen Sie aus, wann der HTML-Code angezeigt werden soll.
  8. Klicken speichern. Der HTML-Code wird auf der Erfahrung gerendert.
  9. Klicken Sie in der linken Navigation auf DIE EINSTELLUNGEN > Fortgeschrittene.
  10. Fügen Sie den folgenden Code in das JavaScript-URL-Feld ein. 
    https://static.pigeonhole.at/brightcove-plugin/phl-brightcove-gallery-inpage.js
  11. Klicken speichern.
  12. Veröffentlichen die Änderungen an der Erfahrung.

Wenn Sie eine Vorschau der Erfahrung anzeigen, sollte das Pigeonhole angezeigt werden.
