Erstellen des Post-Event-Status für eine Portal-Event-Erfahrung

{{ page.description }}

Die Vorlage für das Live Event Portal enthält Vorlagen für Vorveranstaltungen, Live-Events, Post-Event, Videoarchiv und Videodetailseiten-Vorlagen. Diese Seitenvorlagen können so angepasst werden, dass die Site vor, während und nach einem Live-Event anders aussieht. Der Status nach dem Ereignis wird auf der Site angezeigt, wenn das Live-Ereignis abgeschlossen ist.

Erstellen Sie eine neue Live Event Portal-Erfahrung, um den Status der Homepage nach dem Ereignis für eine Site zu erstellen. Klicken Sie im Galerie-Site-Editor auf SEITEN in der linken Navigation und dann klicken Startseite: Nach dem Event.

Der Status der Startseite nach dem Ereignis enthält die folgenden Inhaltsbereiche.

  • A - Home Logo und Name - Wird verwendet, um den Betrachter zur Startseite zurückzukehren. Hier gibt es zwei Platzhalter, sodass Sie beispielsweise ein Bild gefolgt von Text auswählen können.
  • B - Link zum Videoarchiv - Dient zum Anzeigen aller Videosammlungen, die für die Site erstellt wurden.
  • C - Twitter-Einstellungen - Bietet die Möglichkeit, ein Twitter-Feed-Widget zu verwenden. Sehen Konfigurieren eines Twitter-Feeds.
  • D - Hintergrundbild, Text und Links - Wird verwendet, um ein Hintergrundbild, Untertiteltext und Links hinzuzufügen
  • E - Name des Live-Ereignisses - Statische Textnachricht
  • F - Live-Stream hat Nachricht beendet - Statische Textnachricht
  • G - Einstellungen für die E-Mail-Anmeldung - Wird zum Konfigurieren der E-Mail-Anmeldung verwendet. Sehen E-Mail-Anmeldung konfigurieren.
  • H - Videos hinzufügen - Wird verwendet, um der Site Videosammlungen hinzuzufügen. Sehen Videos hinzufügen für mehr Informationen.
  • ICH - Video Grid Box - Wird verwendet, um Handlungsaufforderungen hinzuzufügen, einschließlich benutzerdefiniertem HTML oder Bildern. Sehen Konfigurieren von Handlungsaufrufen in einem Portal Experience für mehr Informationen.
  • J - Fußzeilenobjekte - Wird verwendet, um Text und Links zur Seitenfußzeile hinzuzufügen

Informationen zum Hinzufügen von Bildern, Links und Text zu einer Galerie-Seitenvorlage finden Sie unter Anpassen des Inhalts von Portal Experience Pages.

Hinweis: Die Farben der Website variieren je nach ausgewähltem Thema.

Konfigurieren eines Twitter-Feeds

Die Site kann mit einem auf der Seite eingebetteten Twitter konfiguriert werden. Überprüfen Sie die verfügbaren Einbettungstypen auf Twitter-Website. Eine Beispiel-Likes-Timeline ist unten dargestellt.

Gehen Sie folgendermaßen vor, um einen Twitter-Feed hinzuzufügen:

  1. Erstellen Sie ein Widget auf der Twitter-Site. Hilfe zum Erstellen eines Twitter-Widgets finden Sie in der Twitter Entwicklerdokumentation.
  2. Klick auf das Twitter-Einstellungen Link auf der Seite.
  3. Kopieren Sie den HTML-Code für das Twitter-Widget und fügen Sie ihn in das ein Twitter-Feed die Einstellungen.
  4. Klicken speichern.

E-Mail-Anmeldung konfigurieren

Die Site kann so konfiguriert werden, dass Site-Besucher ihre E-Mail-Adresse angeben können.

Gehen Sie folgendermaßen vor, um eine E-Mail-Anmeldung hinzuzufügen:

  1. Klick auf das E-Mail-Anmeldeeinstellungen Link auf der Seite.
  2. Konfigurieren Sie die E-Mail-Anmeldeeigenschaften nach Bedarf.
    • Zeigen Sie kein E-Mail-Lead-Formular an - Zeigen Sie das E-Mail-Formular nicht an
    • Verwenden Sie das mit der Vorlage gelieferte E-Mail-Formular - Wählen Sie diese Option, um das mit der Vorlage gelieferte Standard-E-Mail-Formular zu verwenden
    • Verwenden Sie benutzerdefiniertes HTML, das von Ihrer Marketing-Automatisierungssoftware bereitgestellt wird - Wählen Sie diese Option, um Ihr benutzerdefiniertes Formular-HTML für die E-Mail-Erfassung zu verwenden
  3. Klicken speichern.
  4. Nachdem Sie die E-Mail-Einstellungen konfiguriert haben, kehren Sie zur Seite zurück und klicken Sie auf E-Mail-Anmeldemeldung Verknüpfen Sie und legen Sie den Text fest, der über dem E-Mail-Anmeldefeld angezeigt wird.

Alle vom E-Mail-Formular erfassten Daten können mit dem exportiert werden Galerie-Leads exportieren Option im Zielgruppenmodul. Mit der benutzerdefinierten HTML-Option erfasste Leads werden in Ihrer Marketing-Automatisierungssoftware angezeigt.

Videos hinzufügen

Drücke den + Videos hinzufügen Link zum Hinzufügen von Videosammlungen zur Site. Informationen zum Erstellen von Sammlungen finden Sie unter Hinzufügen von Videos zu einer Portal-Erfahrung.

Die erste Sammlung in der Liste der Sammlungen wird auf der Homepage nach dem Ereignis angezeigt. Um alle Sammlungen anzuzeigen, klicken Sie auf eines der Videos auf der Startseite oder konfigurieren Sie die Videoarchiv Link Objekt, damit Betrachter darauf klicken können, um alle Sammlungen anzuzeigen.
