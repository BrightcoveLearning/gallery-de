    Erstellen eines In-Page-Erlebnisses für Live-Events

    In diesem Thema erfahren Sie, wie Sie den Inhalt eines In-Page-Erlebnisses für Live-Events anpassen können.

    Gallery bietet eine In-Page-Vorlage für Live-Events, die verwendet werden kann, um Live-Events im Rahmen eines In-Page Experience zu liefern. Brightcove bietet verschiedene Optionen zur Übertragung von Live-Streams. Eine Liste aller Optionen finden Sie unter Bereitstellen von Live-Streams . Eine detaillierte Reihe von Schritten zur Verwendung des Live-Moduls zur Übertragung eines Live-Events finden Sie unter:

    • Schritt für Schritt: Verwenden des Live-Moduls mit Telestream Wirecast
    • Schritt für Schritt: Verwenden des Live-Moduls mit Open Broadcaster Software (OBS)
    Hinweis: In-Page-Erlebnisse bei Live-Events spielen automatisch in Desktop-Browsern ab, auch wenn der Player so konfiguriert ist, dass er nicht automatisch abgespielt wird. Das ist von Grund auf. Die Galerie zwingt den Spieler zum Spielen, sobald der Countdown abgeschlossen ist, sodass Benutzer, die auf die Seite sitzen und auf den Start des Events warten, sie ansehen können, ohne erneut auf Play klicken zu müssen. In-Page-Erlebnisse bei Live-Events werden nicht automatisch auf iOS- oder Android-Geräten abgespielt.

    Um ein neues In-Page-Erlebnis für Live-Events zu erstellen, erstellen Sie ein neues In-Page-Erlebnis und wählen Sie die In-Page-Vorlage für Live-Events aus.

    Die In-Page-Vorlage für Live-Ereignisse enthält einige Einstellungen und Inhaltselemente, die in den anderen In-Page Experience-Vorlagen nicht vorhanden sind.

    1. Wird jetzt den Zuschauern angezeigt
    2. Status der Live-Veranstaltung
    3. Soziales Teilen
    4. Bild-Platzhalter
    5. Countdown der Veranstaltung
    6. Platzhalter für Text
    7. Kalender-Erinnerung
    8. Platzhalter für Videos

    Wird jetzt den Zuschauern angezeigt

    Die Einstellung „ Jetzt wird den Betrachtern angezeigt “ wird verwendet, um zu konfigurieren, welcher Live-Ereignisstatus den Zuschauern angezeigt wird.

    Klicken Sie auf Ändern und wählen Sie den Status aus, der angezeigt werden soll. Nach der Änderung dieser Einstellung muss die Erfahrung NICHT erneut veröffentlicht werden, es kann jedoch eine Minute dauern, bis die Statusänderung auf der Produktionsstätte erscheint.

    Hinweis: Sie, der Herausgeber, sind dafür verantwortlich, den Status des Live-Ereignisses zu ändern. Die Galerie wird niemals automatisch den Status des Live-Ereignisses wechseln.

    Live-Event-Staaten

    Die Erfahrungszustände für ein Live-Event In-Page Experience unterscheiden sich von den anderen In-Page Experience-Typen. Das Erscheinungsbild jedes dieser Zustände kann durch Klicken auf die Links oben auf der Seite angepasst werden.

    Live-Event In-Page Experiences haben die folgenden Zustände:

    • PRE-EVENT - Wie das Live-Erlebnis vor Beginn des Live-Events erscheinen wird
    • LIVE - Wie das Erlebnis während des Live-Events erscheinen wird
    • POST EVENT - Wie das Erlebnis erscheinen wird, wenn das Live-Event vorbei ist

    Um die verschiedenen Erlebnisstände zu bearbeiten, klicken Sie auf einen Status und fügen Sie dann nach Bedarf Komponenten zur Verfügung. Das Menü LAYOUT bietet weitere Anpassungsoptionen für jeden Bundesstaat.

    Soziales Teilen

    Die Social-Sharing-Optionen bieten Links, um die Veranstaltung und die Videos mit Ihren Social-Media-Konten zu teilen.

    Um die Social-Sharing-Einstellungen zu konfigurieren, klicken Sie auf den Link Freigabe konfigurieren oder klicken Sie im linken Navigationsbereich auf EINSTELLUNGEN > Soziales . Die sozialen Einstellungen bestehen aus:

    • Event-Shar
    • Video-Sharing

    Event-Shar

    Die Einstellungen für die Freigabe von Veranstaltungen ermöglichen es dem Zuschauer, die Veranstaltungsseite mit Social-Media-Konten zu teilen. Befolgen Sie diese Schritte, um die Ereignisdetails zu konfigurieren:

    1. Um ein Ereignisbild hinzuzufügen, klicken Sie auf Bild hinzufügen und dann:
      • Ziehen Sie ein Bild aus dem Dateisystem und legen Sie es dann auf den Drop-Bereich
      • Klicken Sie auf „ Durchsuchen “ und wählen Sie ein Bild aus dem Dateisystem
      • Geben Sie die URL zu einer Remote-Image-Datei ein
    2. Geben Sie einen Veranstaltungstitel und eine Ereignisbeschreibungein.
    3. Wählen Sie aus, wie die Optionen zum Teilen von Veranstaltungen angezeigt werden, Text teilen oder die sozialen Symbole anzeigen.
    4. Wählen Sie die verfügbaren Sharing-Optionen aus.
    5. Geben Sie die Seiten-URL ein, in die das Erlebnis eingebettet wird.
    6. Klicken Sie auf Erstellen. Dadurch werden die Meta-Tags generiert, die zum <head> Abschnitt Ihrer Seite hinzugefügt werden können. Diese Funktion ermöglicht es sozialen Websites, das Ereignisbild, den Titel und die Beschreibung anzuzeigen, wenn die Zuschauer Ihre Erfahrungen teilen.

    Video-Sharing

    Die Einstellungen für die Videofreigabe ermöglichen es dem Zuschauer, die Videos mit Social-Media-Konten zu teilen. Befolgen Sie diese Schritte, um die Videofreigabe zu konfigurieren:

    1. Wählen Sie aus, wie die Optionen für das Teilen von Videos angezeigt werden, Text freigeben oder die sozialen Symbole anzeigen.
    2. Wählen Sie die verfügbaren Sharing-Optionen aus.
    3. Klicken Sie auf Save, um die Änderungen zu speichern.

    Bild-Platzhalter

    Die Status PRE-EVENT und POST EVENT bieten die Möglichkeit, ein Hero-Bild anzuzeigen.

    So fügen Sie ein Hero-Bild hinzu:

    1. Klicken Sie auf den Link Bild hinzufügen .
    2. Geben Sie einen Namen für das Bild ein (nur ASCII-Zeichen sind erlaubt).
    3. Um ein Bild anzugeben, haben Sie folgende Vorteile:
      • Ziehen Sie ein Bild aus dem Dateisystem und legen Sie es dann auf den Drop-Bereich
      • Klicken Sie auf „ Durchsuchen “ und wählen Sie ein Bild aus dem Dateisystem
      • Geben Sie die URL zu einer Remote-Image-Datei ein
    4. Stellen Sie die Bildgröße ein.
      • Bild zuschneiden - Schneidet das Bild in der Mitte so zu, dass es den gesamten 16:9 -Bereich ausfüllt, aber ein Teil des oberen/unteren oder linken Rechts-Verhältnisses geht je nach Bildausschnitt des Bildes verloren
      • Bild in Komponentengrenzen enthalten - Enthält das Bild innerhalb der Grenzen des 16:9 -Bereichs, aber je nach Seitenverhältnis wird etwas Leerzeichen oberem/unterhalb oder links/rechts vom Bild angezeigt
      • Bild dehnen - Ignoriert das Seitenverhältnis des Bildes und zwingt es, den 16:9 -Bereich vollständig zu füllen; dies wird das Bild verzerren
      • Originalbildgröße - Verwenden Sie die ursprüngliche Bildgröße, es wird keine Größenänderung durchgeführt; reagiert nicht
    5. Wählen Sie die Status aus, in denen das Bild angezeigt werden soll.
    6. Klicken Sie auf Speichern , um das Bild zum Erlebnis hinzuzufügen.

    Countdown der Veranstaltung

    Der Countdown-Timer zeigt die Anzahl der Monate, Tage, Stunden, Minuten und Sekunden vor Beginn des Live-Events an. Ein Beispiel für eine Countdown-Uhr ist unten dargestellt.

    Hinweis: Der Stil des Countdown-Timers kann durch Klicken auf STYLE im linken Navigationsbereich gestaltet werden.

    Führen Sie die folgenden Schritte aus, um eine Countdown-Uhr hinzuzufügen:

    1. Klicken Sie auf der Seite auf den Link „ Ereignis-Countdown konfigurieren “. Die Seite „Veranstaltungseinstellungen“ wird geöffnet.
    2. Markieren Sie „ Ereignis-Countdown anzeigen
    3. Stellen Sie das Startdatum, die Startzeit und die Ereigniszeitzoneein.
    4. Geben Sie eine Übergangsnachricht Diese Meldung wird angezeigt, wenn der Countdown-Timer Null erreicht, aber bevor der Live-Stream beginnt.
    5. Klicken Sie auf Save.

    Platzhalter für Text

    Die Status PRE-EVENT und POST EVENT bieten die Möglichkeit, statischen Text anzuzeigen.

    So fügen Sie statischen Text hinzu:

    1. Klicken Sie auf den Link Text hinzufügen .
    2. Geben Sie den Text ein, der zum Erlebnis hinzugefügt werden soll. Die Symbolleiste kann verwendet werden, um den Text zu formatieren.
    3. Klicken Sie auf Speichern , um den Text zur Erfahrung hinzuzufügen.

    Kalender-Erinnerung

    Eine Kalendererinnerung kann so konfiguriert werden, dass Benutzer auf einen Link klicken können, um eine Kalendererinnerung herunterzuladen. Ein Beispiel für eine Kalendererinnerung ist unten dargestellt.

    Wenn Sie auf die Erinnerung klicken, werden Kalenderoptionen angezeigt.

    Hinweis: Der Stil der Kalendererinnerung kann durch Klicken auf STYLE im linken Navigationsbereich gestaltet werden.

    Um eine Kalendererinnerung hinzuzufügen, gehen Sie folgendermaßen vor:

    1. Klicken Sie auf der Seite auf den Link Configure Calendar Reminder . Die Seite „Veranstaltungseinstellungen“ wird geöffnet.
    2. Aktivieren Sie „Zum Kalender hinzufügen“ -Erinnerung anzeigen.
    3. Stellen Sie das Enddatum, die Endzeit und den Ereignistitelein. Die Veranstaltungsbeschreibung und der Veranstaltungsort sind optional.
      Hinweis: Die Benutzeroberfläche wird etwas anders aussehen, wenn die Option Ereignis-Countdown anzeigen ausgewählt ist.
    4. Klicken Sie auf Save.

    Platzhalter für Videos

    Während der Status PRE-EVENT, LIVE und POST EVENT können verschiedene Videos angezeigt werden. Video Cloud-Videos und YouTube-Videos können dem Erlebnis hinzugefügt werden.

    Hinweis: Bevor YouTube-Videos zu einem Erlebnis hinzugefügt werden können, müssen Sie die YouTube-Einstellungen konfigurieren.

    So wählen Sie die anzuzeigenden Videos aus:

    1. Klicken Sie auf den Link Video auswählen oder klicken Sie im linken Navigationsbereich auf VIDEOS. Das Dialogfeld „ Video auswählen “ wird angezeigt.
    2. Wählen Sie den Status aus, für den Sie das Video auswählen möchten:
      • Pre-Event
      • Live
      • Post-Event
    3. Wählen Sie das Video Source:
      • Video Cloud - Video Cloud-Videoinhalte werden verwendet
      • YouTube - YouTube-Videoinhalte werden verwendet
    4. Suchen Sie nach einem Video, klicken Sie darauf, um es auszuwählen, und klicken Sie dann auf Auswählen. Das Video wird zur Videoliste hinzugefügt.
    5. Klicken Sie auf den Live- oder Post-Event-Bereich oder klicken Sie auf „Videos auswählen “, um zusätzliche Videos auszuwählen.

    Interaktionen und Begleitkomponenten können zu Videos vor und nach der Veranstaltung hinzugefügt und bei der Wiedergabe des Videos angezeigt werden. Interaktionen umfassen Links und Karten und werden beim Abspielen eines Videos als Teil des Players angezeigt. Zu den Begleitkomponenten gehören HTML, Bilder und Textobjekte und werden außerhalb des Players angezeigt.

    Hinweis: Interaktionen und Begleitkomponenten sind für das Live-Video nur verfügbar, wenn es sich um ein VOD-Asset handelt.

    Videos, die mit Interaktivität konfiguriert wurden, zeigen ein Symbol in der Spalte Interaktivität an, das die Art der Interaktion (en) angibt. Informationen zum Hinzufügen und Konfigurieren von Interaktionen und Companion Components finden Sie unter:

    • Hinzufügen von Interaktionen zu einem In-Page-Erlebnis
    • Konfigurieren von Companion-Komponenten für ein In-Page-Erlebnis

    Auswahl des Live-Videos

    Wenn der Live-Status ausgewählt ist, muss eine Video Cloud-Video-ID oder Referenz-ID für das Live-Video-Asset eingegeben werden. Die Quelle für das Live-Video wird immer Video Cloud sein.

    Wenn Sie die Option Referenz-ID auswählen, können Sie die Referenz-ID für das Live-Video im Medienmodul ändern, ohne die Video-ID für das Erlebnis ändern zu müssen. Sie können beispielsweise das aktuelle Live-Video mit einer Referenz-ID des Live-Eventsbezeichnen. Wenn Sie das Live-Erlebnis erstellen, wählen Sie Referenz-ID aus und geben Sie Live-Eventein.

    Jetzt, wann immer es ein neues Live-Event gibt, verwenden Sie das Medienmodul, um die Referenz-ID für dieses Video auf Live-Eventfestzulegen.

    Durch die Verwendung der Referenz-ID muss das Live-Erlebnis nicht bearbeitet und erneut veröffentlicht werden, wenn ein neues Ereignis beginnt. Wenn ein neues Ereignis gestartet wird, verwenden Sie das Medienmodul, um die Video-Referenz-ID des Live-Video-Assets zu bearbeiten.

    Suchen nach Videos

    Das Suchfeld kann verwendet werden, um nach Videos zu suchen. Bei der Suche nach Video Cloud-Videos sucht Gallery den Videonamen, Kurz- und Langbeschreibungen, Tags und benutzerdefinierten Feldern. Wenn Sie YouTube-Videos durchsuchen, verwendet Gallery die YouTube-Such-AP

    Klicken Sie auf Weitere Suchoptionen , um eine genauere Kontrolle darüber zu erhalten, welche Felder durchsucht werden, und um Text einzubeziehen und auszuschließen. Beachten Sie, dass diese Optionen nur verfügbar sind, wenn eine Video Cloud-Quelle verwendet wird.
