    Schritt für Schritt: Erstellen und Veröffentlichen eines Portal Experience

    Der Zweck dieses Themas besteht darin, Ihnen zu zeigen, wie Sie mit Brightcove Gallery ein ansprechendes Portal Experience erstellen können.
     

    Nach Abschluss dieses Themas sollten Sie in der Lage sein:

    • Gallery-Einstellungen konfigurieren
    • Verwenden Sie Gallery, um ein neues Portal Experience zu
    • Videos zu einem Portal Experience hinzufügen
    • Anpassen einer Portalerfahrung
    • Vorschau und Veröffentlichung eines Portal Experience
    Hinweis: Eine Schritt-für-Schritt-Anleitung zum Erstellen eines In-Page-Erlebnisses finden Sie unter Schritt-für-Schritt: Erstellen und Veröffentlichen eines In-Page-Erlebnisses.

    Zielgruppe

    Jeder Herausgeber von Video Cloud, der ansprechende Portal-Erlebnisse erstellen muss, ohne dass irgendeine Art von kundenspezifischer Entwicklung erforderlich ist.

    Voraussetzungen

    • Ein Video Cloud-Konto mit Zugriff auf das Gallery-Modul
    • Video Cloud-Kenntnisse
    • Die in diesem Thema verwendeten Beispielvideos stehen bei Bedarf zum Download zur Verfügung
    • Hilfreich, um das Building Portal zu besuchen/besuchen Erfahrungen mit Galerie-Schulungen

    Bevor Sie beginnen

    Um die bereitgestellten Beispielvideos zu verwenden, führen Sie die folgenden Schritte aus.

    1. Laden Sie die Beispieldateien herunter.
    2. Laden Sie die Videos in Ihr Video Cloud-Konto hoch.
    3. Füge dem Chickadee-Video ein Tag vonbird in flight' ' hinzu.
    4. Erstellen Sie drei Wiedergabelisten, und fügen Sie die Videos den Wiedergabelisten wie folgt hinzu.
      Video Wiedergabeliste
      Clown Fish
      Crab
      Dolphins
      Leafy Seadragon
      Lionfish
      Sea Turtle
      Seahorse      		 Sea Creatures
      Chickadee
      Peacock
      Woodpecker      		 Birds
      Elephant
      Tiger
      Zebra      		 Animals
    5. Füge einige der Videos einige verwandte Link-URLs hinzu. In einem späteren Abschnitt werden wir das Portal so konfigurieren, dass zugehörige Links für Videos angezeigt werden.
    6. Erstellen Sie mit dem Spieler-Modul einen neuen Player namens Gallery Player.
    7. Konfigurieren Sie den Player so, dass er den Titel und die Beschreibung NICHT anzeigt. Veröffentliche den Spieler.

    Erstellen eines Portalerlebnisses

    Um ein Portal Experience mit Brightcove Gallery zu erstellen, führen Sie die folgenden Aufgaben aus:

    • Konfigurieren Sie die Galerie mit einem Standard-Player
    • Erstellen Sie einen benutzerdefinierten Domainnamen
    • Erstellen Sie ein neues Portal Experience
    • Bearbeiten Sie die Site-Details
    • Hinzufügen von Objekten zu einer Seitenvorlage
    • Vorschau auf das Erlebnis
    • Erstellen Sie Sammlungen und fügen Sie den Sammlungen Videos hinzu
    • Ändern des Site-Stils
    • Website-Suche aktivieren
    • Passen Sie die sozialen Einstellungen an
    • Anpassen von Video- und Wiedergabeeinstellungen
    • Zuweisen einer benutzerdefinierten Domäne
    • Veröffentlichen Sie das Erlebnis

    Konfigurieren von Gallery mit einem Standard-Player

    Brightcove Gallery kann mit einem Standard-Videoplayer konfiguriert werden, der auf Ihren Gallery-Websites verwendet wird. Wenn kein Standardplayer angegeben ist, ist der Brightcove Default Player der Standardwert. Um die Galerie mit einem Standardplayer zu konfigurieren, führen Sie diese Schritte aus.

    1. Melden Sie sich bei Video Cloud Studioan.
    2. Der aktuelle Video Cloud-Kontoname erscheint auf der Seite oben rechts. Wenn Sie mehrere Konten haben, klicken Sie auf das Dropdown-Menü Kontoauswahl und wählen Sie das Konto aus, in dem Sie das Erlebnis erstellen möchten. Hierbei sollte es sich um das Konto handeln, das auch die zu verwendenden Videos enthält.
    3. Klicken Sie im Navigationsmenü auf Gallery .
    4. Klicken Sie oben auf der Seite auf den Link Einstellungen .
    5. Wählen Sie den auf Ihren Gallery-Websites verwendeten Player. Wenn Sie die Schritte befolgen, wählen Sie den Galerie-Player aus, den Sie zuvor erstellt haben.
      Hinweis: Als Best Practice empfehlen wir dir, engagierte Spieler für deine Galerie-Erlebnisse zu erstellen und zu gestalten.
    6. Klicken Sie auf Save.

    Erstellen eines benutzerdefinierten Domainnamens

    Wenn Sie ein neues Portal Experience erstellen, weist Gallery ihm eine zufällige URL zu (d. h. site-12345.bcvp0rtal.com). Portalerlebnisse kann eine von Brightcove bereitgestellte Domain zugewiesen werden (was wir in diesem Thema tun werden) oder Sie können Ihre eigene benutzerdefinierte Domain verwenden. Gehen Sie folgendermaßen vor, um eine neue von Brightcove bereitgestellte Domäne zu erstellen.

    1. Klicken Sie auf der Seite „ Einstellungen “ im linken Navigationsbereich auf Benutzerdefinierte Domains & SSL .
    2. Klicken Sie auf Create Custom Domain.
    3. Wählen Sie entweder eine Domäne von brightcovegallery.com oder gallery.video aus.
    4. Geben Sie eine benutzerdefinierte Domain ein. Verwenden Sie Ihren eigenen Namen oder Firmennamen. Wenn die Domain nicht verfügbar ist, wird eine Meldung angezeigt. Geben Sie einen anderen Namen ein.
    5. Wenn Sie eine gültige Domäne eingegeben haben, klicken Sie auf Speichern.
    Hinweis: Die benutzerdefinierte Domäne wird dem Portal in einem späteren Abschnitt zugewiesen.

    Erstellen eines neuen Portalerlebnisses

    Gehen Sie folgendermaßen vor, um ein neues Gallery Portal Experience zu erstellen.

    1. Klicken Sie oben auf der Seite auf den Link Erlebnisse .
    2. Klicken Sie auf „ Neue Erfahrung“
    3. Klicken Sie oben auf der Seite auf die Registerkarte Portal . Eine Liste der Portalvorlagen wird angezeigt. Suchen Sie die Marquee-Vorlage und klicken Sie auf Auswählen.
      Hinweis: Sobald Sie eine Vorlage ausgewählt haben, kann sie nicht mehr geändert werden. Weitere Informationen Galerie-Vorlagen finden Sie im Thema Überblick über Galerieportal-Vorlagen .
    4. Geben Sie Animal World für den Titel der Erfahrungein. Für das Pre-Populate-Erlebnis? , wählen Sie Nein dankeaus.
    5. Klicken Sie auf Erfahrung erstellen.

    Der Gallery Site Editor wird geöffnet. Der Galerie-Site-Editor wird verwendet, um das Erlebnis anzupassen und bietet die Möglichkeit, Seitenlayouts zu verwalten, Videos hinzuzufügen, Stile auszuwählen und andere Einstellungen für das Portal zu konfigurieren. Auf alle Anpassungseinstellungen wird über das linke Navigationsmenü zugegriffen.

    Bearbeiten der Site-Details

    Der Titel, die Beschreibung und die Spracheinstellungen der Erfahrung können mit dem Galerie-Site-Editor bearbeitet werden.

    1. Klicken Sie in der linken Navigationsleiste auf ÜBERBLICK .
    2. Fügen Sie für die Beschreibung der Erfahrung Animal World Video Portalhinzu.
    3. Klicken Sie auf Save.

    Objekte einer Seitenvorlage hinzufügen

    Jede Portalvorlage enthält eine Reihe von Seitenvorlagen, die angepasst werden können. Die bereitgestellten Seitenvorlagen hängen von der Vorlage ab, die bei der ersten Erstellung des Erlebnisses ausgewählt wurde. Die Marquee-Vorlage enthält Home- , Playback- und Kategorieseitenvorlagen. Diese Seiten können am linken Rand als Teil des PAGES-Menüs angezeigt werden.

    Weitere Informationen zum Hinzufügen von Objekten zu Seitenvorlagen finden Sie unter Anpassen des Inhalts von Portalerlebnisseiten.

    Um einer Seitenvorlage Objekte hinzuzufügen, gehen Sie wie folgt vor:

    1. Klicken Sie in der linken Navigationsleiste auf SEITEN .
    2. Klicken Sie am linken Rand auf die Seite Home. Plus-Symbole () zeigen Bereiche auf der Seite an, in denen Objekte hinzugefügt werden können. Statische Textobjekte auf der Seite können durch Anklicken bearbeitet werden.
    3. Klicken Sie oben auf der Seite auf + Bild oder Text hinzufügen.
    4. Klicken Sie auf die Registerkarte „ Bild “.
    5. Klicken Sie auf Durchsuchen , durchsuchen und wählen Sie Animal World Logo.pngaus. Diese Datei ist Teil der Beispielanlagen. Klicken Sie auf Save.
    6. Bestätigen Sie, dass das Bild in der Kopfzeile erscheint.
    7. Scrollen Sie zum Ende der Seite. Klicken Sie auf [IHRE MARKE]. Geben Sie als Titel Animal World ein, und klicken Sie auf Save.
    8. Klicken Sie am linken Rand auf die Seiten Wiedergabe und Kategorie . Beachten Sie, dass das Kopfzeilenbild und der Titeltext auch auf diesen Seiten angezeigt werden. Navigieren Sie zurück zur Seite Home.

    Vorschau der Erfahrung

    In der Vorschau eines Erlebnisses können Sie sehen, wie das Portal auf Mobil-, Tablet- und Desktop-Geräten angezeigt wird. Sie sollten eine Vorschau des Portals anzeigen, während Sie die Schritte zum Anpassen durchlaufen. Das Portal kann in einer neuen Browserregisterkarte in der Vorschau angezeigt werden oder ein gemeinsam nutzbarer Link kann generiert werden, damit andere das Portal in der Vorschau anzeigen können. Weitere Informationen zur Vorschau und Veröffentlichung eines Portalerlebnisses finden Sie unter Vorschau und Veröffentlichung eines Portalerlebnisses.

    Befolgen Sie diese Schritte, um eine Vorschau des Portals anzuzeigen.

    1. Klicken Sie beim Bearbeiten der Website mit dem Galerie-Site-Editor auf Vorschau und dann auf Responsive Preview.
    2. Die Vorschau wird in einem neuen Browser-Tab geöffnet.
    3. Schließen Sie den Tab Site-Vorschau.

    Sammlungen erstellen und den Sammlungen Videos hinzufügen

    Mit dem Galerie-Site-Editor können Sie auswählen, welche Videos auf dem Portal angezeigt werden sollen. Beim Erstellen eines Portal Experience werden Videos in Sammlungen organisiert. Sammlungen werden basierend auf Video Cloud-Video-Metadaten erstellt und können Folgendes umfassen:

    • Video Cloud-Wiedergabelisten
    • Schlüsselwörter
    • Benutzerdefinierte Metadaten
    • Tags

    Videos von einem YouTube-Konto können auch zu Sammlungen hinzugefügt werden. Weitere Informationen zum Erstellen von Sammlungen und zum Hinzufügen von Videos zu Ihrer Website finden Sie unter Hinzufügen von Videos zu einem Portal Experience.

    Um Videos zu einem Portal Experience hinzuzufügen, führen Sie diese Schritte aus.

    1. Klicken Sie in der linken Navigationsleiste auf VIDEOS .
    2. Nennen Sie die erste Sammlung Birds und drücken Sie dann die Eingabetaste
    3. Klicken Sie auf den Namen der Kollektion, um den Kollektionseditor
    4. Wählen Sie Video Cloud als Sammlungsquelle aus und wählen Sie für die Kriterien Playlist aus.
    5. Klicke auf die Birds Playlist. Alle Videos in der Birds-Wiedergabeliste werden in der Videeliste angezeigt. Klicken Sie auf Save.
      Hinweis: Bei Verwendung einer Playlist-Videozuweisung basiert die Reihenfolge der Videos auf den Playlist-Einstellungen, die im Medienmodul festgelegt wurden.
    6. Erstelle eine weitere Sammlung namens Animals mit der Animes-Wiedergabeliste als Kriterium.
    7. Erstelle eine weitere Sammlung namens Sea Creatures unter Verwendung der Sea Creatures -Playlist als Kriterium.
    8. Erstellen Sie eine weitere Sammlung mit dem Namen Birds in Flight Wählen Sie Benutzerdefiniert als Kriterium aus.
    9. Damit die Videos enthalten sein sollen, wählen Sie Tags aus und fügen Sie ein Tag von 'bird in flight hinzu. Drücken Sie die Eingabetaste.
    10. Bestellen Sie Videos von Most Recent.
    11. Bestätige, dass das Chickadee-Video in der Liste der Videosammlung angezeigt wird. Klicken Sie auf Save.
    12. Bestätigen Sie, dass 4 Sammlungen erstellt wurden.
      Hinweis: Sie können die Reihenfolge der Sammlungen ändern, indem Sie auf die Sammlung klicken und sie an eine neue Position ziehen.

    Ändern des Site-Stils

    Die Stileinstellungen können verwendet werden, um die Website-Farben und Schriftarten anzupassen, die für das Erlebnis verwendet werden. Weitere Informationen zum Ändern des Website-Stils finden Sie unter Anpassen des Stils eines Portalerlebnisses.

    Gehen Sie folgendermaßen vor, um den Site-Style zu ändern.

    1. Klicken Sie in der linken Navigationsleiste auf STIL .
    2. Ein Theme ist eine Reihe von Farben und Schriftarten, die auf eine Vorlage angewendet werden können. Wählen Sie ein Themeaus. Dieses Portal wird das Triton-Thema verwenden.
    3. Wählen Sie eine zu verwendende primäre und sekundäre Schriftart aus. Dieses Portal wird die Open Sans Schriftart verwenden.
    4. Klicken Sie oben im Stilmenü auf den Link < Zurück zu Erfahrung .

    Zusätzliche Styling-bezogene Einstellungen sind auch als Teil der Einstellungen für ERSCHEINUNG UND VERHALTEN verfügbar.

    1. Klicken Sie in der linken Navigationsleiste > auf Einstellungen für ERSCHEINUNGSBILD UND VERHALTEN
    2. Wählen Sie Das vorgestellte Video auf das erste Video der ausgewählten Kategorie ändernaus.
      Hinweis: Die verfügbaren Vorlageneinstellungen basieren auf der Vorlage, die bei der Erstellung des Erlebnisses ausgewählt wurde. Nicht alle Vorlagen haben Vorlageneinstellungen.
    3. Klicken Sie auf Save.

    Weitere Informationen zu den Einstellungen für Aussehen und Verhalten finden Sie unter Anpassen des Erscheinungsbilds und des Verhaltens eines Portalerlebnisses.

    Aktivieren der Website-Suche

    Die Einstellungen für die Websitesuche können zum Aktivieren und Anpassen der Websitesuchfunktionen verwendet werden. Durch das Aktivieren der Websitesuche werden den Webseiten Suchfunktionen hinzugefügt, mit denen Betrachter Ihren Content durchsuchen können. Weitere Informationen zum Anpassen der Websitesuche finden Sie unter Anpassen der Website-Funktionen einer Portal-Erfahrung.

    Um die Website-Suche zu aktivieren, führen Sie diese Schritte aus.

    1. Klicken Sie im linken Navigationsbereich auf > SITE-FUNKTIONEN Websitesuche
    2. Aktivieren Sie die Option Websitesuche aktivieren.
    3. Klicken Sie auf Speichern , um die Einstellungen zu speichern.

    Anpassen der sozialen Einstellungen

    Die Social-Einstellungen bieten Zuschauern die Möglichkeit, Ihre Inhalte einfach auf beliebte Social-Media-Sites zu teilen. Sie können Ihrem Publikum auch ermöglichen, schnell auf Ihre Social-Media-Websites zuzugreifen. Weitere Informationen zum Anpassen der sozialen Einstellungen finden Sie unter Anpassen der sozialen Einstellungen für ein Portalerlebnis.

    Gehen Sie folgendermaßen vor, um die Social-Einstellungen zu konfigurieren.

    1. Klicken Sie im linken Navigationsbereich auf SITE FEATURES > Social .
    2. Aktivieren Sie alle Sharing Options. Dies fügt Social-Sharing-Symbole neben dem Spieler hinzu.
    3. Wählen Sie die Option Dunkel-Symbol bei Licht aus.
    4. Klicken Sie auf Save.
    5. Klicken Sie oben auf der Seite auf den Tab Bleiben Sie Connected . Wenn Sie Ihre Kontoinformationen für soziale Medien angeben, erscheinen auf der Gallery-Website in der Fußzeile Symbole von Seiten, mit denen Betrachter den direkten Zugriff auf Ihre Social-Media-Websites erhalten.
    6. Geben Sie Ihre Social-Media-Informationen ein oder geben Sie einfach hellcove für den Facebook-Namen , das Twitter-Handle und den LinkedIn-Benutzernamen ein
    7. Wählen Sie die Option Licht-Symbol bei Dunkelheit aus.
    8. Klicken Sie auf Speichern , um die Einstellungen zu speichern.

    Anpassen von Video- und Wiedergabeeinstellungen

    Die Video- und Wiedergabeeinstellungen können verwendet werden, um einen benutzerdefinierten Player für das Erlebnis auszuwählen, ein Lead-Formular hinzuzufügen und andere videobezogene Einstellungen anzupassen. Weitere Informationen zum Anpassen der sozialen Einstellungen finden Sie unter Anpassen der Video- und Wiedergabeeinstellungen eines Portalerlebnisses.

    Gehen Sie folgendermaßen vor, um die Einstellungen für Video und Wiedergabe anzupassen:

    1. Klicken Sie im Galerie-Site-Editor auf VIDEO UND > PLAYBACK Player and Lead Form.
    2. Stellen Sie sicher, dass die beiden Einstellungen für das Spielerverhalten überprüft Klicken Sie bei Bedarf auf Speichern .
    3. Klicken Sie in der linken Navigationsleiste auf > VIDEO UND PLAYBACK Video .
    4. Wählen Sie den zugehörigen Link für jedes Video anzeigen aus.
    5. Klicken Sie auf Save.

    Zuweisen einer benutzerdefinierten Domäne

    Wenn Portal Experiences erstellt werden, wird ihnen eine zufällige, statische URL zugewiesen. In den meisten Fällen möchten Sie über eine freundlichere URL auf die Website zugreifen. Weitere Informationen zum Zuweisen einer benutzerdefinierten Domäne zu einem Portal Experience finden Sie unter Zuweisen einer benutzerdefinierten Domäne zu einem Portalerlebnis.

    Gehen Sie folgendermaßen vor, um die einem Portal zugewiesene URL anzuzeigen und eine benutzerdefinierte URL zuzuweisen:

    1. Klicken Sie im Galerie-Site-Editor auf SITE > CONFIGURATION Die der Website zugewiesene URL wird angezeigt.
    2. Klicken Sie auf Add Custom Domain.
      Hinweis: Wenn es keine Schaltfläche „ Benutzerdefinierte Domäne hinzufügen “ gibt, wurden keine benutzerdefinierten Domains erstellt.
    3. Wählen Sie die zuvor erstellte Domain aus.
    4. Klicken Sie auf Save.

    Veröffentlichung der Erfahrung

    Durch das Veröffentlichen eines Portal Experience wird das Erlebnis unter einer öffentlichen URL verfügbar. Weitere Informationen zur Vorschau und Veröffentlichung eines Portalerlebnisses finden Sie unter Vorschau und Veröffentlichung eines Portalerlebnisses.

    Gehen Sie folgendermaßen vor, um das Erlebnis zu veröffentlichen.

    1. Klicken Sie im Galerie-Site-Editor auf Veröffentlichen.
      Hinweis: Die benutzerdefinierte Domain, die Sie der Site zugewiesen haben, sollte im Dialogfeld aufgeführt werden.
    2. Klicken Sie im Dialogfeld Website veröffentlichen auf Jetzt veröffentlichen , um die Veröffentlichung der Website zu bestätigen.
