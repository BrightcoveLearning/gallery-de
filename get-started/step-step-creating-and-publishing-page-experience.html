    Schritt für Schritt: Erstellen und Veröffentlichen eines In-Page Experience

    Der Zweck dieses Quick Start ist es, Ihnen zu zeigen, wie Sie mit Brightcove Gallery ein In-Page-Erlebnis erstellen können.

    Nach der Lektüre dieser Kurzanleitung sind Sie mit den folgenden Aufgaben vertraut:

    • Verwenden Sie Gallery, um ein neues Erlebnis zu schaffen
    • Hinzufügen von Videos zu einem In-Page-Erlebnis
    • Anpassen und Gestalten eines In-Page-Erlebnisses
    • Vorschau und Veröffentlichung eines In-Page-Erlebnisses
    Hinweis: Eine Schritt-für-Schritt-Anleitung zum Erstellen einer Portal-Erfahrung finden Sie unter Schritt für Schritt: Erstellen und Veröffentlichen eines Portal Experience.

    Zielgruppe

    Jeder Herausgeber von Video Cloud, der In-Page Experiences erstellen möchte, die in Webseiten eingebettet werden können.

    Voraussetzungen

    • Ein Video Cloud-Konto mit Zugriff auf das Gallery-Modul
    • Video Cloud-Kenntnisse
    • Bei Bedarf können Sie die in dieser Kurzanleitung verwendeten Beispiel-Videos herunterladen.
    • Hilfreich, um die In-Page-Erlebnisse mit Galerie-Schulungen anzuzeigen

    Anlagen der Kurzanleitung verwenden

    Führen Sie die folgenden Schritte aus, wenn Sie die mit dieser Kurzanleitung bereitgestellten Videos verwenden möchten.

    1. Laden Sie die Beispieldatei herunter.
    2. Verwenden Sie das Upload-Modul , um die Videos auf Ihr Video Cloud-Konto hochzuladen .
    3. Verwenden Sie das Medienmodul , um eine Tiers-Wiedergabeliste zu erstellen und die Videos zur Playlist hinzuzufügen.
    4. Verwenden Sie die Spieler Modul zu Erstellen Sie einen neuen Brightcove-Player namens Galerie-Player.
    Hinweis: Das Bild in der Beispieldatei wird später im Quick Start verwendet.

    Mit der Galerie ein In-Page-Erlebnis erstellen

    Um ein In-Page-Erlebnis mit Brightcove Gallery zu erstellen, führen Sie die folgenden Aufgaben aus:

    • Wählen Sie ein Konto
    • Erstellen Sie ein neues In-Page Experience
    • Passen Sie das Erlebnis
    • Fügen Sie dem Erlebnis Videos hinzu
    • Hinzufügen von Interaktivität zu einem Video
    • Passen Sie den Stil
    • Konfigurieren Sie die Erlebniseinstellungen
    • Überprüfen Sie die Details zur Erfahrung
    • Vorschau und Veröffentlichung des In-Page Experience

    Auswählen eines Kontos

    Abhängig von Ihrem Video Cloud-Abonnement haben Sie möglicherweise Zugriff auf mehrere Video Cloud-Konten. Bevor Sie beginnen, vergewissern Sie sich, dass Sie das richtige Konto verwenden.

    1. Melden Sie sich bei Video Cloud Studioan.
    2. Der aktuelle Video Cloud-Kontoname erscheint auf der Seite oben rechts. Wenn Sie mehrere Konten haben, klicken Sie auf das Dropdown-Menü Kontoauswahl und wählen Sie das Konto aus, in dem Sie das In-Page Experience erstellen möchten. Hierbei sollte es sich um das Konto handeln, das auch die zu verwendenden Videos enthält.
    3. Klicken Sie im Navigationskopf auf Gallery .

    Erstellen eines neuen In-Page Experience

    Gehen Sie folgendermaßen vor, um ein neues In-Page Experience zu erstellen.

    1. Klicken Sie oben auf der Seite auf den Link Erlebnisse .
    2. Wenn Sie keine Entdeckungen erstellt haben, wird eine Seite ohne Projekte angezeigt. Klicken Sie neben In-Page auf Erstellen.

      Wenn Sie bereits über Erfahrungen verfügen, klicken Sie auf Neue Erfahrung und dann oben auf der Seite auf den Link In-Page .

    3. Eine Liste der In-Page Experience-Vorlagen wird angezeigt. Suchen Sie die Vorlage Vertical Playlist und klicken Sie auf Auswählen.
      Hinweis: Sobald Sie eine Vorlage ausgewählt haben, kann sie nicht mehr geändert werden. Weitere Informationen zu Vorlagen für In-Page Experience finden Sie im Thema Überblick über In-Page Experience Templates für die Galerie .
    4. Geben Sie einen Erfahrungstitel und eine Erlebnisbeschreibungein. Wählen Sie für die Layoutgröße Responsiveaus.
      Hinweis: Diese Werte können später geändert werden. Dies wird später in diesem Quick Start behandelt.
    5. Klicken Sie auf Erfahrung erstellen.

    Der Layouteditor wird geöffnet. Der Titel der Erfahrung wird oben auf der Seite angezeigt. Der Titel kann als Teil der Erfahrungsdetailsgeändert werden.

    Anpassen des Erlebnisses

    Dem Erlebnis können Objekte hinzugefügt werden, um das Erscheinungsbild anzupassen. In-Page Experiences haben unterschiedliche Zustände und jeder Bundesstaat kann angepasst werden. Die folgenden Zustände stehen zur Anpassung zur Verfügung:

    • VOR DEM SPIELEN - Wie wird das Erlebnis beim ersten Rendern vor Beginn einer Videowiedergabe angezeigt
    • PLAYING - Wie das Erlebnis erscheint, wenn ein Video abgespielt wird
    • AFTER PLAY - Wie das Erlebnis nach dem Abspielen eines Videos aussehen wird

    Um die verschiedenen Status zu bearbeiten, klicken Sie auf einen Status und fügen Sie dann nach Bedarf Komponenten zur Verfügung. Weitere Informationen finden Sie unter Anpassen einer In-Page-Erfahrung.

    Um Objekte zu einem In-Page Experience hinzuzufügen, führen Sie die folgenden Schritte aus.

    1. Klicken Sie im linken Navigationsbereich auf LAYOUT .
    2. Klicken Sie auf den Status BEFORE PLAY Plus-Symbole () zeigen an, wo Komponenten zum Erlebnis hinzugefügt werden können. Klicken Sie oben im Erlebnis auf das Plus-Symbol. Die Komponentenauswahl wird geöffnet.
    3. Klicken Sie auf die Image-Komponente . Klicken Sie auf den Link zum Durchsuchen und wählen Sie ein Bild aus, das als Header für das Erlebnis verwendet werden soll. Beachten Sie, dass die Schnellstart-Beispieldateien ein Bild enthalten, das verwendet werden kann.
    4. Klicken Sie auf Save. Das Bild wird auf dem Erlebnis erscheinen.
    5. Klicken Sie auf das Plus-Symbol über der Bildkomponente und fügen Sie dem Erlebnis eine Text-Komponente hinzu. Geben Sie Text ein und klicken Sie dann auf Speichern.
    6. Bestätigen Sie, dass das Bild und der Text auf dem Erlebnis erscheinen.

    Um eine der Komponenteneigenschaften zu bearbeiten oder eine Komponente zu löschen, bewegen Sie den Mauszeiger über die Komponente und klicken Sie auf die Symbole zum Bearbeiten oder Löschen.

    Hinzufügen von Videos zum Erlebnis

    Abhängig von der ausgewählten Vorlage wird in der Erfahrung möglicherweise ein oder mehrere Videos gleichzeitig angezeigt. Videos können manuell zu einem Erlebnis hinzugefügt, eine Playlist ausgewählt oder eine Referenz-ID bereitgestellt werden. Weitere Informationen zum Hinzufügen von Videos zu einem In-Page-Erlebnis finden Sie unter Hinzufügen von Videos zu einem In-Page-Erlebnis.

    Um Videos zu einem In-Page Experience hinzuzufügen, führen Sie die folgenden Schritte aus.

    1. Klicken Sie in der linken Navigationsleiste auf das Symbol VIDEOS .
    2. Klicken Sie auf „ Videos hinzufügen“
    3. Wählen Sie eine Methode zur Auswahl von Videos aus , um manuell hinzuzufügenden Videos auszuwählen, ODER wählen Sie eine Methode zur Auswahl der Wiedergabeliste aus und suchen Sie die Option Animals Playlist, die Sie mit den Beispielvideos erstellt haben.
    4. Klicken Sie auf „Select“. Die Liste der Videos wird angezeigt.

    Hinzufügen von Interaktivität zu einem Video

    Link- und Karteninteraktionen können zu Videos hinzugefügt werden. Interaktionen werden angezeigt, während das Video abgespielt wird. Weitere Informationen zum Hinzufügen von Interaktivität zu einem Erlebnis finden Sie unter Hinzufügen von Interaktionen zu einem In-Page-Erlebnis.

    Gehen Sie folgendermaßen vor, um die Interaktivität für ein Video zu konfigurieren.

    1. Klicken Sie in der linken Navigationsleiste auf das Symbol VIDEOS .
    2. Wählen Sie ein Video aus, um Interaktivität hinzuzufügen, und klicken Sie auf das Link-Symbol ( ) in dem Aktionen Säule.
    3. Fügen Sie Linktext und eine Link-URLhinzu. Der Vorschauspieler zeigt eine Vorschau des Links an.
    4. Klicken Sie auf den Link Konfigurieren .
    5. Klicken Sie auf den Link Wechsel% . Stellen Sie die Startzeit auf den 50% -Punkt im Video ein.
    6. Aktivieren Sie für die Dauer bis das Video endet.
      Hinweis: Der Schieberegler unter dem Vorschauspieler kann auch verwendet werden, um die Startzeit und die Dauer für den Link festzulegen.
    7. Klicken Sie auf den Link Design . Legen Sie fest, dass der Link unten rechts neben dem Spieler angezeigt wird.
    8. Klicken Sie auf Save. Eine Liste aller für das Video konfigurierten Interaktionen wird angezeigt.
    9. Klicken Sie auf das X im Titel des Dialogs, um den Interaktivitätseditor zu schließen.

    Videos mit Interaktivität zeigen ein Symbol in der Spalte Interaktivität an, das den Typ (e) der konfigurierten Interaktionen angibt.

    Anpassen des Stils

    Der Stil steuert die Schriftarten und Farben, die für das Erlebnis verwendet werden. Weitere Informationen zum Anpassen des Stils finden Sie unter Anpassen des Stils einer In-Page-Erfahrung.

    Gehen Sie folgendermaßen vor, um den Stil zu konfigurieren.

    1. Klicken Sie in der linken Navigationsleiste auf STIL .
    2. Suchen Sie den GLOBALEN Stil. Klicken Sie auf den Kreis Textfarbe und ändern Sie die Textfarbe in Rot. Die Textkomponente, die dem Erlebnis früher hinzugefügt wurde, sollte zu Rot wechseln.
    3. Klicken Sie auf das X , um den Farbwähler zu schließen.
    4. Suchen Sie den Stil VIDEO TITLE . Klicken Sie auf den Kreis Textfarbe und ändern Sie die Textfarbe in Blau. Klicken Sie auf das X , um den Farbwähler zu schließen.
    5. Scrollen Sie durch die linke Navigation nach oben und klicken Sie auf den Link Zurück zum Layout .

    Konfigurieren der Erlebniseinstellungen

    Die Erlebniseinstellungen steuern verschiedene Aspekte des Erlebnisses, z. B. den zu verwendenden Spieler, soziale Einstellungen und andere Einstellungen. Weitere Informationen zum Anpassen der Einstellungen finden Sie unter Konfigurieren der Einstellungen für ein In-Page-Erlebnis.

    Gehen Sie folgendermaßen vor, um die Erlebniseinstellungen zu konfigurieren.

    1. Klicken Sie in der linken Navigationsleiste auf EINSTELLUNGEN > Player .
    2. Verwenden Sie die Dropdown-Liste Verfügbare Spieler , um den zuvor erstellten Galerie-Player auszuwählen. Dies ist der Spieler, der bei der In-Page Experience verwendet wird.
    3. Aktivieren Sie die Option Automatisch zum nächsten Video vorrücken .
    4. Klicken Sie auf Save.
    5. Klicken Sie in der linken Navigationsleiste auf EINSTELLUNGEN > Social .
    6. Wählen Sie aus, wie die Social Sharing-Optionen angezeigt werden sollen.
    7. Wählen Sie die Social-Media-Websites aus, auf denen Videos geteilt werden können. Einige Websites erlauben auch das Verfassen benutzerdefinierter Nachrichten.
    8. Klicken Sie auf Save.

    Überprüfung der Erfahrungsdetails

    Die Erfahrungsdetails sind der Ort, an dem der Titel und die Größe der Erfahrung konfiguriert werden. Weitere Informationen zum Anpassen von Erlebnisdetails finden Sie unter Konfigurieren der Details für ein In-Page-Erlebnis.

    Gehen Sie folgendermaßen vor, um die Erfahrungsdetails zu konfigurieren.

    1. Klicken Sie in der linken Navigationsleiste auf ÜBERBLICK .
    2. Bearbeiten Sie bei Bedarf den Titel und die Beschreibung der Erfahrung.
    3. Bestätigen Sie, dass die Layoutgröße auf Responsivefestgelegt ist. Wenn die Layoutgröße auf „Responsive“ festgelegt ist, wird die Größe an den Container auf der Seite angepasst.
    4. Wenn Änderungen vorgenommen wurden, klicken Sie auf Speichern.

    Vorschau und Veröffentlichung der In-Page Experience

    Wenn Sie eine Vorschau auf ein In-Page-Erlebnis anzeigen, können Sie sehen, wie das Erlebnis angezeigt wird, ohne es in der Galerie-Benutzeroberfläche anzeigen zu müssen. Durch die Veröffentlichung der In-Page Experience wird der Einbettungscode generiert, der erforderlich ist, um das Erlebnis innerhalb einer Webseite anzuzeigen. Weitere Informationen zur Vorschau und Veröffentlichung eines In-Page Experience finden Sie unter Vorschau und Veröffentlichung eines In-Page-Erlebnisses.

    1. Um eine Vorschau einer In-Page-Erfahrung anzuzeigen, bearbeiten Sie das Erlebnis und klicken Sie dann auf Vorschau.
    2. Eine Desktop-Vorschau wird in einem neuen Browser-Tab geöffnet.
    3. Die Dropdown-Listen in der oberen linken Ecke können verwendet werden, um eine Vorschau des Erlebnisses auf verschiedenen Gerätetypen und auch in verschiedenen Erfahrungszuständen anzuzeigen. Eine mobile Vorschau wird unten angezeigt.
    4. Um ein In-Page Experience zu veröffentlichen, klicken Sie auf Veröffentlichen & Einbetten.
    5. Klicken Sie auf Publish oder Publish
    6. Der Veröffentlichungsprozess beginnt und wenn Sie fertig sind, ändert sich der Veröffentlichungsstatus in Diese Erfahrung ist auf dem neuesten Stand.  Beachten Sie, dass es einige Minuten dauern kann, bis das Erlebnis aktualisiert wird.

    Nachdem die In-Page Experience veröffentlicht wurde, kann die teilbare URL mit anderen geteilt werden, damit sie eine Vorschau des Erlebnisses anzeigen können. Der Einbettungscode kann verwendet werden, um die In-Page Experience in eine bestehende Webseite einzubetten.

    Hinweis: Alle Änderungen, die Sie an einer In-Page Experience vornehmen, erfordern eine erneute Veröffentlichung der Erfahrung. Es sindKEINE Änderungen an einem vorhandenen Einbettungscode erforderlich, es sei denn , Sie ändern die Layoutgröße von einer festen Größe auf Responsive oder umgekehrt.
    Hinweis: Wenn die Erlebnisgröße auf Responsive festgelegt ist, muss der Einbettungscode „ Empfohlen“ verwendet werden. Der Einbettungscode „ Kein JavaScript “ reagiert nicht.
