Schritt für Schritt: Erstellen und Veröffentlichen einer In-Page-Erfahrung

{{ page.description }}

Nach Abschluss dieses Schnellstarts sollten Sie in der Lage sein:

  • Verwenden Sie Galerie, um eine neue Erfahrung zu erstellen
  • Fügen Sie Videos zu einer In-Page-Erfahrung hinzu
  • Passen Sie ein In-Page-Erlebnis an und gestalten Sie es
  • Vorschau und Veröffentlichung einer In-Page-Erfahrung
Hinweis: Eine schrittweise Anleitung zum Erstellen einer Portal-Erfahrung finden Sie unter Schritt für Schritt: Erstellen und Veröffentlichen eines Portal-Erlebnisses

Zielgruppe

Jeder Video Cloud-Publisher, der In-Page-Erlebnisse erstellen möchte, die in Webseiten eingebettet werden können.

Voraussetzungen

  • Ein Video Cloud-Konto mit Zugriff auf das Galerie-Modul
  • Grundkenntnisse in Video Cloud
  • Die in diesem Schnellstart verwendeten Beispielvideos sind für verfügbar herunterladen wenn benötigt
  • Hilfreich beim Anzeigen der Erstellen von In-Page-Erfahrungen mithilfe der Galerie Ausbildung

Verwenden der Schnellstart-Assets

Wenn Sie die mit diesem Schnellstart gelieferten Beispielvideos verwenden möchten, führen Sie die folgenden Schritte aus.

  1. Herunterladen die Beispieldatei.
  2. Verwenden Sie die Hochladen Modul zu Laden Sie die Videos hoch zu Ihrem Video Cloud-Konto.
  3. Verwenden Sie die Medien Modul zu Erstellen Sie eine Tier-Wiedergabeliste und fügen Sie die Videos zur Wiedergabeliste hinzu.
  4. Verwenden Sie die Spieler Modul zu Erstellen Sie einen neuen Brightcove-Player namens Galerie-Player.
Hinweis: Das Bild in der Beispieldatei wird später im Schnellstart verwendet.

Verwenden der Galerie zum Erstellen einer In-Page-Erfahrung

Um eine In-Page-Erfahrung mit Brightcove Gallery zu erstellen, führen Sie die folgenden Aufgaben aus:

  • Wählen ein Konto
  • Erstellen Sie eine neue In-Page-Erfahrung
  • Passen Sie die Erfahrung an
  • Fügen Sie der Erfahrung Videos hinzu
  • Fügen Sie einem Video Interaktivität hinzu
  • Passen Sie den Stil an
  • Konfigurieren Sie die Erlebniseinstellungen
  • Überprüfen Sie die Details der Erfahrung
  • Vorschau und Veröffentlichung der In-Page-Erfahrung

Konto auswählen

Abhängig von Ihrem Video Cloud-Abonnement haben Sie möglicherweise Zugriff auf mehrere Video Cloud-Konten. Stellen Sie vor dem Start sicher, dass Sie das richtige Konto verwenden.

  1. Einloggen in Video Cloud Studio.
  2. Der aktuelle Name des Video Cloud-Kontos wird in der oberen rechten Ecke der Seite angezeigt. Wenn Sie über mehrere Konten verfügen, klicken Sie auf die Dropdown-Liste Kontoauswahl und wählen Sie das Konto aus, in dem Sie die In-Page-Erfahrung erstellen möchten. Beachten Sie, dass dies auch das Konto sein sollte, das die Videos enthält, die Sie ebenfalls verwenden möchten.
  3. Klicken Galerie im Navigationsheader.

Erstellen einer neuen In-Page-Erfahrung

Führen Sie die folgenden Schritte aus, um eine neue In-Page-Erfahrung zu erstellen.

  1. Drücke den Erfahrungen Link oben auf der Seite.
  2. Wenn Sie keine Erfahrungen erstellt haben, wird eine Seite ohne Projekte angezeigt. Klicken Erstellen neben an Auf Seite.

    Wenn Sie bereits Erfahrungen gesammelt haben, klicken Sie auf Neue Erfahrung und klicken Sie dann auf Auf Seite Registerkarte oben auf der Seite.

  3. Eine Liste der In-Page Experience-Vorlagen wird angezeigt. Suchen Sie die Vertikale Wiedergabeliste Vorlage und klicken Wählen.
    Hinweis: Sobald Sie eine Vorlage ausgewählt haben, kann diese nicht mehr geändert werden. Weitere Informationen zu In-Page Experience-Vorlagen finden Sie im Übersicht über die Vorlagen für die Galerie-In-Page-Erfahrung Thema.
  4. Geben Sie ein Titel erleben und Erfahrungsbeschreibung. Wählen Sie für die Layoutgröße Responsive aus.
    Hinweis: Diese Werte können später geändert werden. Dies wird später in diesem Quick Start behandelt.
  5. Klicken Erfahrung schaffen.

Der Layout-Editor wird geöffnet. Der Erfahrungstitel wird oben auf der Seite angezeigt. Der Titel kann im Rahmen von geändert werden Details erleben.

Anpassen der Erfahrung

Der Erfahrung können Objekte hinzugefügt werden, um das Erscheinungsbild anzupassen. In-Page-Erfahrungen haben unterschiedliche Status und jeder Status kann angepasst werden. Die folgenden Status können angepasst werden:

  • VOR DEM SPIELEN - Wie die Erfahrung beim ersten Rendern angezeigt wird, bevor eine Videowiedergabe gestartet wurde
  • SPIELEN - Wie die Erfahrung angezeigt wird, wenn ein Video abgespielt wird
  • Nach dem Spielen - Wie die Erfahrung nach der Wiedergabe eines Videos angezeigt wird

Um die verschiedenen Status zu bearbeiten, klicken Sie auf einen Status und fügen Sie der Erfahrung nach Bedarf Komponenten hinzu. Weitere Informationen finden Sie unter Anpassen einer In-Page-Erfahrung.

Führen Sie die folgenden Schritte aus, um Objekte zu einer In-Page-Erfahrung hinzuzufügen.

  1. Klicken LAYOUT in der linken Navigation.
  2. Drücke den VOR DEM SPIELEN Zustand. Plus-Symbole () geben an, wo Komponenten zum Erlebnis hinzugefügt werden können. Klicken Sie oben in der Erfahrung auf das Plus-Symbol. Die Komponentenauswahl wird geöffnet.
  3. Klick auf das Bild Komponente. Drücke den Durchsuche Verknüpfen Sie ein Bild und wählen Sie es aus, um es als Überschrift für die Erfahrung zu verwenden. Beachten Sie, dass die Schnellstart-Beispieldateien ein Bild enthalten, das verwendet werden kann.
  4. Klicken speichern. Das Bild wird auf der Erfahrung erscheinen.
  5. Klicken Sie auf das Plus-Symbol über der Bildkomponente und fügen Sie ein hinzu Text Bestandteil der Erfahrung. Geben Sie einen Text ein und klicken Sie dann auf speichern.
  6. Vergewissern Sie sich, dass das Bild und der Text in der Erfahrung angezeigt werden.

Um eine der Komponenteneigenschaften zu bearbeiten oder eine Komponente zu löschen, bewegen Sie den Mauszeiger über die Komponente und klicken Sie auf die Symbole zum Bearbeiten oder Löschen.

Hinzufügen von Videos zur Erfahrung

Abhängig von der ausgewählten Vorlage werden in der Erfahrung möglicherweise ein oder mehrere Videos gleichzeitig angezeigt. Videos können manuell zu einer Erfahrung hinzugefügt, eine Wiedergabeliste ausgewählt oder eine Referenz-ID bereitgestellt werden. Weitere Informationen zum Hinzufügen von Videos zu einer In-Page-Erfahrung finden Sie unter Hinzufügen von Videos zu einer In-Page-Erfahrung.

Führen Sie die folgenden Schritte aus, um Videos zu einer In-Page-Erfahrung hinzuzufügen.

  1. Klicken VIDEOS Symbol in der linken Navigation.
  2. Klicken Videos hinzufügen.
  3. Wählen Sie a Methode von Videos auswählen um Videos zum Hinzufügen manuell auszuwählen, ODER , wählen Sie a Methode von Wählen Sie Playlist und durchsuchen und wählen Sie die Tiere Playlist, die Sie mit den Beispielvideos erstellt haben.
  4. Klicken Wählen. Die Liste der Videos wird angezeigt.

Hinzufügen von Interaktivität zu einem Video

Link- und Karteninteraktionen können zu Videos hinzugefügt werden. Interaktionen werden angezeigt, während das Video abgespielt wird. Weitere Informationen zum Hinzufügen von Interaktivität zu einer Erfahrung finden Sie unter Hinzufügen von Interaktionen zu einer In-Page-Erfahrung.

Führen Sie die folgenden Schritte aus, um die Interaktivität für ein Video zu konfigurieren.

  1. Klicken VIDEOS Symbol in der linken Navigation.
  2. Wählen Sie ein Video aus, dem Sie Interaktivität hinzufügen möchten, und klicken Sie auf das Linksymbol () in dem Aktionen Säule.
  3. Hinzufügen Link Text und ein URL verknüpfen. Der Preview-Player zeigt eine Vorschau des Links an.
  4. Drücke den Konfigurieren Verknüpfung.
  5. Drücke den wechseln zu % Verknüpfung. Stellen Sie die Startzeit um der 50% -Punkt im Video zu sein.
  6. Für die Dauer , prüfen Bis das Video endet.
    Hinweis: Der Schieberegler unter dem Vorschauspieler kann auch verwendet werden, um die Startzeit und Dauer für den Link festzulegen.
  7. Drücke den Design Verknüpfung. Stellen Sie den Link so ein, dass er unten rechts im Player angezeigt wird.
  8. Klicken speichern. Eine Liste aller für das Video konfigurierten Interaktionen wird angezeigt.
  9. Drücke den X. im Dialogtitel, um den Interaktivitätseditor zu schließen.

Videos mit Interaktivität zeigen ein Symbol in der Interaktivität Spaltenangabe die Art (en) der konfigurierten Interaktionen.

Anpassen des Stils

Der Stil steuert die Schriftarten und Farben, die für das Erlebnis verwendet werden. Weitere Informationen zum Anpassen des Stils finden Sie unter Anpassen des Stils einer In-Page-Erfahrung.

Führen Sie die folgenden Schritte aus, um den Stil zu konfigurieren.

  1. Klicken STIL in der linken Navigation.
  2. Suchen Sie die GLOBAL Stil. Drücke den Textfarbe kreisen Sie ein und ändern Sie die Textfarbe in Rot. Die Textkomponente, die zuvor zur Erfahrung hinzugefügt wurde, sollte in Rot geändert werden.
  3. Drücke den X. um den Farbwähler zu schließen.
  4. Suchen Sie die VIDEOTITEL Stil. Drücke den Textfarbe kreisen Sie ein und ändern Sie die Textfarbe in Blau. Drücke den X. um den Farbwähler zu schließen.
  5. Scrollen Sie mit der linken Navigation nach oben und klicken Sie auf Zurück zum Layout Verknüpfung.

Konfigurieren der Erlebniseinstellungen

Die Erfahrungseinstellungen steuern verschiedene Aspekte der Erfahrung, wie z. B. den zu verwendenden Player, soziale Einstellungen und andere Einstellungen. Weitere Informationen zum Anpassen der Einstellungen finden Sie unter Konfigurieren der Einstellungen für eine In-Page-Erfahrung.

Führen Sie die folgenden Schritte aus, um die Erlebniseinstellungen zu konfigurieren.

  1. Klicken EINSTELLUNGEN> Player in der linken Navigation.
  2. Verwenden Sie die Verfügbare Spieler Dropdown-Liste zur Auswahl der Galerie-Player Sie haben früher erstellt. Dies ist der Player, der für die In-Page-Erfahrung verwendet wird.
  3. Überprüf den Fahren Sie automatisch mit dem nächsten Video fort Möglichkeit.
  4. Klicken speichern.
  5. Klicken EINSTELLUNGEN> Sozial in der linken Navigation.
  6. Wählen Sie aus, wie die Social-Sharing-Optionen angezeigt werden sollen.
  7. Wählen Sie die Social-Media-Websites aus, für die Videos freigegeben werden können. Auf einigen Websites können auch benutzerdefinierte Nachrichten veröffentlicht werden.
  8. Klicken speichern.

Überprüfen der Erfahrungsdetails

In den Erfahrungsdetails werden der Erfahrungstitel und die Größe konfiguriert. Weitere Informationen zum Anpassen von Erfahrungsdetails finden Sie unter Konfigurieren der Details für eine In-Page-Erfahrung.

Führen Sie die folgenden Schritte aus, um die Erfahrungsdetails zu konfigurieren.

  1. Klicken ÜBERBLICK in der linken Navigation.
  2. Bearbeiten Sie bei Bedarf den Titel und die Beschreibung der Erfahrung.
  3. Bestätige das Layoutgröße ist eingestellt auf Reaktionsschnell. Wenn die Layoutgröße auf "Ansprechend" eingestellt ist, wird die Größe an den Container auf der Seite angepasst.
  4. Wenn Änderungen vorgenommen wurden, klicken Sie auf speichern.

Vorschau und Veröffentlichung der In-Page-Erfahrung

In der Vorschau einer In-Page-Erfahrung können Sie sehen, wie die Erfahrung angezeigt wird, ohne sie in der Galerie-Oberfläche anzeigen zu müssen. Durch das Veröffentlichen der In-Page-Erfahrung wird der Einbettungscode generiert, der zum Anzeigen der Erfahrung innerhalb einer Webseite erforderlich ist. Weitere Informationen zum Anzeigen einer Vorschau und zum Veröffentlichen einer In-Page-Erfahrung finden Sie unter Vorschau und Veröffentlichung einer In-Page-Erfahrung.

  1. Um eine Vorschau einer In-Page-Erfahrung anzuzeigen, bearbeiten Sie die Erfahrung und klicken Sie dann auf Vorschau.
  2. Eine Desktop-Vorschau wird in einem neuen Browser-Tab geöffnet.
  3. Die Dropdown-Listen in der oberen linken Ecke können verwendet werden, um eine Vorschau der Erfahrung auf verschiedenen Gerätetypen und auch während verschiedener Erfahrungszustände anzuzeigen. Eine mobile Vorschau wird unten angezeigt.
  4. Klicken Sie auf, um eine In-Page-Erfahrung zu veröffentlichen Veröffentlichen & Einbetten.
  5. Klicken Veröffentlichen oder Änderungen veröffentlichen.
  6. Der Veröffentlichungsprozess beginnt und wenn Sie fertig sind, wird der Veröffentlichungsstatus wird sich ändern zu Diese Erfahrung ist aktuell.  Beachten Sie, dass es einige Minuten dauern kann, bis die Erfahrung aktualisiert ist.

Nachdem die In-Page-Erfahrung veröffentlicht wurde, wird die Gemeinsam nutzbare URL kann mit anderen geteilt werden, damit sie eine Vorschau der Erfahrung sehen können. Der Einbettungscode kann verwendet werden, um die In-Page-Erfahrung in eine vorhandene Webseite einzubetten.

Hinweis: Für alle Änderungen, die Sie an einer In-Page-Erfahrung vornehmen, muss die Erfahrung erneut veröffentlicht werden. KEINE Änderungen an einem vorhandenen Einbettungscode sind erforderlich, ES SEI DENN, Sie ändern die Layoutgröße von einer festen Größe in responsive oder umgekehrt.
Hinweis: Wenn die Erlebnisgröße auf Responsive festgelegt ist, muss der Empfohlene Einbettungscode verwendet werden. Die Kein JavaScript Einbettungscode reagiert nicht.
