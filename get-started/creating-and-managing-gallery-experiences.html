    Erstellen und Verwalten von Galerie-Erfahrungen

    In diesem Thema erfahren Sie, wie Sie die Details für ein In-Page Experience konfigurieren.

    Wenn Sie schnell mit der Erstellung und Veröffentlichung eines Galerieerlebnisses beginnen möchten, sehen Sie sich eines dieser Dokumente an:

    • Schritt für Schritt: Erstellen und Veröffentlichen eines Portal Experience
    • Schritt für Schritt: Erstellen und Veröffentlichen eines In-Page Experience

    Wenn Sie das Galerie-Modul öffnen, werden auf der Seite „ Erlebnisse “ alle erstellten Erlebnisse aufgeführt.

    Wenn keine Erlebnisse erstellt wurden, werden Sie aufgefordert, ein Erlebnis zu schaffen. Die Seite „ Erlebnisse “ bietet die Möglichkeit:

    • Erfahrungen löschen
    • Nehmen Sie Erlebnisse offline
    • Doppelte Erlebnisse
    • Erlebnisse bearbeiten
    • Schaffen Sie Erfahrungen
    • Filtern Sie Erfahrungen
    • Suche nach Erlebnissen

    Klicken Sie auf eine Spaltenüberschrift, um die Daten zu sortieren. Die Spaltenauswahl () kann verwendet werden, um die Spalten auszuwählen, die auf der Seite angezeigt werden sollen.

    Die folgenden Spalten können angezeigt werden.

    • Titel - Titel der Erfahrung
    • Template - Vorlage, die von der Erfahrung verwendet wird. Die Art der Erfahrung wird unten angezeigt:
      • Portal - Website, die eine Inhaltsbibliothek jeder Größe mit optionalem Call-to-Action- und Anzeigensupport unterstützt
      • In-Page - Veröffentlichen Sie ein oder mehrere Videos und fügen Sie Interaktivität hinzu und fügen Sie dieses Layout dann Ihrer Website hinzu
      • Landing Page - Einzelseite mit Lead-Formular und optionalem Call-to-Action- und Anzeigensupport
      • Veranstaltung - Website optimiert für die Förderung und Übertragung von Live-Events
    • Vorlagenstatus - Status der Vorlage
      • Aktuell - Die Erfahrung nutzt die neueste Version der Projektvorlage
      • Benötigt Update - Ein Template-Update für das Erlebnis ist verfügbar
      • Resume Update - Vorlagenaktualisierung ist im Gange, wurde aber nicht veröffentlicht
    • Aktualisiert - Datum und Uhrzeit des letzten Erlebnis-Updates
    • Erfahrungsstatus - Status der Erfahrung
      • Online - Die Erfahrung ist online und kann abgerufen werden
      • Offline - Die Erfahrung ist offline und kann nicht abgerufen werden
      • Unveröffentlichte Änderungen - Die Erfahrung ist online und kann abgerufen werden, aber Änderungen wurden an der Erfahrung vorgenommen und nicht veröffentlicht

    Wenn Sie auf eine Zeile klicken, werden die Erlebnisdetails rechts auf der Seite angezeigt.

    Erlebnisse schaffen

    Um ein neues Erlebnis zu erstellen, klicken Sie auf New Experience. Wenn Sie ein neues Erlebnis erstellen, wählen Sie zunächst eine Vorlage aus. Die Vorlage steuert das allgemeine Erscheinungsbild und das Verhalten. Nachdem Sie eine Vorlage ausgewählt haben, können Sie das Erlebnis anpassen. Weitere Informationen finden Sie in einem der folgenden Themen:

    • Übersicht: Erstellung, Bearbeitung und Veröffentlichung von Portal-Erlebnissen
    • Übersicht: Erstellen, Bearbeiten und Veröffentlichen von In-Page-Erlebnissen

    Erfahrungen duplizieren

    Das Duplizieren eines Erlebnisses kann nützlich sein, wenn Sie ein Erlebnis in mehreren Sprachen erstellen müssen oder wenn Sie Microsites aus dem Erscheinungsbild einer bestehenden Erfahrung erstellen. Um ein Erlebnis zu duplizieren, klicken Sie auf eine Zeile und klicken Sie dann auf Duplizieren.

    Geben Sie einen Erfahrungstitel und eine Beschreibung ein und klicken Sie dann auf Speichern.

    Redigieren von

    Um ein Erlebnis zu bearbeiten, klicken Sie auf den Erfahrungstitel. Durch das Bearbeiten einer Erfahrung wird der Galerie-Site-Editor geöffnet. Informationen zum Bearbeiten von Erlebnissen finden Sie in einem der folgenden Themen:

    • Verwenden des Site-Editors zum Anpassen einer Portal-Erfahrung
    • Anpassen einer In-Page-Erfahrung

    Entdeckungen offline nehmen

    Wenn Sie ein Erlebnis offline nehmen, ist es nicht mehr zugänglich. Um ein Erlebnis offline zu nehmen, klicken Sie auf eine Zeile und dann auf Offline schalten. Um zu bestätigen, dass die Erfahrung offline geschaltet werden sollte, klicken Sie auf Offline schalten. Um ein Erlebnis wieder verfügbar zu machen, muss die Erfahrung erneut veröffentlicht werden.

    Erlebnisse löschen

    Um ein Erlebnis zu löschen, klicken Sie auf eine Zeile und dann auf Löschen. Bestätigen Sie den Löschvorgang durch Klicken auf Delete.

    Hinweis: Wenn Ihre Erfahrung für die Verwendung einer benutzerdefinierten Domäne konfiguriert ist, sollten Sie die dem Erlebnis zugewiesene benutzerdefinierte Domäne entfernen und das Erlebnis dann erneut veröffentlichen, BEVOR Sie das Erlebnis löschen.

    Filtern von Erleb

    Ein Filter kann auf die Seite Erlebnisse angewendet werden, um die angezeigten Arten von Erlebnissen einzuschränken. Um einen Filter anzuwenden, wählen Sie die Art der anzuzeigenden Erlebnisse aus:

    • Alle - Alle Erlebnisse anzeigen
    • Veröffentlicht - Alle veröffentlichten (Online-) Erlebnisse anzeigen
    • Unveröffentlicht - Alle Erfahrungen anzeigen, die offline geschaltet wurden
    • Template Update benötigt - Alle Erlebnisse anzeigen, für die Vorlagenaktualisierungen verfügbar sind
    • Landing Page - Alle Landingpage-Erlebnisse anzeigen
    • Event - Alle Event-Erlebnisse anzeigen
    • Portal - Erlebnisse im Display
    • In-Page - Alle In-Page-Erlebnisse anzeigen

    Auf der Suche nach

    Das Suchfeld kann verwendet werden, um die Liste der Erlebnisse zu durchsuchen. Bei der Suche werden die folgenden Eigenschaften durchsucht:

    • Titel
    • Beschreibung
    • ID
    • Vorlage
    • URLs
