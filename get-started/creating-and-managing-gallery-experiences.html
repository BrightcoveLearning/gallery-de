Erstellen und Verwalten von Galerieerlebnissen

{{ page.description }}

Wenn Sie schnell mit dem Erstellen und Veröffentlichen einer Galerieerfahrung beginnen möchten, lesen Sie eines der folgenden Dokumente:

  • Schritt für Schritt: Erstellen und Veröffentlichen einer Portal-Erfahrung
  • Schritt für Schritt: Erstellen und Veröffentlichen einer In-Page-Erfahrung

Wenn Sie das Galerie-Modul öffnen, wird die Erfahrungen Auf dieser Seite werden alle erstellten Erfahrungen aufgelistet.

Wenn keine Erfahrungen erstellt wurden, werden Sie dazu aufgefordert eine Erfahrung schaffen. Das Erfahrungen Seite bietet die Möglichkeit:

  • Erfahrungen löschen
  • Erfahrungen offline machen
  • Doppelte Erfahrungen
  • Erfahrungen bearbeiten
  • Erfahrungen schaffen
  • Filtern Sie Erfahrungen
  • Sucherfahrungen

Klicken Sie auf eine Spaltenüberschrift, um die Daten zu sortieren. Die Spaltenauswahl () kann verwendet werden, um die Spalten auszuwählen, die auf der Seite angezeigt werden sollen.

Die folgenden Spalten können angezeigt werden.

  • Titel - Titel der Erfahrung
  • Vorlage - Vorlage, die von der Erfahrung verwendet wird. Die Art der Erfahrung wird unten angezeigt:
    • Portal - Site, die eine Inhaltsbibliothek beliebiger Größe mit optionaler Handlungsaufforderung und Anzeigenunterstützung unterstützt
    • Auf Seite - Veröffentlichen Sie ein oder mehrere Videos, fügen Sie Interaktivität hinzu und fügen Sie dieses Layout Ihrer Website hinzu
    • Landing Page - Einseitige Website mit Lead-Formular und optionaler Handlungsaufforderung und Anzeigenunterstützung
    • Veranstaltung - Website optimiert für die Werbung und Übertragung von Live-Events
  • Vorlagenstatus - Status der Vorlage
    • Auf dem neusten Stand - Die Erfahrung verwendet die neueste Version der Projektvorlage
    • Muss aktualisiert werden - Ein Vorlagen-Update für die Erfahrung ist verfügbar
    • Update fortsetzen - Das Vorlagen-Update wird ausgeführt, aber noch nicht veröffentlicht
  • Aktualisiert - Datum und Uhrzeit des letzten Erfahrungsupdates
  • Erfahrungsstatus - Status der Erfahrung
    • Online - Die Erfahrung ist online und kann abgerufen werden
    • Offline - Die Erfahrung ist offline und kann nicht abgerufen werden
    • Unveröffentlichte Änderungen - Die Erfahrung ist online und kann abgerufen werden, es wurden jedoch Änderungen an der Erfahrung vorgenommen und nicht veröffentlicht

Wenn Sie auf eine Zeile klicken, werden die Erfahrungsdetails auf der rechten Seite angezeigt.

Erfahrungen schaffen

Klicken Sie auf, um eine neue Erfahrung zu erstellen Neue Erfahrung. Wenn Sie eine neue Erfahrung erstellen, wählen Sie zunächst eine Vorlage aus. Die Vorlage steuert das allgemeine Erscheinungsbild und Verhalten. Nachdem Sie eine Vorlage ausgewählt haben, können Sie die Erfahrung anpassen. Weitere Informationen finden Sie in einem dieser Themen:

  • Überblick: Portalerfahrungen erstellen, bearbeiten und veröffentlichen
  • Überblick: Erstellen, Bearbeiten und Veröffentlichen von In-Page-Erfahrungen

Erfahrungen duplizieren

Das Duplizieren einer Erfahrung kann nützlich sein, wenn Sie eine Erfahrung in mehreren Sprachen erstellen müssen oder wenn Sie Microsites aus dem Erscheinungsbild einer vorhandenen Erfahrung erstellen. Um eine Erfahrung zu duplizieren, klicken Sie auf eine Zeile und dann auf Duplikat.

Geben Sie ein Titel erleben und Beschreibung und dann klicken speichern.

Erfahrungen bearbeiten

Um eine Erfahrung zu bearbeiten, klicken Sie auf die Erfahrung Titel. Durch das Bearbeiten einer Erfahrung wird der Galerie-Site-Editor geöffnet. Informationen zu Bearbeitungserfahrungen finden Sie in einem der folgenden Themen:

  • Verwenden des Site-Editors zum Anpassen einer Portal-Erfahrung
  • Anpassen einer In-Page-Erfahrung

Erfahrungen offline machen

Wenn Sie eine Erfahrung offline schalten, ist sie nicht mehr verfügbar. Um eine Erfahrung offline zu schalten, klicken Sie auf eine Zeile und dann auf Offline nehmen. Klicken Sie auf, um zu bestätigen, dass die Erfahrung offline geschaltet werden soll Offline nehmen. Um eine Erfahrung wieder verfügbar zu machen, muss die Erfahrung erneut veröffentlicht werden.

Erfahrungen löschen

Um eine Erfahrung zu löschen, klicken Sie auf eine Zeile und dann auf Löschen. Bestätigen Sie den Löschvorgang durch Klicken auf Löschen.

Hinweis: Wenn Ihre Erfahrung für die Verwendung einer benutzerdefinierten Domäne konfiguriert ist, sollten Sie die benutzerdefinierte Domäne entfernen, die der Erfahrung zugewiesen ist, und das Erlebnis dann erneut veröffentlichen, BEVOR Sie das Erlebnis löschen.

Erfahrungen filtern

Auf der Seite "Erfahrungen" kann ein Filter angewendet werden, um die Art der angezeigten Erfahrungen einzuschränken. Um einen Filter anzuwenden, wählen Sie die Art der anzuzeigenden Erfahrungen aus:

  • Alle - Alle Erfahrungen anzeigen
  • Veröffentlicht - Alle veröffentlichten (Online-) Erfahrungen anzeigen
  • Unveröffentlicht - Alle Erfahrungen anzeigen, die offline geschaltet wurden
  • Vorlagenaktualisierung erforderlich - Zeigen Sie alle Erfahrungen an, für die Vorlagenaktualisierungen verfügbar sind
  • Landing Page - Alle Landing Page-Erlebnisse anzeigen
  • Veranstaltung - Alle Ereigniserlebnisse anzeigen
  • Portal - Portal-Erfahrungen anzeigen
  • Auf Seite - Alle In-Page-Erfahrungen anzeigen

Erfahrungen suchen

Das Suche Feld kann verwendet werden, um die Liste der Erfahrungen zu durchsuchen. Bei der Suche werden die folgenden Eigenschaften durchsucht:

  • Titel
  • Beschreibung
  • ICH WÜRDE
  • Vorlage
  • URLs
