Erste Schritte mit Brightcove Gallery

{{ page.description }}

Wenn Sie schnell mit dem Erstellen und Veröffentlichen einer Galerieerfahrung beginnen möchten, lesen Sie eines der folgenden Themen:

  • Portal-Erfahrungen - - Schritt für Schritt: Erstellen und Veröffentlichen einer Portal-Erfahrung
  • In-Page-Erfahrungen - - Schritt für Schritt: Erstellen und Veröffentlichen einer In-Page-Erfahrung
  • Live-Event-Portal - - Erstellen eines Live Event Portal-Erlebnisses
  • Live Event In-Page - - Erstellen einer Live-Event-In-Page-Erfahrung
  • Landing Page - - Erstellen einer Landing Page-Erfahrung

Um einen Überblick über die Galerie zu erhalten und mehr über die Arten von Erfahrungen zu erfahren, die erstellt werden können, besuchen Sie Training on Demand: Einführung in die Galerie.

 

Verwenden der Brightcove-Galerie

Um auf das Galerie-Modul zuzugreifen, melden Sie sich bei Video Cloud Studio an und klicken Sie auf Galerie im Navigationsmenü.

Von der Homepage aus können Sie:

  • Erstellen und verwalten Sie Erfahrungen
  • Konfigurieren Sie die Galerieeinstellungen

Bevor Sie eine Portal-Erfahrung erstellen, sollten Sie dies tun Konfigurieren Sie die Galerieeinstellungen.

Erfahrungen schaffen und verwalten

Beim Öffnen des Galerie-Moduls wird die Erfahrungen Die Seite wird standardmäßig angezeigt. Wenn keine Erfahrungen erstellt wurden, werden Sie aufgefordert, eine neue Erfahrung zu erstellen. Eine Übersicht über die Arten von Erfahrungen, die erstellt werden können, finden Sie unter Überblick über Galerieerfahrungen.

Weitere Informationen zum Erstellen und Verwalten von Galerieerlebnissen finden Sie unter Erstellen und Verwalten von Galerieerlebnissen.

Galerieeinstellungen konfigurieren

Beim Erstellen einer Portal-Erfahrung muss die Galerie mit konfiguriert werden ein Standardspieler. Weitere Informationen zu den Galerieeinstellungen finden Sie unter Galerieeinstellungen konfigurieren.
