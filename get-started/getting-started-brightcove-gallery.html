    Erste Schritte mit Brightcove Gallery

    Dieses Thema bietet einen Überblick über das Galerie-Modul.

    Wenn Sie schnell mit der Erstellung und Veröffentlichung eines Galerieerlebnisses beginnen möchten, sehen Sie sich eines dieser Themen an:

    • Portalerfahrungen - Schritt für Schritt: Erstellen und Veröffentlichen eines Portal Experience
    • In-Page Experiences - Schritt für Schritt: Erstellen und Veröffentlichen eines In-Page Experience
    • Live Event Portal - Erstellen eines Live-Event-Portal-Erlebnisses
    • Live-Event In-Page - Erstellen einer In-Page-Erfahrung für Live-Events
    • Landing Page - Erstellen eines Landingpage-Erlebnisses

    Um einen Überblick über die Galerie zu erhalten und mehr über die Arten von Erfahrungen zu erfahren, die erstellt werden können, nehmen Sie an Training on Demand: Einführung in die Gallery.

     

    Brightcove Gallery verwenden

    Melden Sie sich bei Video Cloud Studio an, und klicken Sie im Navigationsmenü auf Gallery, um sich beim Gallery-Modul anzumelden.

    Über die Homepage erledigen Sie folgende Aufgaben:

    • Erstellen und verwalten Sie Erfahrungen
    • Gallery-Einstellungen konfigurieren

    Bevor Sie ein Portal Experience erstellen, sollten Sie die Galerie-Einstellungen konfigurieren.

    Erfahrungen schaffen und verwalten

    Beim Öffnen des Galerie-Moduls wird standardmäßig die Seite „ Erlebnisse “ angezeigt. Wenn keine Erlebnisse erstellt wurden, werden Sie aufgefordert, ein neues Erlebnis zu schaffen. Einen Überblick über die Arten von Erlebnissen, die erstellt werden können, finden Sie unter Überblick über Galerieerlebnisse.

    Weitere Informationen zum Erstellen und Verwalten von Galerie-Erlebnissen finden Sie unter Erstellen und Verwalten von Galerie-Erlebnissen.

    Gallery-Einstellungen konfigurieren

    Beim Erstellen eines Portalerlebnisses muss Gallery mit einem Standardplayer konfiguriert sein. Weitere Informationen über Gallery-Einstellungen finden Sie unter Gallery-Einstellungen konfigurieren.
