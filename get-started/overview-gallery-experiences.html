Überblick über Galerieerfahrungen

{{ page.description }}

Mit der Brightcove-Galerie können Sie auf einfache Weise leistungsstarke, ansprechende Videoerlebnisse erstellen, ohne dass eine benutzerdefinierte Entwicklung erforderlich ist. In der Galerie können Sie die folgenden Arten von Erlebnissen erstellen:

  • Portal
  • Auf Seite
  • Landing Page
  • Veranstaltung
Hinweis: Wenn Sie schnell mit dem Erstellen und Veröffentlichen eines Galerie-Erlebnisses beginnen möchten, sehen Sie sich eines der schrittweisen Themen zum Erstellen von Portal - oder In-Page-Erlebnissen an.

Wenn Sie die Art der zu erstellenden Erfahrung auswählen, bietet Gallery eine Reihe von Vorlagen für beide Portal (einschließlich Landing Page und Event Erfahrungen) und Auf Seite Erfahrungen, die das allgemeine Erscheinungsbild der Erfahrung steuern.

Portal

Portal-Erlebnisse sind eigenständige Video-Sites (dh Video-Portale), auf die unter einer von Brightcove gehosteten URL zugegriffen wird. Im folgenden Beispiel wurden Video Cloud- und Galerie-Schulungsvideos in einer Portal-Erfahrung organisiert. Wenn Sie links eine Kategorie auswählen, werden die zugehörigen Videos in dieser Kategorie im Hauptteil der Seite angezeigt.

Auf Seite

In-Page-Erlebnisse sind in eine vorhandene Webseite eingebettet. In-Page-Erlebnisse ermöglichen es Ihnen, ein Videolayout zu erstellen und es auf Ihrer vorhandenen Website anzuzeigen. In-Page-Erlebnisse unterstützen sowohl die Interaktivität innerhalb des Players als auch außerhalb des Players, z. B. Links und Karten, die während der Wiedergabe des Videos angezeigt werden können. Im folgenden Beispiel wurde eine In-Page-Erfahrung (die Video-Miniaturansichten) in eine vorhandene Webseite eingebettet. Durch Klicken auf eine der Miniaturansichten des Videos wird ein Videoplayer in einem Leuchtkasten geöffnet.

Sehen Sie sich ein Beispiel für eine In-Page-Erfahrung an

Landingpage

Landing Page-Erlebnisse sind einzelne Videoseiten, die von Brightcove gehostet werden und normalerweise als Teil einer E-Mail-Kampagne verwendet werden. Wenn ein Benutzer auf einen Link in einer E-Mail-Nachricht klickt, wird er zu einer Landing Page-Erfahrung weitergeleitet, auf der er ein einzelnes Video anzeigen kann. Das folgende Beispiel enthält auch ein Lead-Formular, mit dem Viewer-Informationen erfasst werden können.

Veranstaltung

Mit Event-Erlebnissen können Sie Ihr Publikum vor, während und nach Ihrem Live-Event einbeziehen, indem Sie ein Markenziel vor dem Event, eine Live-Stream-Event-Seite und ein Videoarchiv nach dem Event erstellen. Ereigniserlebnisse können als eigenständige Portalseiten erstellt oder als In-Page-Erlebnis in eine vorhandene Seite eingebettet werden. Im folgenden Beispiel wurde ein Live-Ereignisportal erstellt, das so konfiguriert ist, dass der Status vor dem Ereignis angezeigt wird (beachten Sie den Countdown-Timer auf der Startseite).
